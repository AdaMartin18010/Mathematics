## 📖 核心概念定义 | Core Concept Definitions

### 1. 数学 (Mathematics)

#### 内涵 (Intension)

**中文定义**:
数学是研究数量、结构、空间、变化以及它们之间关系的抽象科学。它通过逻辑推理、公理化方法和形式化证明来建立知识体系。

**English Definition**:
Mathematics is the abstract science that studies quantity, structure, space, change, and the relationships between them. It establishes knowledge systems through logical reasoning, axiomatic methods, and formal proofs.

#### 外延 (Extension)

- **纯数学**: 代数、几何、分析、数论、拓扑等
- **应用数学**: 计算数学、统计数学、运筹学、金融数学等
- **数学基础**: 集合论、逻辑、证明论、模型论等

#### 属性 (Attributes)

| 属性 | 描述 | 强度 |
| ---- |------| ---- |
| 抽象性 | 脱离具体事物，研究抽象结构 | 极高 |
| 严谨性 | 基于严格逻辑和证明 | 极高 |
| 普遍性 | 适用于多个领域 | 高 |
| 形式化 | 可用形式语言表达 | 高 |
| 可计算性 | 部分内容可算法化 | 中高 |

#### 关系 (Relations)

| 关系类型 | 相关概念 | 关系描述 |
| ---- |---------| ---- |
| 基础 | 逻辑学 | 数学依赖逻辑推理 |
| 应用 | 物理学 | 数学为物理提供建模工具 |
| 应用 | 计算机科学 | 数学为计算提供理论基础 |
| 扩展 | 形式化方法 | 形式化是数学的精确表达 |

### 2. 集合论 (Set Theory)

#### 内涵

**中文定义**:
集合论是研究集合及其运算、关系和性质的数学分支，是现代数学的基础理论之一。

**English Definition**:
Set theory is the branch of mathematics that studies sets and their operations, relations, and properties. It is one of the foundational theories of modern mathematics.

#### 外延

- 朴素集合论 (Naive Set Theory)
- 公理集合论 (Axiomatic Set Theory)
  - ZFC公理系统
  - NBG公理系统
  - 其他公理系统

#### 属性

| 属性 | 描述 | 强度 |
| ---- |------| ---- |
| 基础性 | 为其他数学分支提供基础 | 极高 |
| 抽象性 | 研究抽象集合结构 | 极高 |
| 公理化 | 基于严格的公理系统 | 高 |

#### 关系

| 关系类型 | 相关概念 | 关系描述 |
| ---- |---------| ---- |
| 基础 | 数理逻辑 | 集合论依赖逻辑 |
| 基础 | 所有数学分支 | 集合论为数学提供基础 |
| 扩展 | 范畴论 | 范畴论可视为集合论的推广 |

---

### 3. 群论 (Group Theory)

#### 内涵

**中文定义**:
群论是研究群（满足结合律、有单位元、每个元素有逆元的代数结构）及其性质的数学分支。

**English Definition**:
Group theory is the branch of mathematics that studies groups (algebraic structures satisfying associativity, having an identity element, and where each element has an inverse) and their properties.

#### 外延

- 有限群 (Finite Groups)
- 无限群 (Infinite Groups)
- 交换群 (Abelian Groups)
- 非交换群 (Non-Abelian Groups)
- 李群 (Lie Groups)

#### 属性

| 属性 | 描述 | 强度 |
| ---- |------| ---- |
| 对称性 | 研究对称和变换 | 高 |
| 抽象性 | 抽象代数结构 | 高 |
| 应用性 | 应用于物理、化学等 | 中高 |

#### 关系

| 关系类型 | 相关概念 | 关系描述 |
| ---- |---------| ---- |
| 包含 | 抽象代数 | 群论是抽象代数的核心 |
| 扩展 | 环论 | 环是群的推广 |
| 应用 | 物理学 | 群论描述物理对称性 |

---

### 4. 微积分 (Calculus)

#### 内涵

**中文定义**:
微积分是研究函数的微分和积分及其应用的数学分支，是分析学的基础。

**English Definition**:
Calculus is the branch of mathematics that studies differentiation and integration of functions and their applications. It is the foundation of analysis.

#### 外延

- 一元微积分 (Single-Variable Calculus)
- 多元微积分 (Multivariable Calculus)
- 微分方程 (Differential Equations)
- 积分方程 (Integral Equations)

#### 属性

| 属性 | 描述 | 强度 |
| ---- |------| ---- |
| 实用性 | 广泛应用于科学和工程 | 极高 |
| 连续性 | 研究连续变化 | 高 |
| 极限性 | 基于极限概念 | 高 |

#### 关系

| 关系类型 | 相关概念 | 关系描述 |
| ---- |---------| ---- |
| 基础 | 实分析 | 微积分是实分析的基础 |
| 应用 | 物理学 | 微积分是物理建模的核心工具 |
| 扩展 | 泛函分析 | 泛函分析是微积分的无限维推广 |

---

### 5. 拓扑学 (Topology)

#### 内涵

**中文定义**:
拓扑学是研究空间在连续变形下保持不变性质的数学分支，是几何学的现代扩展。

**English Definition**:
Topology is the branch of mathematics that studies properties of spaces that remain invariant under continuous deformations. It is a modern extension of geometry.

#### 外延

- 一般拓扑学 (General Topology)
- 代数拓扑 (Algebraic Topology)
- 微分拓扑 (Differential Topology)
- 几何拓扑 (Geometric Topology)

#### 属性

| 属性 | 描述 | 强度 |
| ---- |------| ---- |
| 抽象性 | 研究抽象空间结构 | 极高 |
| 几何性 | 研究空间性质 | 高 |
| 应用性 | 应用于物理、计算机等 | 中高 |

#### 关系

| 关系类型 | 相关概念 | 关系描述 |
| ---- |---------| ---- |
| 扩展 | 几何学 | 拓扑学是几何学的现代扩展 |
| 基础 | 微分几何 | 微分几何依赖拓扑学 |
| 应用 | 物理学 | 拓扑学描述物理系统的拓扑性质 |

---

### 6. 概率论 (Probability Theory)

#### 内涵

**中文定义**:
概率论是研究随机现象和不确定性的数学分支，通过概率模型描述随机事件。

**English Definition**:
Probability theory is the branch of mathematics that studies random phenomena and uncertainty, describing random events through probability models.

#### 外延

- 古典概率 (Classical Probability)
- 测度论概率 (Measure-Theoretic Probability)
- 随机过程 (Stochastic Processes)
- 统计推断 (Statistical Inference)

#### 属性

| 属性 | 描述 | 强度 |
| ---- |------| ---- |
| 应用性 | 广泛应用于科学和工程 | 极高 |
| 不确定性 | 研究不确定现象 | 高 |
| 统计性 | 与统计学密切相关 | 极高 |

#### 关系
