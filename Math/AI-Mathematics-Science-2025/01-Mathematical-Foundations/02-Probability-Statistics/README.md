## 💡 核心数学工具

### 概率基础

```python
# 概率空间
(Ω, ℱ, P)  # 样本空间、σ-代数、概率测度

# 随机变量
X: Ω → ℝ  # 可测函数

# 期望
E[X] = ∫ X dP  # Lebesgue积分
```

### 常见分布

```python
# 离散分布
Bernoulli(p)
Binomial(n, p)
Poisson(λ)

# 连续分布
Uniform(a, b)
Normal(μ, σ²)
Exponential(λ)
```

### 重要定理

```python
# 大数定律
X̄ₙ → E[X]  (n → ∞)

# 中心极限定理
√n(X̄ₙ - μ) →ᵈ N(0, σ²)

# 贝叶斯公式
P(θ|D) = P(D|θ)P(θ) / P(D)
```
