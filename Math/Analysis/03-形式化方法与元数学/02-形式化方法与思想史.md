## 5. 形式化论证与代码实践 | Formal Reasoning & Code Practice

### 5.1 Lean定义公理化群结构

```lean
structure Group :=
  (carrier : Type)
  (op : carrier → carrier → carrier)
  (id : carrier)
  (inv : carrier → carrier)
  (assoc : ∀ x y z, op x (op y z) = op (op x y) z)
  (id_l : ∀ x, op id x = x)
  (id_r : ∀ x, op x id = x)
  (inv_l : ∀ x, op (inv x) x = id)
```
