## 2. 主要公理系统 | Main Axiomatic Systems

### 2.1 ZFC集合论 | ZFC Set Theory

- 理论基础：集合、公理化、无穷、公理独立性。
- 代表人物：策梅洛（Zermelo）、弗兰克尔（Fraenkel）、冯·诺依曼（von Neumann）
- 典型定理：选择公理、连续统假设。
- 形式化片段（Lean）：

```lean
-- ZFC集合的Lean定义（简化）
structure ZFCSet :=
  (elements : set ZFCSet)
```

### 2.2 皮亚诺公理 | Peano Arithmetic (PA)

- 理论基础：自然数、递归、公理化。
- 代表人物：皮亚诺（Peano）
- 典型定理：自然数归纳原理。

### 2.3 类型论 | Type Theory

- 理论基础：类型、构造性、依赖类型。
- 代表人物：马丁-勒夫（Martin-Löf）
- 典型定理：类型同构、构造性证明。

### 2.4 同伦类型论 | Homotopy Type Theory (HoTT)

- 理论基础：类型、同伦、等价、范畴。
- 代表人物：Voevodsky
- 典型定理：等价公理、单纯型集合。

## 概念定义 | Concept Definition

### 公理系统与证明论 | Axiomatic Systems & Proof Theory

- 中文定义：公理系统是由一组基本假设（公理）和推理规则组成的形式系统，用于系统化地建立数学理论。证明论是研究数学证明的结构、性质和可构造性的数学分支，关注证明的形式化、可计算性与一致性。
- English Definition: An axiomatic system is a formal system consisting of a set of basic assumptions (axioms) and inference rules, used to systematically construct mathematical theories. Proof theory is a branch of mathematical logic that studies the structure, properties, and constructibility of mathematical proofs, focusing on formalization, computability, and consistency.
- 国际标准/权威来源：
  - ISO 80000-2:2019
  - Stanford Encyclopedia of Philosophy: Axiomatic Method, Proof Theory
  - Encyclopedia of Mathematics: Axiomatic system, Proof theory
  - Wikipedia: Axiomatic system, Proof theory
- 相关批判与哲学反思：
  - 公理系统的选择决定了理论的基础与可证明性
  - 希尔伯特计划推动了形式化与元数学的发展，但哥德尔不完备定理揭示了公理系统的局限
  - 证明论不仅关注证明的存在性，还关注其构造性与算法实现
  - 直觉主义、构造主义等流派对经典公理系统提出批判，强调可构造性和有效性

### ZFC集合论 | ZFC Set Theory

- 中文定义：ZFC集合论是策梅洛-弗兰克尔集合论加上选择公理的形式系统，是现代数学的基础理论。
- English Definition: ZFC set theory is the Zermelo-Fraenkel set theory with the axiom of choice, a formal system that serves as the foundation of modern mathematics.
- 国际标准/权威来源：
  - ISO 80000-2:2019
  - Stanford Encyclopedia of Philosophy: ZFC Set Theory
  - Encyclopedia of Mathematics: ZFC Set Theory
  - Wikipedia: Zermelo–Fraenkel set theory
- 相关批判与哲学反思：
  - ZFC虽然提供了强大基础，但面临选择公理的争议
  - 集合论与类型论之间存在张力

### 皮亚诺公理 | Peano Axioms

- 中文定义：皮亚诺公理是描述自然数基本性质的形式化公理系统，包括归纳原理等。
- English Definition: Peano axioms are a formal axiomatic system describing the basic properties of natural numbers, including the principle of mathematical induction.
- 国际标准/权威来源：
  - ISO 80000-2:2019
  - Stanford Encyclopedia of Philosophy: Peano Axioms
  - Encyclopedia of Mathematics: Peano Axioms
  - Wikipedia: Peano axioms
- 相关批判与哲学反思：
  - 皮亚诺公理虽然简洁，但可能面临非标准模型问题
  - 归纳原理与构造性之间存在张力

### 类型论 | Type Theory

- 中文定义：类型论是一种形式化系统，将数学对象分类为不同类型，强调构造性和依赖类型。
- English Definition: Type theory is a formal system that classifies mathematical objects into different types, emphasizing constructivity and dependent types.
- 国际标准/权威来源：
  - ISO 80000-2:2019
  - Stanford Encyclopedia of Philosophy: Type Theory
  - Encyclopedia of Mathematics: Type Theory
  - Wikipedia: Type theory
- 相关批判与哲学反思：
  - 类型论虽然提供了构造性基础，但可能面临复杂性挑战
  - 类型与集合之间存在张力

### 同伦类型论 | Homotopy Type Theory

- 中文定义：同伦类型论是类型论与同伦论的结合，提供了数学基础的新视角。
- English Definition: Homotopy type theory is the combination of type theory and homotopy theory, providing a new perspective on mathematical foundations.
- 国际标准/权威来源：
  - ISO 80000-2:2019
  - Stanford Encyclopedia of Philosophy: Homotopy Type Theory
  - Encyclopedia of Mathematics: Homotopy Type Theory
  - Wikipedia: Homotopy type theory
- 相关批判与哲学反思：
  - 同伦类型论虽然提供了新视角，但可能面临理解困难
  - 几何直觉与形式化之间存在张力

### 证明论 | Proof Theory

- 中文定义：证明论是研究数学证明结构、性质和可构造性的数学分支。
- English Definition: Proof theory is a branch of mathematical logic that studies the structure, properties, and constructibility of mathematical proofs.
- 国际标准/权威来源：
  - ISO 80000-2:2019
  - Stanford Encyclopedia of Philosophy: Proof Theory
  - Encyclopedia of Mathematics: Proof Theory
  - Wikipedia: Proof theory
- 相关批判与哲学反思：
  - 证明论虽然提供了形式化工具，但可能面临可读性问题
  - 形式化与直觉之间存在张力

## 6. 理论历史与代表人物 | Theoretical History & Key Figures

### 6.1 策梅洛与集合论 | Zermelo & Set Theory

**代表人物与贡献：**

- 恩斯特·策梅洛（Ernst Zermelo, 1871-1953）
- 发展了集合论公理化
- 提出了选择公理

**原话引用：**
> "Every set can be well-ordered."
> "每个集合都可以良序化。" — 策梅洛

**集合论框架：**

```lean
-- ZFC集合论
def ZFC_set_theory : Set → Theory
| set => axiomatize set

-- 选择公理
def axiom_of_choice : Set → Choice
| set => well_order set

-- 替换公理
def axiom_of_replacement : Function → Set → Set
| function, set => replace function set
```

### 6.2 皮亚诺与算术公理 | Peano & Arithmetic Axioms

**代表人物与贡献：**

- 朱塞佩·皮亚诺（Giuseppe Peano, 1858-1932）
- 发展了算术公理化
- 提出了皮亚诺公理

**原话引用：**
> "Mathematics is the science of quantity."
> "数学是量的科学。" — 皮亚诺

**算术公理框架：**

```lean
-- 皮亚诺公理
def peano_axioms : NaturalNumber → Axioms
| number => axiomatize number

-- 归纳原理
def induction_principle : Property → NaturalNumber → Prop
| property, number => induct property number

-- 后继函数
def successor_function : NaturalNumber → NaturalNumber
| number => succ number
```

### 6.3 马丁-勒夫与类型论 | Martin-Löf & Type Theory

**代表人物与贡献：**

- 佩尔·马丁-勒夫（Per Martin-Löf, 1942-）
- 发展了直觉类型论
- 提出了依赖类型

**原话引用：**
> "A type is defined by prescribing how to form canonical objects of that type."
> "类型是通过规定如何形成该类型的规范对象来定义的。" — 马丁-勒夫

**类型论框架：**

```lean
-- 直觉类型论
def intuitionistic_type_theory : Type → Theory
| type => construct_type type

-- 依赖类型
def dependent_type : Type → Type → Type
| base_type, dependent_type => depend base_type dependent_type

-- 类型同构
def type_isomorphism : Type → Type → Isomorphism
| type1, type2 => isomorphic type1 type2
```

### 6.4 沃耶沃茨基与同伦类型论 | Voevodsky & Homotopy Type Theory

**代表人物与贡献：**

- 弗拉基米尔·沃耶沃茨基（Vladimir Voevodsky, 1966-2017）
- 发展了同伦类型论
- 提出了等价公理

**原话引用：**
> "Univalence is the principle that equivalent types are equal."
> "等价公理是等价类型相等的原理。" — 沃耶沃茨基

**同伦类型论框架：**

```lean
-- 同伦类型论
def homotopy_type_theory : Type → Theory
| type => homotopic_type type

-- 等价公理
def univalence_axiom : Type → Type → Equivalence
| type1, type2 => equivalent type1 type2

-- 单纯型集合
def simplicial_set : Type → Set
| type => simplicial type
```

### 6.5 根岑与证明论 | Gentzen & Proof Theory

**代表人物与贡献：**

- 格哈德·根岑（Gerhard Gentzen, 1909-1945）
- 发展了证明论
- 提出了切消定理

**原话引用：**
> "The consistency of arithmetic can be proved by transfinite induction."
> "算术的一致性可以通过超限归纳法证明。" — 根岑

**证明论框架：**

```lean
-- 证明论
def proof_theory : Proof → Theory
| proof => analyze_proof proof

-- 切消定理
def cut_elimination : Proof → Proof
| proof => eliminate_cuts proof

-- 自然演绎
def natural_deduction : Premise → Conclusion → Proof
| premise, conclusion => deduce premise conclusion
```
