## 概念定义 | Concept Definition

### 形式语言 | Formal Language

- 中文定义：形式语言是由符号组成的集合，按照特定规则构造的数学对象，是计算机科学和语言学的基础概念。
- English Definition: A formal language is a mathematical object consisting of a set of symbols constructed according to specific rules, a foundational concept in computer science and linguistics.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Formal Languages
  - Encyclopedia of Mathematics: Formal Language
  - Wikipedia: Formal language
- 相关批判与哲学反思：
  - 形式语言虽然提供了精确性，但可能忽视自然语言的复杂性
  - 形式语言与自然语言之间的关系需要进一步澄清

### 符号系统 | Symbolic System

- 中文定义：符号系统是由符号、规则和关系构成的抽象结构，用于表示和处理信息，是现代计算和认知科学的基础。
- English Definition: A symbolic system is an abstract structure consisting of symbols, rules, and relationships, used for representing and processing information, a foundation of modern computing and cognitive science.
- 国际标准/权威来源：
  - Stanford Encyclopedia of Philosophy: Symbolic Systems
  - Encyclopedia of Mathematics: Symbolic Systems
  - Wikipedia: Symbolic system
- 相关批判与哲学反思：
  - 符号系统虽然提供了结构化表示，但可能忽视语义的丰富性
  - 符号与意义之间的关系是哲学和认知科学的核心问题

### 语法 | Syntax

- 中文定义：语法是研究语言结构规则的学科，包括词法、句法和语法的形式化描述，是形式语言理论的核心。
- English Definition: Syntax is the study of the structural rules of language, including morphological, syntactic, and grammatical formal descriptions, a core of formal language theory.
- 国际标准/权威来源：
  - Stanford Encyclopedia of Philosophy: Syntax
  - Encyclopedia of Mathematics: Syntax
  - Wikipedia: Syntax
- 相关批判与哲学反思：
  - 语法虽然提供了结构规则，但可能忽视语义和语用的重要性
  - 形式语法与自然语法之间存在差异

### 语义 | Semantics

- 中文定义：语义是研究语言意义和解释的学科，包括指称、真值条件和语义关系，是形式语言理论的重要组成部分。
- English Definition: Semantics is the study of meaning and interpretation in language, including reference, truth conditions, and semantic relations, an important component of formal language theory.
- 国际标准/权威来源：
  - Stanford Encyclopedia of Philosophy: Semantics
  - Encyclopedia of Mathematics: Semantics
  - Wikipedia: Semantics
- 相关批判与哲学反思：
  - 语义虽然提供了意义解释，但语义本身需要进一步澄清
  - 形式语义与自然语义之间存在差异

## 3. 理论历史与代表人物 | Theoretical History & Key Figures

### 3.1 乔姆斯基与生成语法 | Chomsky & Generative Grammar

**代表人物与贡献：**

- 诺姆·乔姆斯基（Noam Chomsky, 1928-）
- 发展了生成语法理论
- 提出了Chomsky层级

**原话引用：**
> "Language is a process of free creation; its laws and principles are fixed, but the manner in which the principles of generation are used is free and infinitely varied."
> "语言是自由创造的过程；其法则和原理是固定的，但生成原理的使用方式是自由且无限变化的。" — 乔姆斯基

**生成语法形式化：**

```lean
-- 上下文无关文法
inductive ContextFreeGrammar : Type
| terminal : string → ContextFreeGrammar
| nonterminal : string → ContextFreeGrammar
| production : string → List ContextFreeGrammar → ContextFreeGrammar

-- Chomsky层级
def chomsky_hierarchy : Language → GrammarType
| L => if regular L then Regular
      else if context_free L then ContextFree
      else if context_sensitive L then ContextSensitive
      else RecursivelyEnumerable
```

### 3.2 图灵与可计算性 | Turing & Computability

**代表人物与贡献：**

- 艾伦·图灵（Alan Turing, 1912-1954）
- 发展了图灵机理论
- 提出了可计算性概念

**原话引用：**
> "A computer would deserve to be called intelligent if it could deceive a human into believing that it was human."
> "如果一台计算机能够欺骗人类相信它是人类，那么它就值得被称为智能的。" — 图灵

**图灵机形式化：**

```lean
-- 图灵机的基本结构
inductive TuringMachine : Type
| mk : (states : List State) → 
      (alphabet : List Symbol) → 
      (transition : State → Symbol → State × Symbol × Direction) →
      (initial : State) → 
      (accept : List State) → 
      TuringMachine

-- 可计算性定义
def computable : (string → string) → Prop
| f => exists (tm : TuringMachine),
      forall (input : string), 
        tm_computes tm input (f input)
```

### 3.3 蒙塔格与形式语义 | Montague & Formal Semantics

**代表人物与贡献：**

- 理查德·蒙塔格（Richard Montague, 1930-1971）
- 发展了形式语义学
- 建立了蒙塔格语法

**原话引用：**
> "There is in my opinion no important theoretical difference between natural languages and the artificial languages of logicians."
> "在我看来，自然语言和逻辑学家的人工语言之间没有重要的理论差异。" — 蒙塔格

**形式语义框架：**

