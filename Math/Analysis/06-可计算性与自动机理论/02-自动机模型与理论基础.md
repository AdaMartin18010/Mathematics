## 4. 形式化论证与代码实践 | Formal Reasoning & Code Practice

### 4.1 伪代码：有限自动机模拟

```python
def simulate_fa(fa, input_str):
    state = fa.start
    for c in input_str:
        state = fa.transition(state, c)
    return state in fa.accept_states
```

### 4.2 Lean定义有限自动机

```lean
structure FA :=
  (states : Type)
  (alphabet : Type)
  (start : states)
  (accept : states → bool)
  (trans : states → alphabet → states)
```
