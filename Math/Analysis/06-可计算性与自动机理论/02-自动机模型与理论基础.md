## 2. 主要自动机模型 | Main Automata Models

### 2.1 有限自动机 | Finite Automata (FA)

- 理论基础：状态、输入字母表、转移函数、初态、终态。
- 代表人物：拉宾（Rabin）、斯科特（Scott）
- 典型理论：正则语言、自动机等价。
- 形式化片段（Lean）：

```lean
-- 有限自动机的Lean定义（简化）
structure FA (Q Σ : Type) :=
  (start : Q)
  (accept : set Q)
  (trans : Q → Σ → Q)
```

### 2.2 堆栈自动机 | Pushdown Automata (PDA)

- 理论基础：状态、输入、栈操作。
- 代表人物：霍普克罗夫特（Hopcroft）、乌尔曼（Ullman）
- 典型理论：上下文无关语言。

### 2.3 图灵机 | Turing Machine (TM)

- 理论基础：无限带、读写头、状态转移。
- 代表人物：图灵（Turing）
- 典型理论：可计算性、判定性。
- 伪代码：


## 概念定义 | Concept Definition

### 自动机模型 | Automata Models

- 中文定义：自动机模型是用于描述计算过程的抽象数学结构，包括有限自动机、下推自动机、图灵机等。自动机理论基础研究这些模型的结构、运算能力及其与形式语言的关系，是理论计算机科学的核心内容。
- English Definition: Automata models are abstract mathematical structures used to describe computational processes, including finite automata, pushdown automata, Turing machines, etc. The foundations of automata theory study the structure, computational power of these models, and their relationship with formal languages, forming a core part of theoretical computer science.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Automata Theory
  - Encyclopedia of Mathematics: Automaton, Automata theory
  - Wikipedia: Automaton, Automata theory
- 相关批判与哲学反思：
  - 自动机模型的抽象性使其适用于理论分析，但与实际计算机系统存在差距
  - 不同自动机模型的运算能力界定了形式语言的分类边界
  - 自动机理论推动了编译原理、人工智能等领域的发展，但也面临复杂性与可实现性的挑战

### 有限自动机 | Finite Automaton

- 中文定义：有限自动机是最简单的自动机模型，具有有限个状态，能够识别正则语言，是形式语言理论的基础。
- English Definition: A finite automaton is the simplest automaton model with a finite number of states, capable of recognizing regular languages, forming the foundation of formal language theory.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Finite Automaton
  - Encyclopedia of Mathematics: Finite Automaton
  - Wikipedia: Finite-state machine
- 相关批判与哲学反思：
  - 有限自动机虽然简洁，但表达能力有限
  - 有限性与表达能力之间存在张力

### 下推自动机 | Pushdown Automaton

- 中文定义：下推自动机是具有栈结构的自动机模型，能够识别上下文无关语言，是语法分析的理论基础。
- English Definition: A pushdown automaton is an automaton model with a stack structure, capable of recognizing context-free languages, forming the theoretical foundation of syntax analysis.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Pushdown Automaton
  - Encyclopedia of Mathematics: Pushdown Automaton
  - Wikipedia: Pushdown automaton
- 相关批判与哲学反思：
  - 下推自动机虽然增强了表达能力，但增加了复杂性
  - 表达能力与复杂性之间存在张力

### 图灵机 | Turing Machine

- 中文定义：图灵机是具有无限带的自动机模型，能够模拟任何可计算函数，是计算理论的基础。
- English Definition: A Turing machine is an automaton model with an infinite tape, capable of simulating any computable function, forming the foundation of computational theory.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Turing Machine
  - Encyclopedia of Mathematics: Turing Machine
  - Wikipedia: Turing machine
- 相关批判与哲学反思：
  - 图灵机虽然提供了通用计算模型，但可能忽视物理限制
  - 抽象模型与物理实现之间存在张力

### 线性有界自动机 | Linear Bounded Automaton

- 中文定义：线性有界自动机是带长度受输入长度限制的图灵机，能够识别上下文相关语言。
- English Definition: A linear bounded automaton is a Turing machine with tape length bounded by input length, capable of recognizing context-sensitive languages.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Linear Bounded Automaton
  - Encyclopedia of Mathematics: Linear Bounded Automaton
  - Wikipedia: Linear bounded automaton
- 相关批判与哲学反思：
  - 线性有界自动机虽然限制了资源，但保持了表达能力
  - 资源限制与表达能力之间存在平衡

**有限自动机框架：**

```lean

structure FiniteAutomaton : Type :=
  (states : List State)
  (alphabet : List Symbol)
  (transitions : State → Symbol → State)
  (initial : State)
  (accepting : List State)

-- 自动机等价性
def automaton_equivalence : FiniteAutomaton → FiniteAutomaton → Prop
```
