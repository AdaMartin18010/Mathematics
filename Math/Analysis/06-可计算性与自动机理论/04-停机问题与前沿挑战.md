## 概念定义 | Concept Definition

### 停机问题与前沿挑战 | Halting Problem & Frontier Challenges

- 中文定义：停机问题是可计算性理论中的核心问题，询问是否存在一个算法能够判定任意程序在给定输入下是否会停机。不可判定性是指某些问题在理论上无法通过算法解决的性质。复杂系统是指由大量相互作用的组件组成的系统，其行为难以预测。
- English Definition: The halting problem is a central problem in computability theory, asking whether there exists an algorithm that can determine if any program will halt on a given input. Undecidability refers to the property that certain problems cannot be solved by algorithms in principle. Complex systems are systems composed of many interacting components whose behavior is difficult to predict.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015
  - Stanford Encyclopedia of Philosophy: Halting Problem, Undecidability
  - Encyclopedia of Mathematics: Halting problem, Undecidability
  - Wikipedia: Halting problem, Undecidability
- 相关批判与哲学反思：
  - 停机问题揭示了计算的本质限制，但也引发了关于计算本质的哲学思考
  - 不可判定性挑战了希尔伯特的形式化理想，但也推动了新的数学方法
  - 复杂系统理论挑战了还原论，但也面临预测和控制困难

### 停机问题 | Halting Problem

- 中文定义：停机问题是询问是否存在一个算法能够判定任意程序在给定输入下是否会停机的问题。
- English Definition: The halting problem asks whether there exists an algorithm that can determine if any program will halt on a given input.

  - 形式化与实用性之间存在张力

### 不可判定性 | Undecidability

- English Definition: Undecidability refers to the property that certain problems cannot be solved by algorithms in principle.

  - 不可判定性虽然揭示了理论限制，但可能过于悲观
  - 理论限制与实践需求之间存在张力

### 复杂系统 | Complex Systems

- 中文定义：复杂系统是指由大量相互作用的组件组成的系统，其行为难以预测。
- English Definition: Complex systems are systems composed of many interacting components whose behavior is difficult to predict.

### 递归可枚举 | Recursively Enumerable
