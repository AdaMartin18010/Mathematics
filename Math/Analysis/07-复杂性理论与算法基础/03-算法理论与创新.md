```lean
-- 分治法的Lean定义（简化）
def divide_and_conquer (problem : Type) : Type :=
  if is_simple problem then solve problem else
    let subproblems := decompose problem in
    combine (map solve subproblems)
```

## 概念定义 | Concept Definition

### 算法理论与创新 | Algorithmic Theory & Innovation

- 中文定义：算法理论研究算法的设计、分析与最优性，关注计算问题的高效求解方法。算法创新推动了计算机科学、人工智能、数据科学等领域的发展，是理论与应用结合的核心。
- English Definition: Algorithm theory studies the design, analysis, and optimality of algorithms, focusing on efficient methods for solving computational problems. Algorithmic innovation drives the development of computer science, artificial intelligence, data science, and other fields, serving as a core link between theory and application.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Algorithm
  - Encyclopedia of Mathematics: Algorithm
  - Wikipedia: Algorithm, Algorithmic efficiency
- 相关批判与哲学反思：
  - 算法理论揭示了理论与实际计算之间的差距
  - 算法创新推动了新技术与新应用，但也带来可解释性、公平性等伦理挑战
  - 算法最优性、可扩展性与实际约束之间的平衡是持续研究的难题

### 分治法 | Divide and Conquer

- 中文定义：分治法是一种算法设计策略，将问题分解为更小的子问题，递归求解后合并结果。
- English Definition: Divide and conquer is an algorithm design strategy that decomposes a problem into smaller subproblems, solves them recursively, and combines the results.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Divide and Conquer
  - Encyclopedia of Mathematics: Divide and Conquer
  - Wikipedia: Divide and conquer algorithm
- 相关批判与哲学反思：
  - 分治法虽然提供了清晰结构，但可能面临递归深度问题
  - 分解与合并之间存在张力

### 动态规划 | Dynamic Programming

- 中文定义：动态规划是一种通过将复杂问题分解为重叠子问题来求解的算法设计方法。
- English Definition: Dynamic programming is an algorithm design method that solves complex problems by breaking them down into overlapping subproblems.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Dynamic Programming
  - Encyclopedia of Mathematics: Dynamic Programming
  - Wikipedia: Dynamic programming
- 相关批判与哲学反思：
  - 动态规划虽然提高了效率，但可能面临空间复杂度问题
  - 时间与空间之间存在张力

### 贪心算法 | Greedy Algorithm

- 中文定义：贪心算法是一种在每一步选择中都采取当前状态下最好或最优的选择，从而希望导致结果是最好或最优的算法策略。
- English Definition: A greedy algorithm is a strategy that makes the locally optimal choice at each step with the hope of finding a global optimum.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Greedy Algorithm
  - Encyclopedia of Mathematics: Greedy Algorithm
  - Wikipedia: Greedy algorithm
- 相关批判与哲学反思：
  - 贪心算法虽然简单高效，但可能无法得到全局最优解
  - 局部最优与全局最优之间存在张力

### 启发式算法 | Heuristic Algorithm

- 中文定义：启发式算法是一种基于经验或直觉的算法，用于在合理时间内找到问题的近似解。
- English Definition: A heuristic algorithm is an algorithm based on experience or intuition, used to find approximate solutions to problems within reasonable time.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Heuristic Algorithm
  - Encyclopedia of Mathematics: Heuristic Algorithm
  - Wikipedia: Heuristic
- 相关批判与哲学反思：
  - 启发式算法虽然实用，但可能缺乏理论保证
  - 实用性与理论性之间存在张力

### 随机算法 | Randomized Algorithm

- 中文定义：随机算法是一种使用随机性作为算法设计核心要素的算法，通常用于提高效率或简化设计。
- English Definition: A randomized algorithm is an algorithm that uses randomness as a core element in its design, often used to improve efficiency or simplify design.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Randomized Algorithm
  - Encyclopedia of Mathematics: Randomized Algorithm
  - Wikipedia: Randomized algorithm
- 相关批判与哲学反思：
  - 随机算法虽然提高了效率，但可能面临确定性挑战
  - 随机性与确定性之间存在张力

### 近似算法 | Approximation Algorithm

- 中文定义：近似算法是一种在多项式时间内找到问题近似最优解的算法，通常用于NP难问题。
- English Definition: An approximation algorithm is an algorithm that finds approximate optimal solutions to problems in polynomial time, often used for NP-hard problems.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Approximation Algorithm
  - Encyclopedia of Mathematics: Approximation Algorithm
  - Wikipedia: Approximation algorithm
- 相关批判与哲学反思：
  - 近似算法虽然实用，但可能面临精度问题
  - 精度与效率之间存在张力

### 5.1 高德纳与算法分析 | Knuth & Algorithm Analysis

**代表人物与贡献：**

- 唐纳德·高德纳（Donald Knuth, 1938-）
- 发展了算法分析理论
- 编写了《计算机程序设计艺术》

**原话引用：**
> "Premature optimization is the root of all evil."
> "过早优化是万恶之源。" — 高德纳

- 提出了最优性原理

**原话引用：**
> "Dynamic programming is a method for solving complex problems by breaking them down into simpler subproblems."
> "动态规划是通过将复杂问题分解为更简单的子问题来求解的方法。" — 贝尔曼
