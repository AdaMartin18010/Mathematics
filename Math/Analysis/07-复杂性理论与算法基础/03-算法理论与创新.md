## 4. 典型案例与代码实践 | Typical Cases & Code Practice

### 4.1 伪代码：分治算法（归并排序）

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)
```

### 4.2 Lean定义递归算法结构

```lean
def merge_sort : list ℕ → list ℕ
| [] := []
| [x] := [x]
| (x::xs) := merge [x] (merge_sort xs)
```
