## 概念定义 | Concept Definition

### 自动定理证明 | Automated Theorem Proving

- 中文定义：自动定理证明是利用计算机程序自动验证或发现数学定理证明的方法。AI辅助证明则指人工智能系统在证明过程中为人类提供建议、验证或自动化部分推理。两者推动了数学、逻辑、计算机科学等领域的自动化与智能化。
- English Definition: Automated theorem proving refers to the use of computer programs to automatically verify or discover proofs of mathematical theorems. AI-assisted proving means that artificial intelligence systems provide suggestions, verification, or automate parts of the reasoning process during proofs. Both drive automation and intelligence in mathematics, logic, and computer science.
- 国际标准/权威来源：
  - ISO/IEC 2382:2015 (Information technology — Vocabulary)
  - Stanford Encyclopedia of Philosophy: Automated Theorem Proving
  - Encyclopedia of Mathematics: Automated theorem proving
  - Wikipedia: Automated theorem proving, Interactive theorem proving
- 相关批判与哲学反思：
  - 自动定理证明推动了形式化数学的发展，但也面临可解释性、可靠性等挑战
  - AI辅助证明提升了人机协作效率，但对证明的创造性与理解力提出新要求
  - 形式化证明与直觉、创造性之间的关系是持续研究的难题

### 归结原理 | Resolution Principle

- 中文定义：归结原理是一阶逻辑的完整推理规则，通过消解互补文字来证明定理，是自动定理证明的核心技术。
- English Definition: The resolution principle is a complete inference rule for first-order logic, proving theorems by resolving complementary literals, a core technique in automated theorem proving.
- 国际标准/权威来源：
  - Stanford Encyclopedia of Philosophy: Resolution Principle
  - Encyclopedia of Mathematics: Resolution Principle
  - Wikipedia: Resolution (logic)
- 相关批判与哲学反思：
  - 归结原理虽然提供了完整的推理规则，但可能产生指数级搜索空间
  - 自动化推理与人类直觉之间存在张力

### SMT求解 | SMT Solving

- 中文定义：SMT（Satisfiability Modulo Theories）求解是结合布尔可满足性和理论推理的自动推理技术，广泛应用于软件验证和形式化方法。
- English Definition: SMT (Satisfiability Modulo Theories) solving is an automated reasoning technique that combines Boolean satisfiability with theory reasoning, widely used in software verification and formal methods.
- 国际标准/权威来源：
  - Stanford Encyclopedia of Philosophy: SMT Solving
  - Encyclopedia of Mathematics: SMT Solving
  - Wikipedia: Satisfiability modulo theories
- 相关批判与哲学反思：
  - SMT求解虽然高效，但可能简化了复杂理论
  - 理论组合与可判定性之间存在张力

### 交互式定理证明 | Interactive Theorem Proving

- 中文定义：交互式定理证明是结合人类直觉和计算机严谨性的证明方法，通过人机协作完成复杂定理的证明。
- English Definition: Interactive theorem proving is a proof method that combines human intuition with computer rigor, completing complex theorem proofs through human-computer collaboration.
- 国际标准/权威来源：
  - Stanford Encyclopedia of Philosophy: Interactive Theorem Proving
  - Encyclopedia of Mathematics: Interactive Theorem Proving
  - Wikipedia: Interactive theorem proving
- 相关批判与哲学反思：
  - 交互式证明虽然结合了人机优势，但可能影响证明的自动化程度
  - 人机协作与完全自动化之间存在张力

### 形式化验证 | Formal Verification

- 中文定义：形式化验证是使用数学方法严格证明系统正确性的技术，包括模型检查、定理证明等方法。
- English Definition: Formal verification is the technique of using mathematical methods to strictly prove the correctness of systems, including model checking, theorem proving, and other methods.
- 国际标准/权威来源：
  - Stanford Encyclopedia of Philosophy: Formal Verification
  - Encyclopedia of Mathematics: Formal Verification
  - Wikipedia: Formal verification
- 相关批判与哲学反思：
  - 形式化验证虽然严谨，但可能面临规模和复杂性的挑战
  - 形式化方法与实际应用之间存在张力

**原话引用：**
> "Resolution is a complete inference rule for first-order logic."
> "归结是一阶逻辑的完整推理规则。" — 罗宾逊

**归结原理框架：**

```lean
-- 归结原理的基本定义
def resolution : Formula → Formula → Formula
| F1, F2 => resolve F1 F2

-- 归结证明
def resolution_proof : List Formula → Formula → Prop
| premises, conclusion => 
  exists (derivation : List Formula),
    valid_resolution_derivation derivation ∧ 
    last derivation = conclusion

-- 归结策略
def resolution_strategy : ResolutionStrategy
| strategy => optimize_resolution strategy
```

```coq
Inductive nat : Type :=
  | O : nat
  | S : nat → nat.

-- 归纳原理
Fixpoint plus (n m : nat) : nat :=
  match n with
  | O => m
  | S p => S (plus p m)
  end.

-- 定理证明
Theorem plus_O_n : forall n : nat, 0 + n = n.
Proof.
  intros n.
  simpl.
  reflexivity.
Qed.
```

**Lean框架：**

```lean
-- Lean中的类型定义
inductive Nat : Type
| zero : Nat
| succ : Nat → Nat

-- 函数定义
def add : Nat → Nat → Nat
| Nat.zero, m => m
| Nat.succ n, m => Nat.succ (add n m)

-- 定理证明
theorem add_zero : forall n : Nat, add n Nat.zero = n
| Nat.zero => rfl
| Nat.succ n => congr_arg Nat.succ (add_zero n)
```
