## 5. 形式化论证与代码实践 | Formal Reasoning & Code Practice

### 5.1 Lean自动证明加法交换律

```lean
theorem add_comm (a b : Nat) : a + b = b + a :=
Nat.recOn a
  (by simp)
  (fun n ih => by simp [ih])
```

### 5.2 Coq自动证明归纳法

```coq
Theorem plus_n_O : forall n : nat, n = n + 0.
Proof.
  induction n as [| n' IHn']. 
  - reflexivity.
  - simpl. rewrite -> IHn'. reflexivity.
Qed.
```
