## 4. 典型案例与代码实践 | Typical Cases & Code Practice

### 4.1 伪代码：中断恢复机制

```python
def resume_execution(plan):
    for step in plan.steps:
        if step.is_interrupted:
            resume(step)
        else:
            execute(step)
```
