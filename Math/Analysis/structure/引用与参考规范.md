# 引用与参考规范 | Citation & References

## 目的

- 统一全库引用与外链格式，确保可追溯与可维护。

## 基本格式

- 文内引用：作者（年份）：标题（版本/页码可选）
- 参考文献列表：按作者字母序；中文在前，英文在后
- 外部链接：使用带锚文本的 Markdown 链接，注明访问日期

## 统一细则

- 文内编号：优先使用作者-年份体（如 Mac Lane, 1998），必要时补充页码如 (Mac Lane, 1998, p. 42)
- DOI 规范：优先写标准 DOI 链接（例如 `https://doi.org/10.xxxx/xxxx`），避免裸 DOI 字符串
- 访问日期：统一格式 `访问日期: YYYY-MM-DD`
- 书名/期刊名：英文标题使用标题式首字母大写，期刊名斜体；中文维持原样
- 课程/网站：给出机构/站点名与稳定链接；如为课程，标注学期或版本
- 页内脚注：如需脚注，使用上标 [^1]，在文末“参考与版本信息”之前集中给出

### Wikipedia 对齐规则（新增）

- 条目选择：对齐英文 Wikipedia 主条目为基线，必要时补充各语种差异点。
- 结构对齐：至少匹配 Lead、Definition/History/Core concepts/Applications/See also/References 的分节结构。
- 链接策略：尽量链接到稳定锚点（如 `#Definition`），并在本库内同步创建对应锚点。
- 版本固化：每处 Wikipedia 链接均标注“访问日期”；涉及关键结论处应补充教科书/论文的权威二次来源。
- 术语一致：条目标题与小节名优先采用 Wikipedia 主流译名，差异译名于首次处以括号列出。
- 模板化注明：在页面“参考与版本信息”处新增“Wikipedia 对齐版本：条目名＋访问日期”。

### 大学课程与机构权威来源（新增）

- 大学课程：MIT OCW（18.x 系列）、Harvard（Math 1xx/2xx）、Princeton（MAT 3xx/5xx）、Cambridge（Part II/III）、Oxford（Algebra/Topology 模块）。
- 机构标准：AMS MSC（数学主题分类）、Springer 教材系列、Elsevier/AMS 期刊、Encyclopaedia of Mathematics、Wolfram MathWorld、nLab（前沿术语对照性参考）。
- 形式化证明：Lean mathlib、Coq、Isabelle/HOL、Agda 官方库与论文。
- 要求：课程链接需标注学校、课程号、学期/版本；机构资源需优先 DOI/稳定词条链接。

### 历史人物与学术权威库（新增）

- 人物权威对齐：MacTutor、MathSciNet（MR Author ID）、zbMATH Open、VIAF、ISNI、Wikipedia（带访问日期）。
- 条目要求：含生卒、主要贡献、关键论文（DOI）、奖项与机构；提供权威标识链接（如 VIAF/ISNI）。

### MSC 分类与标注（新增）

- 要求在页面底部“标签/分类”处给出主要 MSC 2020 分类码（如：54-XX 拓扑学总论；18-XX 范畴论）。
- 建议至多列 3 个主码与 3 个次码，避免过度标注。

### 校验清单（引用层）（新增）

- 至少 1 本教材/综述 + 1 篇论文/权威词条 + 1 条 Wikipedia（带访问日期）。
- 每处外链点击可达且非裸链接；DOI 优先。
- 本地页面均含“参考与版本信息”区块且已填写。

## 建议条目

- 书籍：作者. 书名. 出版社, 年份.
- 论文：作者. 题目. 期刊, 卷(期): 页码, 年份. DOI/URL.
- 教材/课程：课程名. 学校. URL（访问日期）。
- Web 资源：标题. 站点. URL（访问日期）。

## 模板（复制即用）

### 图书（Book）

```text
作者. 书名. 版本(可选). 出版地: 出版社, 年份. DOI/URL（访问日期: YYYY-MM-DD）
```

示例：

```text
Mac Lane, S. Categories for the Working Mathematician. 2nd ed. New York: Springer, 1998. https://doi.org/10.1007/978-1-4757-4721-8
```

### 学术论文（Article）

```text
作者. 题目. 期刊, 卷(期): 页码, 年份. DOI（访问日期: YYYY-MM-DD）
```

示例：

```text
Grothendieck, A. Sur quelques points d'algèbre homologique. Tôhoku Mathematical Journal, 9(2): 119–221, 1957. https://doi.org/10.2748/tmj/1178244839
```

### 预印本（Preprint）

```text
作者. 题目. arXiv:编号, 年份. URL（访问日期: YYYY-MM-DD）
```

### 教材/课程（Course）

```text
课程名. 学校/机构, 学期或版本. URL（访问日期: YYYY-MM-DD）
```

示例：

```text
MIT 18.901 Introduction to Topology. MIT OpenCourseWare, 2024 Fall. https://ocw.mit.edu/ (访问日期: 2025-01-09)
```

## 版本与更新

- 在页面末尾添加“参考与版本信息”小节：
  - 参考来源（至少3条）
  - 首次创建日期 / 最近更新日期
  - 维护人/责任域（可选）

## 示例（Markdown）

- 文内：[Hatcher(2002)](https://pi.math.cornell.edu/~hatcher/AT/AT.pdf) 指代代数拓扑教材.
- 参考：
  - Hatcher, A. Algebraic Topology. 2002. URL（访问日期: 2025-01-09）
  - Mac Lane, S. Categories for the Working Mathematician. Springer, 1998.

## 文内引用实践建议

- 首次出现给出全称与标准引文；多次出现可使用简称与年份
- 公式、定理后的出处建议以括号体补充，如 (Hatcher, 2002, Thm. 2.13)
- 与 Wikipedia 等可变页面对齐时务必标注访问日期

## 标注来源与缩写（新增）

- W=Wikipedia；EoM=Encyclopaedia of Mathematics；MW=MathWorld；MSC=AMS MSC；nLab=nLab；OCW=MIT OCW；HT=Harvard；PU=Princeton；CU=Cambridge；OU=Oxford。
- 用法：在术语或条目首次行末以内联代码标注主要来源，如 “拓扑空间 Topological space `W` `MSC`”。

## 脚注与注释

- 使用 Markdown 脚注语法 [^1]；脚注集中放在“参考与版本信息”前。

示例：

正文引用处：这是一个带脚注的句子[^note-example]。

脚注区：

[^note-example]: 这是脚注示例，包含补充说明与扩展链接。
