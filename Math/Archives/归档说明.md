# 项目文件归档说明

## 归档目的

本次归档旨在清理项目根目录，将与核心主题无关的临时性、重复性文档归档，提升项目的组织性和可维护性。

## 核心主题模块

项目的核心主题包括：

1. **Analysis模块**：数学认知理论体系
2. **Refactor模块**：数学知识重构（包括思脉体系）
3. **Lean模块**：Lean形式化验证
4. **Matter模块**：数学知识体系
5. **AI-Mathematics-Science-2025模块**：AI数学与科学
