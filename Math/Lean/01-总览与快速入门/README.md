## 📚 核心内容概览 | Core Content Overview

### Lean是什么？

Lean是一个基于依赖类型论的交互式定理证明助手和函数式编程语言，由微软研究院开发。它结合了强大的类型系统、自动化证明策略和丰富的数学库。

### 主要特色

- **类型安全**：基于依赖类型论的严格类型系统
- **交互式证明**：人机协作的定理证明环境
- **数学库丰富**：包含大量形式化的数学概念和定理
- **AI集成**：支持AI辅助的自动化证明
- **跨平台**：支持Windows、macOS、Linux

### 适用场景

- 数学定理的形式化证明
- 计算机程序的正确性验证
- 数学教育和研究
- 形式化方法研究
- AI与形式化证明的结合
