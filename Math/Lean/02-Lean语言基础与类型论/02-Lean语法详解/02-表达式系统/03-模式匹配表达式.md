# 03-模式匹配表达式 | Pattern Matching (Flat Entry)

## 覆盖范围

- 基本模式、守卫、嵌套模式、依赖模式

## 快速示例（Lean）

```lean
def length {α} : List α → Nat
  | [] => 0
  | _ :: xs => 1 + length xs

def head {α} : List α → Option α
  | [] => none
  | h :: _ => some h
```
