# 04-计算表达式 | Computation Expressions (Flat Entry)

## 覆盖范围

- 计算链（let/where）、条件分支、递归与等式链

## 快速示例（Lean）

```lean
def calc_chain (a b c : Nat) : Nat :=
  let s := a + b
  let t := s * c
  t - a

def conditional (n : Nat) : Nat :=
  if n > 0 then (n * 2) * (n * 2) else 0
```
