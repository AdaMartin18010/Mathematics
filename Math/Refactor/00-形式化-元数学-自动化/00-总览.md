## 核心内容特色 | Core Content Features

### 🎯 严格的概念定义

**形式化定义示例**：

```text
公理系统 = (Σ, F, A, R)
其中：
- Σ = 符号字母表
- F ⊆ Σ* = 合式公式集
- A ⊆ F = 公理集
- R ⊆ P(F) × F = 推理规则集
```

**定理表述示例**：

```text
哥德尔第一不完备定理：
设 T 是包含 PA 的一致递归公理化理论，则存在句子 G 使得：
1. T ⊬ G
2. T ⊬ ¬G
3. G 在标准模型中为真
```

### 🔬 完整的推理论证

**证明思路**：

1. **哥德尔编码**：将公式编码为自然数
2. **自指构造**：构造"我不可证明"的句子
3. **对角线论证**：利用自指性产生矛盾

**构造算法**：

```text
子集构造法（NFA转DFA）：
输入：NFA N = (Q, Σ, δ, q₀, F)
输出：DFA D = (Q', Σ, δ', q₀', F')

算法：
1. Q' = P(Q)
2. q₀' = ε-closure({q₀})
3. δ'(S, a) = ε-closure(∪_{q∈S} δ(q, a))
4. F' = {S | S ∩ F ≠ ∅}
```
