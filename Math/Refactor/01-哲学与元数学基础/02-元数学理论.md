## 1. 引言

元数学（Metamathematics）是研究数学理论本身的形式属性的数学分支。它不关心具体数学定理的内容，而是关心承载这些定理的形式系统具有哪些普遍性质。元数学为数学提供了严格的基础，帮助我们理解数学的本质和局限性。

元数学的主要分支包括：

- **证明论**：研究证明的结构和性质
- **模型论**：研究数学结构的语义解释
- **递归论**：研究可计算性和算法
- **集合论**：研究数学的基础结构

## 2. 形式系统理论

### 2.1 形式语言

#### 2.1.1 形式语言的定义

**定义：** 形式语言是由符号集和形成规则组成的系统。

**形式化表达：**
对于形式语言 $L$：
$$L = (\Sigma, \mathcal{F})$$
其中：

- $\Sigma$ 是符号集（字母表）
- $\mathcal{F}$ 是形成规则集

**符号集的分类：**

1. **逻辑符号：** $\land, \lor, \neg, \rightarrow, \leftrightarrow, \forall, \exists$
2. **非逻辑符号：** 常量、函数符号、谓词符号
3. **辅助符号：** 括号、逗号等

#### 2.1.2 形成规则

**定义：** 形成规则定义了哪些符号串是合式公式（well-formed formulas）。

**基本形成规则：**

1. **原子公式：** 如果 $P$ 是 $n$ 元谓词，$t_1, \ldots, t_n$ 是项，则 $P(t_1, \ldots, t_n)$ 是公式
2. **逻辑连接：** 如果 $\phi, \psi$ 是公式，则 $\neg\phi, \phi\land\psi, \phi\lor\psi, \phi\rightarrow\psi$ 是公式
3. **量词：** 如果 $\phi$ 是公式，$x$ 是变量，则 $\forall x\phi, \exists x\phi$ 是公式

**形式化表达：**
$$\text{If } \phi, \psi \in \text{Formulas}, \text{ then } \neg\phi, \phi\land\psi, \phi\lor\psi, \phi\rightarrow\psi \in \text{Formulas}$$

### 2.2 公理系统

#### 2.2.1 公理的概念

**定义：** 公理是形式系统中不证自明的基本命题。

**公理的类型：**

1. **逻辑公理：** 普遍有效的逻辑真理
2. **非逻辑公理：** 特定理论的基本假设

**逻辑公理示例（命题逻辑）：**

1. $\phi \rightarrow (\psi \rightarrow \phi)$
2. $(\phi \rightarrow (\psi \rightarrow \chi)) \rightarrow ((\phi \rightarrow \psi) \rightarrow (\phi \rightarrow \chi))$
3. $(\neg\phi \rightarrow \neg\psi) \rightarrow (\psi \rightarrow \phi)$

#### 2.2.2 公理系统的性质

**一致性（Consistency）：**
$$\text{Consistent}(\mathcal{A}) \iff \nexists \phi: \mathcal{A} \vdash \phi \text{ and } \mathcal{A} \vdash \neg\phi$$

**独立性（Independence）：**
$$\text{Independent}(\mathcal{A}) \iff \forall \alpha \in \mathcal{A}: \mathcal{A} \setminus \{\alpha\} \nvdash \alpha$$

**完备性（Completeness）：**
$$\text{Complete}(\mathcal{A}) \iff \forall \phi: \mathcal{A} \vdash \phi \text{ or } \mathcal{A} \vdash \neg\phi$$

### 2.3 推理规则

#### 2.3.1 基本推理规则

**分离规则（Modus Ponens）：**
$$\frac{\phi \rightarrow \psi \quad \phi}{\psi}$$

**普遍化规则（Universal Generalization）：**
$$\frac{\phi}{\forall x \phi} \text{ (if } x \text{ is not free in any premise)}$$

**存在量词引入（Existential Introduction）：**
$$\frac{\phi[t/x]}{\exists x \phi}$$

#### 2.3.2 推理规则的性质

**可靠性（Soundness）：**
$$\text{Sound}(\mathcal{R}) \iff \text{If } \Gamma \vdash_{\mathcal{R}} \phi, \text{ then } \Gamma \models \phi$$

**完备性（Completeness）：**
$$\text{Complete}(\mathcal{R}) \iff \text{If } \Gamma \models \phi, \text{ then } \Gamma \vdash_{\mathcal{R}} \phi$$

### 2.4 形式系统的性质

#### 2.4.1 可判定性

**定义：** 形式系统是可判定的，如果存在算法可以判定任意公式是否为定理。

**形式化表达：**
$$\text{Decidable}(\mathcal{S}) \iff \exists \text{ algorithm } A: \forall \phi, A(\phi) = 1 \iff \mathcal{S} \vdash \phi$$

**示例：**

- 命题逻辑是可判定的
- 一阶谓词逻辑是不可判定的（丘奇定理）

#### 2.4.2 可公理化性

**定义：** 形式系统是可公理化的，如果存在有限或递归的公理集。

**形式化表达：**
$$\text{Axiomatizable}(\mathcal{S}) \iff \exists \text{ recursive } \mathcal{A}: \text{Th}(\mathcal{A}) = \text{Th}(\mathcal{S})$$

## 3. 证明论

### 3.1 证明的概念

#### 3.1.1 证明的定义

**定义：** 证明是从公理和假设出发，通过推理规则得到结论的有限序列。

**形式化表达：**
$$\text{Proof}(\phi) = \langle \phi_1, \phi_2, \ldots, \phi_n \rangle$$
其中：

- $\phi_n = \phi$
- 每个 $\phi_i$ 要么是公理，要么通过推理规则从前面的公式得到

#### 3.1.2 证明的结构

**证明的组成部分：**

1. **假设：** 证明的起点
2. **推理步骤：** 应用推理规则
3. **结论：** 证明的目标

**证明的类型：**

1. **直接证明：** 从假设直接推导结论
2. **反证法：** 假设结论的否定，导出矛盾
3. **构造性证明：** 构造满足条件的对象

### 3.2 证明系统

#### 3.2.1 自然演绎系统

**特征：** 使用引入和消除规则。

**示例（命题逻辑）：**

**合取引入：**
$$\frac{\phi \quad \psi}{\phi \land \psi}$$

**合取消除：**
$$\frac{\phi \land \psi}{\phi} \quad \frac{\phi \land \psi}{\psi}$$

**蕴含引入：**
$$\frac{[\phi] \quad \psi}{\phi \rightarrow \psi}$$

#### 3.2.2 希尔伯特系统

**特征：** 使用公理和少量推理规则。

**基本推理规则：**
$$\frac{\phi \rightarrow \psi \quad \phi}{\psi} \text{ (Modus Ponens)}$$

### 3.3 证明的复杂性

#### 3.3.1 证明长度

**定义：** 证明的长度是证明中公式的数量。

**形式化表达：**
$$|\text{Proof}(\phi)| = n \text{ where } \text{Proof}(\phi) = \langle \phi_1, \ldots, \phi_n \rangle$$

**复杂性理论：**

- 某些定理的证明长度可能非常大
- 存在证明长度与定理复杂性的关系

#### 3.3.2 证明深度

**定义：** 证明的深度是证明树的最大深度。

**形式化表达：**
$$\text{Depth}(\text{Proof}(\phi)) = \max\{\text{depth of subproofs}\}$$

### 3.4 构造性证明

#### 3.4.1 构造性的含义

**定义：** 构造性证明不仅证明存在性，还提供构造方法。

**形式化表达：**
$$\text{Constructive}(\text{Proof}(\exists x \phi(x))) \iff \text{Proof provides } t \text{ such that } \phi(t)$$

#### 3.4.2 构造性数学

**直觉主义逻辑：** 只承认构造性证明。

**主要特征：**

- 否定双重否定律：$\neg\neg\phi \not\equiv \phi$
- 否定排中律：$\phi \lor \neg\phi$ 不总是成立
- 强调构造性存在

## 4. 模型论

### 4.1 模型的概念

#### 4.1.1 模型的定义

**定义：** 模型是形式语言的语义解释。

**形式化表达：**
$$\mathcal{M} = (M, I)$$
其中：

- $M$ 是论域（domain）
- $I$ 是解释函数（interpretation function）

**解释函数的作用：**

- 将常量映射到论域中的元素
- 将函数符号映射到论域上的函数
- 将谓词符号映射到论域上的关系

#### 4.1.2 满足关系

**定义：** 模型满足公式，记作 $\mathcal{M} \models \phi$。

**递归定义：**

1. **原子公式：** $\mathcal{M} \models P(t_1, \ldots, t_n) \iff (t_1^{\mathcal{M}}, \ldots, t_n^{\mathcal{M}}) \in P^{\mathcal{M}}$
2. **逻辑连接：** $\mathcal{M} \models \phi \land \psi \iff \mathcal{M} \models \phi \text{ and } \mathcal{M} \models \psi$
3. **量词：** $\mathcal{M} \models \forall x \phi \iff \text{for all } a \in M, \mathcal{M} \models \phi[a/x]$

### 4.2 模型的存在性

#### 4.2.1 紧致性定理

**定理：** 如果理论 $T$ 的每个有限子集都有模型，则 $T$ 本身有模型。

**形式化表达：**
$$\text{If } \forall \text{ finite } T' \subseteq T: \exists \mathcal{M}: \mathcal{M} \models T', \text{ then } \exists \mathcal{M}: \mathcal{M} \models T$$

**应用：**

- 证明非标准模型的存在
- 证明无穷基数模型的存在

#### 4.2.2 勒文海姆-斯科伦定理

**向下勒文海姆-斯科伦定理：** 如果可数理论有无限模型，则它有任意大基数的模型。

**向上勒文海姆-斯科伦定理：** 如果理论有无限模型，则它有任意大基数的模型。

**形式化表达：**
$$\text{If } T \text{ has infinite model, then } \forall \kappa \geq |L|: T \text{ has model of cardinality } \kappa$$

### 4.3 模型论的基本定理

#### 4.3.1 完全性定理

**哥德尔完全性定理：** 一阶逻辑是语义完全的。

**形式化表达：**
$$\Gamma \models \phi \iff \Gamma \vdash \phi$$

**意义：**

- 语义有效性等价于语法可证性
- 为形式系统提供了语义基础

#### 4.3.2 模型论的基本概念

**初等等价：** 两个模型初等等价，如果它们满足相同的句子。

**形式化表达：**
$$\mathcal{M} \equiv \mathcal{N} \iff \text{Th}(\mathcal{M}) = \text{Th}(\mathcal{N})$$

**初等嵌入：** 一个模型初等嵌入到另一个模型，如果存在保持所有公式的嵌入。

**形式化表达：**
$$f: \mathcal{M} \prec \mathcal{N} \iff \forall \phi, \mathcal{M} \models \phi \iff \mathcal{N} \models \phi$$

### 4.4 模型论的应用

#### 4.4.1 代数中的应用

**代数闭域：** 使用模型论方法研究代数闭域的性质。

**示例：** 代数闭域的勒文海姆-斯科伦定理应用。

#### 4.4.2 数论中的应用

**非标准分析：** 使用超实数模型研究分析学。

**形式化表达：**
$$\mathbb{R}^* \text{ is non-standard model of } \mathbb{R}$$

## 5. 递归论与可计算性

### 5.1 递归函数

#### 5.1.1 原始递归函数

**定义：** 从基本函数出发，通过复合和原始递归得到的函数。

**基本函数：**

1. **零函数：** $Z(x) = 0$
2. **后继函数：** $S(x) = x + 1$
3. **投影函数：** $P_i^n(x_1, \ldots, x_n) = x_i$

**运算：**

1. **复合：** $h(x) = f(g_1(x), \ldots, g_m(x))$
2. **原始递归：** $h(x, 0) = f(x), h(x, y+1) = g(x, y, h(x, y))$

#### 5.1.2 一般递归函数

**定义：** 包含原始递归函数和 $\mu$ 递归的最小函数类。

**$\mu$ 递归：**
$$\mu y[f(x, y) = 0] = \text{least } y \text{ such that } f(x, y) = 0$$

### 5.2 可计算性理论

#### 5.2.1 可计算性的定义

**定义：** 函数是可计算的，如果存在算法可以计算它。

**形式化表达：**
$$\text{Computable}(f) \iff \exists \text{ algorithm } A: \forall x, A(x) = f(x)$$

#### 5.2.2 可计算性的等价性

**丘奇-图灵论题：** 所有可计算函数都是图灵可计算的。

**等价形式：**

- 图灵可计算
- 递归函数
- $\lambda$ 可定义
- 寄存器机器可计算

### 5.3 图灵机

#### 5.3.1 图灵机的定义

**定义：** 图灵机是抽象的计算模型。

**组成部分：**

- 无限长的纸带
- 读写头
- 有限状态控制器
- 转移函数

**形式化表达：**
$$M = (Q, \Sigma, \Gamma, \delta, q_0, F)$$
其中：

- $Q$ 是状态集
- $\Sigma$ 是输入字母表
- $\Gamma$ 是纸带字母表
- $\delta$ 是转移函数
- $q_0$ 是初始状态
- $F$ 是接受状态集

#### 5.3.2 图灵机的计算

**配置：** $(q, w, i)$ 表示状态、纸带内容、读写头位置。

**转移：** $\delta(q, a) = (q', b, D)$ 表示在状态 $q$ 读 $a$ 时，写入 $b$，移动到方向 $D$，进入状态 $q'$。

### 5.4 丘奇-图灵论题

#### 5.4.1 论题的内容

**丘奇-图灵论题：** 任何可计算的函数都是图灵可计算的。

**形式化表达：**
$$\text{Computable}(f) \iff \text{Turing-computable}(f)$$

#### 5.4.2 论题的意义

**哲学意义：** 为可计算性提供了精确的数学定义。

**实践意义：** 为计算机科学提供了理论基础。

**局限性：** 这是一个经验性假设，无法严格证明。

## 6. 哥德尔不完备性定理

### 6.1 第一不完备性定理

#### 6.1.1 定理的表述

**哥德尔第一不完备性定理：** 任何包含算术的一致形式系统都是不完备的。

**形式化表达：**
$$\text{If } T \text{ is consistent and contains arithmetic, then } T \text{ is incomplete}$$

#### 6.1.2 证明的核心思想

**自指构造：** 构造一个句子 $G$，它说"$G$ 在 $T$ 中不可证"。

**形式化表达：**
$$G \leftrightarrow \neg\text{Prov}_T(\ulcorner G \urcorner)$$

**关键步骤：**

1. 构造自指句子
2. 证明 $T \nvdash G$
3. 证明 $T \nvdash \neg G$

### 6.2 第二不完备性定理

#### 6.2.1 定理的表述

**哥德尔第二不完备性定理：** 任何包含算术的一致形式系统都不能证明自己的一致性。

**形式化表达：**
$$\text{If } T \text{ is consistent and contains arithmetic, then } T \nvdash \text{Con}(T)$$

#### 6.2.2 定理的意义

**认识论意义：** 数学系统无法完全证明自己的可靠性。

**哲学意义：** 揭示了数学知识的局限性。

**实践意义：** 影响数学基础的研究方向。

### 6.3 不完备性定理的意义

#### 6.3.1 对数学基础的影响

**形式主义的挑战：** 形式主义无法完全实现其目标。

**直觉主义的支持：** 为直觉主义提供了支持。

**构造性数学：** 推动了构造性数学的发展。

#### 6.3.2 对计算机科学的影响

**程序验证：** 影响程序验证的理论基础。

**人工智能：** 影响人工智能的理论基础。

**复杂性理论：** 与复杂性理论有深刻联系。
