## 2. 表格对比

| 主题         | 形式化系统           | 元数学理论         |
|--------------|----------------------|--------------------|
| 研究对象     | 公理、符号、推理规则 | 形式系统的性质与极限 |

| 应用         | 证明助手、编程语言   | 自动证明、复杂性理论 |

## 3. 代码与形式化示例

### Lean 示例

```lean
axiom PeanoZero : ℕ
axiom PeanoSucc : ℕ → ℕ
-- 皮亚诺公理部分形式化
```

### Haskell 示例

```haskell
data Nat = Zero | Succ Nat
-- 皮亚诺自然数的Haskell定义
```

### Rust 示例

```rust
enum Nat {
    Zero,
    Succ(Box<Nat>),
}
// 皮亚诺自然数的Rust定义
```
