## 3.6.2.4 AI与自动化形式化

- 现代证明助手（如Lean、Coq）可形式化复杂数学理论。
- 典型Lean代码示例：

```lean
theorem add_zero (n : ℕ) : n + 0 = n :=
begin
  induction n,
  { refl },
  { simp [nat.succ_add] }
end
```

- 典型Coq代码示例：

```coq
Theorem add_0_r : forall n : nat, n + 0 = n.
Proof.
  induction n as [| n' IHn].
  - reflexivity.
  - simpl. rewrite -> IHn. reflexivity.
Qed.
```
