## 1. 概述

数学基础与逻辑是现代数学的理论基石，为所有数学分支提供形式化基础和推理工具。本主题涵盖：

- **集合论基础** - 现代数学的通用语言
- **逻辑系统** - 推理和证明的形式化工具
- **证明论** - 形式化证明的理论基础
- **模型论** - 语义解释的理论框架
- **递归论** - 可计算性理论

### 1.1 历史背景

数学基础与逻辑的发展经历了几个重要阶段：

1. **古典逻辑** (亚里士多德) - 三段论推理
2. **符号逻辑** (布尔、德摩根) - 代数化逻辑
3. **形式化逻辑** (弗雷格、罗素) - 数学的形式化
4. **现代逻辑** (哥德尔、图灵) - 不完备性和可计算性

### 1.2 核心目标

- **形式化数学推理** - 将直观论证转化为严格的形式推导
- **建立数学基础** - 为所有数学分支提供公理化基础
- **探索可计算性** - 界定算法可解问题的边界
- **理解数学局限性** - 揭示形式系统的固有局限

## 2. 理论基础

### 2.1 公理化方法

公理化方法是现代数学组织知识的基本手段：

#### 2.1.1 公理系统的基本要素

- **形式语言** - 精确的符号系统
- **公理集合** - 基本假设
- **推理规则** - 逻辑推导规则
- **定理** - 从公理推导出的结论

#### 2.1.2 公理系统的性质

- **一致性 (Consistency)** - 不能推导出矛盾
- **完备性 (Completeness)** - 所有真命题都可证
- **独立性 (Independence)** - 公理不冗余
- **可判定性 (Decidability)** - 存在算法判定命题真假

### 2.2 形式化系统

#### 2.2.1 形式语言

形式语言由以下要素构成：

- **逻辑符号** - $\neg, \land, \lor, \rightarrow, \leftrightarrow, \forall, \exists$
- **非逻辑符号** - 常量、函数符号、关系符号
- **变元** - 个体变元、函数变元、谓词变元
- **合式公式** - 按语法规则构造的表达式

#### 2.2.2 语义解释

语义解释为形式语言提供意义：

- **结构** - 包含论域、函数、关系的数学对象
- **赋值** - 将变元映射到论域中的元素
- **满足关系** - 结构在赋值下满足公式的条件
- **真值** - 公式在结构中的真假

### 2.3 证明理论

#### 2.3.1 形式证明

形式证明是公式的有限序列：

- **公理** - 证明的起点
- **推理规则** - 从前提推导结论的规则
- **定理** - 证明的结论
- **可证性** - 存在证明的公式

#### 2.3.2 证明系统

主要的证明系统包括：

- **希尔伯特系统** - 公理化方法
- **自然演绎** - 直观的推理规则
- **相继式演算** - 结构化的证明系统

## 3. 核心分支

### 3.1 集合论基础

集合论为现代数学提供基础框架：

#### 3.1.1 基本概念

- **集合** - 对象的聚集
- **元素** - 集合的成员
- **子集** - 包含关系
- **幂集** - 所有子集的集合

#### 3.1.2 公理系统

ZFC公理系统包括：

- **外延公理** - 集合由其元素唯一确定
- **空集公理** - 存在空集
- **配对公理** - 可构造二元集
- **并集公理** - 可构造并集
- **幂集公理** - 可构造幂集
- **无穷公理** - 存在无穷集
- **替换公理** - 函数下的像仍是集合
- **正则公理** - 集合的良基性
- **选择公理** - 选择函数的存在性

#### 3.1.3 基数理论

- **等势** - 集合间的一一对应
- **基数** - 集合的"大小"
- **可数集** - 与自然数集等势
- **连续统假设** - 关于实数集基数的猜想

**详细内容**：[见 1.1 集合论基础](../01-集合论基础/00-集合论基础总览.md)

### 3.2 逻辑系统

#### 3.2.1 命题逻辑

命题逻辑研究简单命题的逻辑关系：

- **命题变元** - 基本命题符号
- **逻辑联结词** - $\neg, \land, \lor, \rightarrow, \leftrightarrow$
- **真值表** - 命题的真值定义
- **重言式** - 永真命题
- **矛盾式** - 永假命题

#### 3.2.2 谓词逻辑

谓词逻辑扩展命题逻辑，包含量词：

- **个体变元** - 表示对象的变元
- **谓词符号** - 表示性质和关系
- **函数符号** - 表示运算
- **量词** - $\forall$ (全称), $\exists$ (存在)
- **自由变元** - 不受量词约束的变元

#### 3.2.3 模态逻辑

模态逻辑研究必然性和可能性：

- **模态算子** - $\Box$ (必然), $\Diamond$ (可能)
- **可能世界** - 模态语义的基本概念
- **可及关系** - 世界间的可达性
- **模态系统** - K, T, S4, S5等

**详细内容**：[见 2.1 逻辑系统](../02-逻辑系统/00-逻辑系统总览.md)

### 3.3 证明论

#### 3.3.1 形式证明系统

- **公理化系统** - 基于公理的证明
- **自然演绎** - 直观的推理规则
- **相继式演算** - 结构化的证明
- **证明树** - 证明的树状表示

#### 3.3.2 证明论的主要结果

- **切消定理** - 证明的规范化
- **一致性证明** - 系统的无矛盾性
- **保守性** - 扩展系统的性质保持
- **插值定理** - 公式间的逻辑关系

**详细内容**：[见 3.1 证明论](../03-证明论/00-证明论总览.md)

### 3.4 模型论

#### 3.4.1 基本概念

- **结构** - 数学对象的集合
- **解释** - 符号到对象的映射
- **满足关系** - 结构满足公式的条件
- **模型** - 使理论为真的结构

#### 3.4.2 主要定理

- **哥德尔完备性定理** - 语法与语义的等价性
- **紧致性定理** - 有限满足与可满足的关系
- **Löwenheim-Skolem定理** - 模型的大小性质
- **量词消去** - 公式的简化

**详细内容**：[见 4.1 模型论](../04-模型论/00-模型论总览.md)

### 3.5 递归论

#### 3.5.1 可计算性

- **算法** - 机械的计算过程
- **可计算函数** - 存在算法计算的函数
- **图灵机** - 计算的形式模型
- **丘奇-图灵论题** - 可计算性的哲学假设

#### 3.5.2 递归函数

- **原始递归函数** - 基本的递归函数类
- **一般递归函数** - 所有可计算函数
- **递归可枚举集** - 可枚举但不一定可判定的集合
- **停机问题** - 不可判定的经典问题

**详细内容**：[见 5.1 递归论](../05-递归论/00-递归论总览.md)
