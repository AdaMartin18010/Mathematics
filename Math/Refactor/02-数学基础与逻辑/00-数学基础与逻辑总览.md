## 1. 数理逻辑基础

### 1.1 命题逻辑 (Propositional Logic)

#### 1.1.1 基本概念

**定义**：命题逻辑研究命题之间的逻辑关系

**核心概念**：

- **命题 (Proposition)**：能判断真假的陈述句
- **命题变元 (Propositional Variables)**：用符号表示命题，如 P, Q, R
- **逻辑联结词 (Logical Connectives)**：连接命题的符号

#### 1.1.2 逻辑联结词

**基本联结词**：

| 符号 | 名称 | 含义 | 真值表 |
|------|------|------|--------|
| ¬, ~ | 否定 | 非P | P | ¬P |
|      |      |     | T | F  |
|      |      |     | F | T  |
| ∧, · | 合取 | P且Q | P | Q | P∧Q |
|      |      |      | T | T | T  |
|      |      |      | T | F | F  |
|      |      |      | F | T | F  |
|      |      |      | F | F | F  |
| ∨, + | 析取 | P或Q | P | Q | P∨Q |
|      |      |      | T | T | T  |
|      |      |      | T | F | T  |
|      |      |      | F | T | T  |
|      |      |      | F | F | F  |
| →, ⇒ | 蕴含 | 如果P则Q | P | Q | P→Q |
|       |      |          | T | T | T  |
|       |      |          | T | F | F  |
|       |      |          | F | T | T  |
|       |      |          | F | F | T  |
| ↔, ⇔ | 等价 | P当且仅当Q | P | Q | P↔Q |
|       |      |            | T | T | T  |
|       |      |            | T | F | F  |
|       |      |            | F | T | F  |
|       |      |            | F | F | T  |

#### 1.1.3 重要概念

**重言式 (Tautology)**：在所有真值指派下都为真的命题公式

**例子**：

- 排中律：P ∨ ¬P
- 矛盾律：¬(P ∧ ¬P)
- 德摩根定律：¬(P ∧ Q) ↔ (¬P ∨ ¬Q)

**矛盾式 (Contradiction)**：在所有真值指派下都为假的命题公式

**例子**：

- P ∧ ¬P

**可满足式 (Satisfiable)**：至少在一个真值指派下为真的命题公式

#### 1.1.4 推理规则

**基本推理规则**：

1. **肯定前件 (Modus Ponens)**：
   - 前提：P, P → Q
   - 结论：Q

2. **否定后件 (Modus Tollens)**：
   - 前提：P → Q, ¬Q
   - 结论：¬P

3. **假言三段论 (Hypothetical Syllogism)**：
   - 前提：P → Q, Q → R
   - 结论：P → R

4. **析取三段论 (Disjunctive Syllogism)**：
   - 前提：P ∨ Q, ¬P
   - 结论：Q

### 1.2 谓词逻辑 (Predicate Logic)

#### 1.2.1 基本概念

**定义**：在命题逻辑基础上引入个体、谓词和量词

**核心概念**：

- **个体 (Individuals)**：论域中的对象
- **谓词 (Predicates)**：表示个体性质或关系的符号
- **量词 (Quantifiers)**：表示数量的符号

#### 1.2.2 量词

**全称量词 (Universal Quantifier)**：∀

- ∀x P(x) 表示"对所有x，P(x)为真"

**存在量词 (Existential Quantifier)**：∃

- ∃x P(x) 表示"存在x，使得P(x)为真"

#### 1.2.3 重要等价关系

**量词否定**：

- ¬∀x P(x) ↔ ∃x ¬P(x)
- ¬∃x P(x) ↔ ∀x ¬P(x)

**量词分配**：

- ∀x (P(x) ∧ Q(x)) ↔ ∀x P(x) ∧ ∀x Q(x)
- ∃x (P(x) ∨ Q(x)) ↔ ∃x P(x) ∨ ∃x Q(x)

### 1.3 证明理论

#### 1.3.1 证明方法

**直接证明**：从前提直接推导结论

**反证法**：假设结论的否定，导出矛盾

**数学归纳法**：

- 基础步骤：证明P(0)为真
- 归纳步骤：证明∀n (P(n) → P(n+1))
- 结论：∀n P(n)

## 2. 集合论基础

### 2.1 基本概念

#### 2.1.1 集合的定义

**朴素定义**：集合是一些对象的聚集

**公理化定义**：通过ZFC公理系统严格定义

#### 2.1.2 基本运算

**并集 (Union)**：A ∪ B = {x | x ∈ A ∨ x ∈ B}

**交集 (Intersection)**：A ∩ B = {x | x ∈ A ∧ x ∈ B}

**差集 (Difference)**：A \ B = {x | x ∈ A ∧ x ∉ B}

**补集 (Complement)**：A' = {x | x ∉ A}

**对称差 (Symmetric Difference)**：A △ B = (A \ B) ∪ (B \ A)

### 2.2 ZFC公理系统

#### 2.2.1 基本公理

1. **外延公理 (Axiom of Extensionality)**：
   ∀x ∀y (∀z (z ∈ x ↔ z ∈ y) → x = y)

2. **空集公理 (Axiom of Empty Set)**：
   ∃x ∀y (y ∉ x)

3. **偶集公理 (Axiom of Pairing)**：
   ∀x ∀y ∃z ∀w (w ∈ z ↔ w = x ∨ w = y)

4. **并集公理 (Axiom of Union)**：
   ∀F ∃A ∀x (x ∈ A ↔ ∃B (B ∈ F ∧ x ∈ B))

5. **幂集公理 (Axiom of Power Set)**：
   ∀x ∃y ∀z (z ∈ y ↔ z ⊆ x)

6. **无穷公理 (Axiom of Infinity)**：
   ∃x (∅ ∈ x ∧ ∀y (y ∈ x → y ∪ {y} ∈ x))

7. **替换公理模式 (Axiom Schema of Replacement)**：
   对于每个公式φ(x,y)，有：
   ∀x ∀y ∀z (φ(x,y) ∧ φ(x,z) → y = z) → ∀A ∃B ∀y (y ∈ B ↔ ∃x (x ∈ A ∧ φ(x,y)))

8. **正则公理 (Axiom of Regularity)**：
   ∀x (x ≠ ∅ → ∃y (y ∈ x ∧ y ∩ x = ∅))

9. **选择公理 (Axiom of Choice)**：
   ∀F (∀A (A ∈ F → A ≠ ∅) ∧ ∀A ∀B (A ∈ F ∧ B ∈ F ∧ A ≠ B → A ∩ B = ∅) → ∃C ∀A (A ∈ F → ∃x (A ∩ C = {x})))

### 2.3 序偶与笛卡尔积

#### 2.3.1 序偶的定义

**库拉托夫斯基定义**：(a,b) = {{a}, {a,b}}

**性质**：

- (a,b) = (c,d) ↔ a = c ∧ b = d

#### 2.3.2 笛卡尔积

**定义**：A × B = {(a,b) | a ∈ A ∧ b ∈ B}

**性质**：

- A × (B ∪ C) = (A × B) ∪ (A × C)
- A × (B ∩ C) = (A × B) ∩ (A × C)
- (A × B) × C ≠ A × (B × C)

## 3. 数系理论

### 3.1 自然数

#### 3.1.1 皮亚诺公理

1. **0是自然数**
2. **每个自然数都有唯一的后继**
3. **0不是任何自然数的后继**
4. **不同的自然数有不同的后继**
5. **数学归纳原理**

#### 3.1.2 冯·诺依曼构造

0 = ∅
1 = {0} = {∅}
2 = {0,1} = {∅, {∅}}
3 = {0,1,2} = {∅, {∅}, {∅, {∅}}}
...

### 3.2 整数

#### 3.2.1 构造方法

**等价类方法**：

- 定义关系：(a,b) ~ (c,d) ↔ a + d = b + c
- 整数 = 等价类的集合

### 3.3 有理数

#### 3.3.1 构造方法

**等价类方法**：

- 定义关系：(a,b) ~ (c,d) ↔ a·d = b·c (b,d ≠ 0)
- 有理数 = 等价类的集合

### 3.4 实数

#### 3.4.1 戴德金分割

**定义**：实数是有理数的戴德金分割

**戴德金分割**：将有理数分为两个非空集合A,B，满足：

- A ∪ B = ℚ
- A ∩ B = ∅
- ∀a ∈ A, ∀b ∈ B, a < b
- A没有最大元素

#### 3.4.2 完备性公理

**上确界公理**：每个有上界的非空实数集合都有上确界

### 3.5 复数

#### 3.5.1 构造方法

**定义**：ℂ = ℝ × ℝ，带有运算：

- (a,b) + (c,d) = (a+c, b+d)
- (a,b) · (c,d) = (ac-bd, ad+bc)

## 4. 关系与函数

### 4.1 关系

#### 4.1.1 基本定义

**二元关系**：R ⊆ A × B

**n元关系**：R ⊆ A₁ × A₂ × ... × Aₙ

#### 4.1.2 重要关系类型

**等价关系**：

- 自反性：∀x (xRx)
- 对称性：∀x ∀y (xRy → yRx)
- 传递性：∀x ∀y ∀z (xRy ∧ yRz → xRz)

**偏序关系**：

- 自反性：∀x (xRx)
- 反对称性：∀x ∀y (xRy ∧ yRx → x = y)
- 传递性：∀x ∀y ∀z (xRy ∧ yRz → xRz)

**全序关系**：偏序关系 + 完全性：∀x ∀y (xRy ∨ yRx)

### 4.2 函数

#### 4.2.1 基本定义

**函数**：f: A → B 是满足以下条件的关系：

- ∀x ∈ A ∃y ∈ B ((x,y) ∈ f)
- ∀x ∈ A ∀y₁,y₂ ∈ B ((x,y₁) ∈ f ∧ (x,y₂) ∈ f → y₁ = y₂)

#### 4.2.2 函数类型

**单射 (Injection)**：∀x₁,x₂ ∈ A (f(x₁) = f(x₂) → x₁ = x₂)

**满射 (Surjection)**：∀y ∈ B ∃x ∈ A (f(x) = y)

**双射 (Bijection)**：既是单射又是满射

## 5. 基数与序数

### 5.1 基数

#### 5.1.1 基本概念

**基数**：集合的"大小"

**等势**：存在双射的两个集合具有相同的基数

#### 5.1.2 重要基数

**可数无穷**：|ℕ| = ℵ₀

**连续统**：|ℝ| = 2^ℵ₀

### 5.2 序数

#### 5.2.1 基本概念

**序数**：良序集合的顺序类型

**冯·诺依曼序数**：

- 0 = ∅
- α + 1 = α ∪ {α}
- 极限序数：sup{αᵢ | i < λ}

## 6. 范畴论基础

### 6.1 基本概念

#### 6.1.1 范畴

**定义**：范畴C包含：

- 对象集合 Ob(C)
- 态射集合 Mor(C)
- 复合运算 ∘
- 单位态射 id

**公理**：

- 结合律：(f ∘ g) ∘ h = f ∘ (g ∘ h)
- 单位律：id ∘ f = f = f ∘ id

#### 6.1.2 函子

**定义**：函子F: C → D是：

- 对象映射：Ob(C) → Ob(D)
- 态射映射：Mor(C) → Mor(D)
- 保持复合和单位

### 6.2 重要概念

#### 6.2.1 极限与余极限

**极限**：范畴论中的"乘积"概念

**余极限**：范畴论中的"余积"概念

#### 6.2.2 伴随函子

**定义**：F ⊣ G 表示F是G的左伴随

**性质**：Hom(FX, Y) ≅ Hom(X, GY)
