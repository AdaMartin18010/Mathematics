## 1. 引言

集合论是现代数学的基础语言，为所有数学分支提供统一的表达框架。本章将系统地介绍集合论的基本概念、运算和理论，建立严格的数学基础。

### 1.1 集合论的历史背景

集合论由德国数学家格奥尔格·康托尔（Georg Cantor）在19世纪末创立，旨在为数学提供统一的基础。其发展经历了几个重要阶段：

1. **朴素集合论**：康托尔的原始理论
2. **公理化集合论**：为解决悖论而建立的形式化系统
3. **现代集合论**：ZFC公理系统及其扩展

### 1.2 集合论的重要性

- **数学基础**：为所有数学对象提供统一的表达方式
- **逻辑工具**：提供精确的逻辑推理框架
- **抽象方法**：支持高度抽象的数学思维
- **应用广泛**：在计算机科学、逻辑学等领域有重要应用

## 2. 基本概念

### 2.1 集合的定义

**定义2.1（集合）**：集合是对象的聚集，这些对象称为集合的元素。

**形式化表述**：
设 $A$ 是一个集合，$x$ 是一个对象，则：

- 如果 $x$ 是 $A$ 的元素，记作 $x \in A$
- 如果 $x$ 不是 $A$ 的元素，记作 $x \notin A$

**外延公理**：两个集合相等当且仅当它们包含相同的元素。
$$\forall A \forall B (A = B \leftrightarrow \forall x (x \in A \leftrightarrow x \in B))$$

### 2.2 集合的表示方法

#### 2.2.1 列举法

**定义2.2（列举法）**：通过列出所有元素来表示集合。

**示例**：

- $A = \{1, 2, 3, 4, 5\}$
- $B = \{a, b, c\}$
- $C = \{\text{red}, \text{green}, \text{blue}\}$

#### 2.2.2 描述法

**定义2.3（描述法）**：通过描述元素的性质来表示集合。

**形式化表述**：
$$A = \{x \mid P(x)\}$$
表示所有满足性质 $P$ 的对象 $x$ 的集合。

**示例**：

- $E = \{x \mid x \text{ 是偶数}\}$
- $P = \{x \mid x \text{ 是素数}\}$
- $I = \{x \mid x \in \mathbb{R} \land 0 \leq x \leq 1\}$

### 2.3 特殊集合

#### 2.3.1 空集

**定义2.4（空集）**：不包含任何元素的集合称为空集，记作 $\emptyset$。

**空集公理**：存在一个不包含任何元素的集合。
$$\exists A \forall x (x \notin A)$$

**定理2.1（空集的唯一性）**：空集是唯一的。

**证明**：假设存在两个空集 $\emptyset_1$ 和 $\emptyset_2$。根据外延公理，$\emptyset_1 = \emptyset_2$ 当且仅当它们包含相同的元素。由于两个集合都不包含任何元素，所以 $\emptyset_1 = \emptyset_2$。

**证毕**。

#### 2.3.2 单元素集

**定义2.5（单元素集）**：只包含一个元素的集合称为单元素集。

**配对公理**：对于任意两个集合 $a$ 和 $b$，存在一个集合 $\{a, b\}$，其元素恰好是 $a$ 和 $b$。
$$\forall a \forall b \exists A \forall x (x \in A \leftrightarrow x = a \lor x = b)$$

#### 2.3.3 数集

**定义2.6（数集）**：

- **自然数集**：$\mathbb{N} = \{0, 1, 2, 3, \ldots\}$
- **整数集**：$\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$
- **有理数集**：$\mathbb{Q} = \{\frac{p}{q} \mid p, q \in \mathbb{Z}, q \neq 0\}$
- **实数集**：$\mathbb{R}$
- **复数集**：$\mathbb{C} = \{a + bi \mid a, b \in \mathbb{R}, i^2 = -1\}$

### 2.4 子集关系

**定义2.7（子集）**：集合 $A$ 是集合 $B$ 的子集，记作 $A \subseteq B$，如果 $A$ 的每个元素都是 $B$ 的元素。

**形式化表述**：
$$A \subseteq B \leftrightarrow \forall x (x \in A \rightarrow x \in B)$$

**定义2.8（真子集）**：集合 $A$ 是集合 $B$ 的真子集，记作 $A \subset B$，如果 $A \subseteq B$ 且 $A \neq B$。

**形式化表述**：
$$A \subset B \leftrightarrow A \subseteq B \land A \neq B$$

**定理2.2（子集的性质）**：

1. **自反性**：$\forall A (A \subseteq A)$
2. **传递性**：$\forall A \forall B \forall C (A \subseteq B \land B \subseteq C \rightarrow A \subseteq C)$
3. **反对称性**：$\forall A \forall B (A \subseteq B \land B \subseteq A \rightarrow A = B)$

**证明**：

1. **自反性**：对于任意 $x \in A$，显然 $x \in A$，所以 $A \subseteq A$。

2. **传递性**：假设 $A \subseteq B$ 且 $B \subseteq C$。对于任意 $x \in A$，由于 $A \subseteq B$，所以 $x \in B$；又由于 $B \subseteq C$，所以 $x \in C$。因此 $A \subseteq C$。

3. **反对称性**：假设 $A \subseteq B$ 且 $B \subseteq A$。根据外延公理，$A = B$。

**证毕**。

## 3. 集合运算

### 3.1 基本运算

#### 3.1.1 并集

**定义3.1（并集）**：集合 $A$ 和 $B$ 的并集，记作 $A \cup B$，是包含 $A$ 和 $B$ 中所有元素的集合。

**形式化表述**：
$$A \cup B = \{x \mid x \in A \lor x \in B\}$$

**定理3.1（并集的性质）**：

1. **交换律**：$A \cup B = B \cup A$
2. **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$
3. **幂等律**：$A \cup A = A$
4. **单位元**：$A \cup \emptyset = A$

**证明**：

1. **交换律**：对于任意 $x$，$x \in A \cup B \leftrightarrow x \in A \lor x \in B \leftrightarrow x \in B \lor x \in A \leftrightarrow x \in B \cup A$。

2. **结合律**：对于任意 $x$，$x \in (A \cup B) \cup C \leftrightarrow (x \in A \lor x \in B) \lor x \in C \leftrightarrow x \in A \lor (x \in B \lor x \in C) \leftrightarrow x \in A \cup (B \cup C)$。

3. **幂等律**：对于任意 $x$，$x \in A \cup A \leftrightarrow x \in A \lor x \in A \leftrightarrow x \in A$。

4. **单位元**：对于任意 $x$，$x \in A \cup \emptyset \leftrightarrow x \in A \lor x \in \emptyset \leftrightarrow x \in A$（因为 $x \in \emptyset$ 恒为假）。

**证毕**。

#### 3.1.2 交集

**定义3.2（交集）**：集合 $A$ 和 $B$ 的交集，记作 $A \cap B$，是同时属于 $A$ 和 $B$ 的元素的集合。

**形式化表述**：
$$A \cap B = \{x \mid x \in A \land x \in B\}$$

**定理3.2（交集的性质）**：

1. **交换律**：$A \cap B = B \cap A$
2. **结合律**：$(A \cap B) \cap C = A \cap (B \cap C)$
3. **幂等律**：$A \cap A = A$
4. **零元**：$A \cap \emptyset = \emptyset$

**证明**：类似并集性质的证明。

**证毕**。

#### 3.1.3 差集

**定义3.3（差集）**：集合 $A$ 和 $B$ 的差集，记作 $A \setminus B$，是属于 $A$ 但不属于 $B$ 的元素的集合。

**形式化表述**：
$$A \setminus B = \{x \mid x \in A \land x \notin B\}$$

#### 3.1.4 对称差

**定义3.4（对称差）**：集合 $A$ 和 $B$ 的对称差，记作 $A \triangle B$，是属于 $A$ 或 $B$ 但不同时属于两者的元素的集合。

**形式化表述**：
$$A \triangle B = (A \setminus B) \cup (B \setminus A)$$

### 3.2 德摩根定律

**定理3.3（德摩根定律）**：对于任意集合 $A$、$B$ 和全集 $U$：

1. $(A \cup B)^c = A^c \cap B^c$
2. $(A \cap B)^c = A^c \cup B^c$

其中 $A^c = U \setminus A$ 表示 $A$ 的补集。

**证明**：

1. 对于任意 $x$：
   $$x \in (A \cup B)^c \leftrightarrow x \notin A \cup B \leftrightarrow \neg(x \in A \lor x \in B) \leftrightarrow x \notin A \land x \notin B \leftrightarrow x \in A^c \land x \in B^c \leftrightarrow x \in A^c \cap B^c$$

2. 类似地可以证明第二个等式。

**证毕**。

### 3.3 幂集

**定义3.5（幂集）**：集合 $A$ 的幂集，记作 $\mathcal{P}(A)$，是 $A$ 的所有子集的集合。

**形式化表述**：
$$\mathcal{P}(A) = \{B \mid B \subseteq A\}$$

**幂集公理**：对于任意集合 $A$，存在一个集合 $\mathcal{P}(A)$，其元素是 $A$ 的所有子集。
$$\forall A \exists B \forall C (C \in B \leftrightarrow C \subseteq A)$$

**定理3.4（幂集的性质）**：

1. $\emptyset \in \mathcal{P}(A)$
2. $A \in \mathcal{P}(A)$
3. 如果 $A$ 有 $n$ 个元素，则 $\mathcal{P}(A)$ 有 $2^n$ 个元素

**证明**：

1. 空集是任意集合的子集，所以 $\emptyset \in \mathcal{P}(A)$。
2. 任意集合都是自己的子集，所以 $A \in \mathcal{P}(A)$。
3. 对于 $A$ 的每个元素，可以选择包含或不包含在子集中，所以有 $2^n$ 种选择。

**证毕**。

## 4. 关系与函数

### 4.1 笛卡尔积

**定义4.1（笛卡尔积）**：集合 $A$ 和 $B$ 的笛卡尔积，记作 $A \times B$，是所有有序对 $(a, b)$ 的集合，其中 $a \in A$，$b \in B$。

**形式化表述**：
$$A \times B = \{(a, b) \mid a \in A \land b \in B\}$$

**有序对的定义**：
$$(a, b) = \{\{a\}, \{a, b\}\}$$

**定理4.1（有序对的性质）**：$(a, b) = (c, d)$ 当且仅当 $a = c$ 且 $b = d$。

### 4.2 关系

**定义4.2（关系）**：从集合 $A$ 到集合 $B$ 的关系 $R$ 是 $A \times B$ 的一个子集。

**形式化表述**：
$$R \subseteq A \times B$$

**定义4.3（关系的性质）**：

- **自反性**：$\forall x \in A ((x, x) \in R)$
- **对称性**：$\forall x, y \in A ((x, y) \in R \rightarrow (y, x) \in R)$
- **反对称性**：$\forall x, y \in A ((x, y) \in R \land (y, x) \in R \rightarrow x = y)$
- **传递性**：$\forall x, y, z \in A ((x, y) \in R \land (y, z) \in R \rightarrow (x, z) \in R)$

### 4.3 等价关系

**定义4.4（等价关系）**：集合 $A$ 上的关系 $R$ 是等价关系，如果它是自反的、对称的和传递的。

**定义4.5（等价类）**：设 $R$ 是集合 $A$ 上的等价关系，对于 $a \in A$，$a$ 的等价类是：
$$[a]_R = \{x \in A \mid (a, x) \in R\}$$

**定理4.2（等价类的性质）**：

1. $\forall a \in A (a \in [a]_R)$
2. $\forall a, b \in A ([a]_R = [b]_R \lor [a]_R \cap [b]_R = \emptyset)$
3. $\bigcup_{a \in A} [a]_R = A$

### 4.4 函数

**定义4.6（函数）**：从集合 $A$ 到集合 $B$ 的函数 $f$ 是一个关系，满足：

1. $\forall a \in A \exists b \in B ((a, b) \in f)$（定义域覆盖）
2. $\forall a \in A \forall b_1, b_2 \in B ((a, b_1) \in f \land (a, b_2) \in f \rightarrow b_1 = b_2)$（单值性）

记作 $f: A \to B$。

**定义4.7（函数的性质）**：

- **单射（一对一）**：$\forall a_1, a_2 \in A (f(a_1) = f(a_2) \rightarrow a_1 = a_2)$
- **满射（映上）**：$\forall b \in B \exists a \in A (f(a) = b)$
- **双射（一一对应）**：既是单射又是满射

## 5. 基数理论

### 5.1 等势

**定义5.1（等势）**：集合 $A$ 和 $B$ 等势，记作 $A \sim B$，如果存在从 $A$ 到 $B$ 的双射。

**定理5.1（等势的性质）**：

1. **自反性**：$A \sim A$
2. **对称性**：$A \sim B \rightarrow B \sim A$
3. **传递性**：$A \sim B \land B \sim C \rightarrow A \sim C$

### 5.2 基数

**定义5.2（基数）**：集合 $A$ 的基数，记作 $|A|$，是 $A$ 的等势类的代表。

**定义5.3（有限集与无限集）**：

- 集合 $A$ 是有限的，如果存在自然数 $n$ 使得 $A \sim \{1, 2, \ldots, n\}$
- 集合 $A$ 是无限的，如果它不是有限的

**定义5.4（可数集）**：集合 $A$ 是可数的，如果 $A \sim \mathbb{N}$。

**定理5.2（可数集的性质）**：

1. $\mathbb{N}$ 是可数的
2. $\mathbb{Z}$ 是可数的
3. $\mathbb{Q}$ 是可数的
4. $\mathbb{R}$ 是不可数的

**证明**：

1. $\mathbb{N} \sim \mathbb{N}$ 显然成立。

2. 定义 $f: \mathbb{N} \to \mathbb{Z}$ 如下：
   $$f(n) = \begin{cases}
   \frac{n}{2} & \text{if } n \text{ is even} \\
   -\frac{n+1}{2} & \text{if } n \text{ is odd}
   \end{cases}$$
   这是一个双射。

3. 可以构造一个从 $\mathbb{N}$ 到 $\mathbb{Q}$ 的双射。

4. 使用康托尔对角线法证明 $\mathbb{R}$ 不可数。

**证毕**。

### 5.3 连续统假设

**连续统假设**：不存在基数 $\kappa$ 使得 $\aleph_0 < \kappa < 2^{\aleph_0}$。

这个假设在ZFC公理系统中既不能被证明，也不能被否证。

## 6. 序数理论

### 6.1 良序集

**定义6.1（良序集）**：集合 $A$ 上的全序关系 $\leq$ 是良序，如果 $A$ 的每个非空子集都有最小元素。

**定义6.2（序数）**：序数是良序集的序型。

### 6.2 序数的构造

**定义6.3（后继序数）**：序数 $\alpha$ 的后继是 $\alpha + 1 = \alpha \cup \{\alpha\}$。

**定义6.4（极限序数）**：序数 $\alpha$ 是极限序数，如果它不是任何序数的后继。

**定理6.1（序数的性质）**：

1. 每个序数都是传递集
2. 序数上的 $\in$ 关系是良序
3. 序数的序数还是序数

## 7. 公理系统

### 7.1 ZFC公理系统

ZFC公理系统包含以下公理：

1. **外延公理**：$\forall A \forall B (A = B \leftrightarrow \forall x (x \in A \leftrightarrow x \in B))$
2. **空集公理**：$\exists A \forall x (x \notin A)$
3. **配对公理**：$\forall a \forall b \exists A \forall x (x \in A \leftrightarrow x = a \lor x = b)$
4. **并集公理**：$\forall F \exists A \forall x (x \in A \leftrightarrow \exists B (B \in F \land x \in B))$
5. **幂集公理**：$\forall A \exists B \forall C (C \in B \leftrightarrow C \subseteq A)$
6. **无穷公理**：$\exists A (\emptyset \in A \land \forall x (x \in A \rightarrow x \cup \{x\} \in A))$
7. **替换公理模式**：对于每个公式 $\phi(x, y)$：
   $$\forall A (\forall x \in A \exists! y \phi(x, y) \rightarrow \exists B \forall y (y \in B \leftrightarrow \exists x \in A \phi(x, y)))$$
8. **正则公理**：$\forall A (A \neq \emptyset \rightarrow \exists x \in A (x \cap A = \emptyset))$
9. **选择公理**：对于任意非空集合族，存在选择函数。

### 7.2 选择公理

**选择公理**：对于任意非空集合族 $\{A_i\}_{i \in I}$，存在函数 $f: I \to \bigcup_{i \in I} A_i$ 使得 $f(i) \in A_i$ 对所有 $i \in I$ 成立。

**定理7.1（选择公理的等价形式）**：

1. **佐恩引理**：每个偏序集都有极大链
2. **良序定理**：每个集合都可以良序化
3. **乘积非空**：非空集合族的笛卡尔积非空

## 总结

集合论为现代数学提供了统一的基础语言和严格的逻辑框架。通过公理化方法，我们建立了集合论的形式化系统，为所有数学分支提供了坚实的基础。集合论不仅解决了数学基础问题，还在计算机科学、逻辑学等领域有重要应用。
