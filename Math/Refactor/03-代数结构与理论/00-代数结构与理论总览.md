## 1. 基础代数结构

### 1.1 代数结构的基本概念

#### 1.1.1 运算与代数结构

**定义**：代数结构是一个非空集合S，配备一个或多个运算

**n元运算**：f: Sⁿ → S

**重要性质**：

- **封闭性**：运算结果仍在集合中
- **结合律**：(a·b)·c = a·(b·c)
- **交换律**：a·b = b·a
- **单位元**：存在e使得e·a = a·e = a
- **逆元**：对每个a存在a⁻¹使得a·a⁻¹ = a⁻¹·a = e

#### 1.1.2 同态与同构

**同态**：保持运算结构的映射

- f(a·b) = f(a)·f(b)

**同构**：双射同态

- 两个结构在代数上"相同"

### 1.2 代数结构的分类

**按运算数量分类**：

- 一元代数结构
- 二元代数结构
- 多元代数结构

**按性质分类**：

- 群
- 环
- 域
- 模
- 代数

## 2. 群论

### 2.1 群的基本定义

#### 2.1.1 群的定义

**定义**：群(G,·)是一个非空集合G，配备二元运算·，满足：

1. **封闭性**：∀a,b ∈ G, a·b ∈ G
2. **结合律**：∀a,b,c ∈ G, (a·b)·c = a·(b·c)
3. **单位元**：∃e ∈ G, ∀a ∈ G, e·a = a·e = a
4. **逆元**：∀a ∈ G, ∃a⁻¹ ∈ G, a·a⁻¹ = a⁻¹·a = e

#### 2.1.2 群的基本性质

**唯一性定理**：

- 单位元唯一
- 每个元素的逆元唯一

**消去律**：

- 左消去律：a·b = a·c → b = c
- 右消去律：b·a = c·a → b = c

### 2.2 重要群类

#### 2.2.1 阿贝尔群

**定义**：满足交换律的群

- ∀a,b ∈ G, a·b = b·a

**例子**：

- (ℤ,+)
- (ℝ,+)
- (ℂ,+)

#### 2.2.2 有限群

**定义**：元素个数有限的群

**拉格朗日定理**：子群的阶整除群的阶

**西罗定理**：关于p-子群的存在性

#### 2.2.3 循环群

**定义**：由单个元素生成的群

**结构定理**：每个循环群同构于(ℤ,+)或(ℤₙ,+)

### 2.3 子群与正规子群

#### 2.3.1 子群

**定义**：群G的子集H，在G的运算下构成群

**判定条件**：

- H非空
- ∀a,b ∈ H, a·b ∈ H
- ∀a ∈ H, a⁻¹ ∈ H

#### 2.3.2 正规子群

**定义**：满足∀g ∈ G, gHg⁻¹ ⊆ H的子群

**等价条件**：

- ∀g ∈ G, gHg⁻¹ = H
- ∀g ∈ G, gH = Hg

### 2.4 商群与同态定理

#### 2.4.1 商群

**定义**：G/N = {gN | g ∈ G}，其中N是正规子群

**运算**：(g₁N)·(g₂N) = (g₁g₂)N

#### 2.4.2 同态定理

**第一同构定理**：G/Ker(f) ≅ Im(f)

**第二同构定理**：(H·N)/N ≅ H/(H∩N)

**第三同构定理**：(G/N)/(H/N) ≅ G/H

### 2.5 群作用

#### 2.5.1 群作用的定义

**定义**：群G在集合X上的作用是映射G×X → X，满足：

- e·x = x
- (g₁g₂)·x = g₁·(g₂·x)

#### 2.5.2 轨道与稳定子

**轨道**：Orb(x) = {g·x | g ∈ G}

**稳定子**：Stab(x) = {g ∈ G | g·x = x}

**轨道-稳定子定理**：|Orb(x)| = [G : Stab(x)]

## 3. 环论

### 3.1 环的基本定义

#### 3.1.1 环的定义

**定义**：环(R,+,·)是一个非空集合R，配备两个二元运算+和·，满足：

1. **(R,+)是阿贝尔群**
2. **乘法结合律**：(a·b)·c = a·(b·c)
3. **分配律**：a·(b+c) = a·b + a·c, (a+b)·c = a·c + b·c

#### 3.1.2 环的基本性质

**零元**：加法单位元0满足0·a = a·0 = 0

**负元**：(-a)·b = a·(-b) = -(a·b)

### 3.2 重要环类

#### 3.2.1 交换环

**定义**：乘法满足交换律的环

**例子**：

- ℤ
- ℚ
- ℝ
- ℂ

#### 3.2.2 整环

**定义**：无零因子的交换环

**性质**：消去律成立

#### 3.2.3 主理想整环

**定义**：每个理想都是主理想的整环

**例子**：

- ℤ
- F[x]（F是域）

### 3.3 理想与商环

#### 3.3.1 理想

**定义**：环R的子集I，满足：

- (I,+)是(R,+)的子群
- ∀r ∈ R, ∀a ∈ I, r·a ∈ I, a·r ∈ I

#### 3.3.2 商环

**定义**：R/I = {r+I | r ∈ R}

**运算**：(r₁+I) + (r₂+I) = (r₁+r₂)+I
         (r₁+I) · (r₂+I) = (r₁·r₂)+I

### 3.4 素理想与极大理想

#### 3.4.1 素理想

**定义**：理想P，满足ab ∈ P → a ∈ P 或 b ∈ P

**性质**：R/P是整环

#### 3.4.2 极大理想

**定义**：真理想M，不存在真包含M的理想

**性质**：R/M是域

## 4. 域论

### 4.1 域的基本定义

#### 4.1.1 域的定义

**定义**：域(F,+,·)是一个非空集合F，配备两个二元运算+和·，满足：

1. **(F,+)是阿贝尔群**
2. **(F\{0},·)是阿贝尔群**
3. **分配律**：a·(b+c) = a·b + a·c

#### 4.1.2 域的基本性质

**特征**：最小的正整数p，使得p·1 = 0（如果存在）

**子域**：域的子集，在域运算下构成域

### 4.2 域扩张

#### 4.2.1 代数扩张

**代数元**：α在F上代数，如果存在非零多项式f(x) ∈ F[x]，使得f(α) = 0

**代数扩张**：每个元素都是代数的扩张

#### 4.2.2 有限扩张

**定义**：[K:F] < ∞

**性质**：有限扩张是代数扩张

### 4.3 伽罗瓦理论

#### 4.3.1 伽罗瓦群

**定义**：域扩张K/F的伽罗瓦群是K的自同构群

**性质**：|Gal(K/F)| ≤ [K:F]

#### 4.3.2 伽罗瓦对应

**基本定理**：在伽罗瓦扩张中，子群与中间域之间存在一一对应

## 5. 模论

### 5.1 模的基本定义

#### 5.1.1 左R-模

**定义**：阿贝尔群M，配备标量乘法R×M → M，满足：

1. **(r+s)·m = r·m + s·m**
2. **r·(m+n) = r·m + r·n**
3. **(rs)·m = r·(s·m)**
4. **1·m = m**

#### 5.1.2 右R-模

**类似定义，但标量乘法为M×R → M**

### 5.2 自由模

#### 5.2.1 自由模的定义

**定义**：存在基的模

**性质**：每个元素可以唯一表示为基元素的线性组合

#### 5.2.2 自由模的结构

**秩**：基的基数（在交换环上唯一）

**同构定理**：同构的模有相同的秩

### 5.3 诺特模与阿廷模

#### 5.3.1 诺特模

**定义**：满足升链条件的模

**等价条件**：每个子模都是有限生成的

#### 5.3.2 阿廷模

**定义**：满足降链条件的模

**性质**：阿廷模是诺特模

## 6. 线性代数

### 6.1 向量空间

#### 6.1.1 向量空间的定义

**定义**：域F上的向量空间V，配备加法V×V → V和标量乘法F×V → V

**公理**：

1. **(V,+)是阿贝尔群**
2. **标量乘法满足分配律和结合律**
3. **1·v = v**

#### 6.1.2 线性无关与基

**线性无关**：向量组{v₁,...,vₙ}，如果∑aᵢvᵢ = 0 → aᵢ = 0

**基**：线性无关的生成集

**维数**：基的基数

### 6.2 线性变换

#### 6.2.1 线性变换的定义

**定义**：保持加法和标量乘法的映射T: V → W

**性质**：

- T(v₁ + v₂) = T(v₁) + T(v₂)
- T(av) = aT(v)

#### 6.2.2 矩阵表示

**坐标矩阵**：在给定基下的矩阵表示

**相似矩阵**：同一线性变换在不同基下的矩阵

### 6.3 特征值与特征向量

#### 6.3.1 基本概念

**特征值**：标量λ，使得T(v) = λv有非零解

**特征向量**：对应的非零向量v

#### 6.3.2 对角化

**可对角化**：存在基，使得矩阵为对角矩阵

**条件**：特征向量构成基

## 7. 抽象代数

### 7.1 伽罗瓦理论

#### 7.1.1 可解群

**定义**：存在正规子群列，使得每个商群都是阿贝尔群

**应用**：多项式方程的可解性

#### 7.1.2 伽罗瓦理论的应用

**五次方程**：一般五次方程不可用根式求解

### 7.2 代数几何基础

#### 7.2.1 代数集

**定义**：多项式方程组的解集

**希尔伯特零点定理**：代数集与根理想的对应

#### 7.2.2 仿射代数簇

**定义**：不可约的代数集

**坐标环**：多项式环模去理想

### 7.3 同调代数

#### 7.3.1 复形

**定义**：模的序列，满足d² = 0

**同调群**：Ker(d)/Im(d)

#### 7.3.2 导出函子

**Ext函子**：Extⁿ(M,N)

**Tor函子**：Torₙ(M,N)

**总结**：代数结构与理论为数学提供了统一的语言和工具，从基础的群、环、域到高级的伽罗瓦理论、代数几何，构成了现代数学的重要支柱。
