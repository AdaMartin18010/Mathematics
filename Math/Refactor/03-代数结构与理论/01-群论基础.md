## 1. 引言

群论是抽象代数的核心分支，研究具有二元运算的代数结构。群的概念起源于19世纪，最初是为了研究多项式方程的可解性而发展起来的。今天，群论不仅在数学的各个分支中发挥重要作用，也在物理学、化学、计算机科学等领域有广泛应用。

### 1.1 群论的历史发展

群论的发展经历了几个重要阶段：

1. **早期发展**（19世纪初）：拉格朗日、阿贝尔、伽罗瓦等数学家研究多项式方程的可解性
2. **系统化**（19世纪中后期）：凯莱、克莱因等数学家将群论系统化
3. **抽象化**（20世纪初）：范德瓦尔登等数学家将群论抽象化
4. **现代发展**（20世纪中后期）：有限单群分类、表示论等重大成果

### 1.2 群论的重要性

群论在数学中具有重要地位：

- **统一性**：为许多数学结构提供统一框架
- **对称性**：自然描述对称性概念
- **应用性**：在多个领域有重要应用

## 2. 群的基本概念

### 2.1 群的定义

**定义**：一个群是一个非空集合 $G$ 连同其上的二元运算 $\cdot: G \times G \rightarrow G$，满足以下公理：

1. **封闭性**：对任意 $a, b \in G$，$a \cdot b \in G$
2. **结合律**：对任意 $a, b, c \in G$，$(a \cdot b) \cdot c = a \cdot (b \cdot c)$
3. **单位元**：存在 $e \in G$，使得对任意 $a \in G$，$e \cdot a = a \cdot e = a$
4. **逆元**：对任意 $a \in G$，存在 $a^{-1} \in G$，使得 $a \cdot a^{-1} = a^{-1} \cdot a = e$

**记号**：群通常记作 $(G, \cdot)$ 或简记为 $G$。

#### 2.1.1 群的实现

```haskell
-- 群的数据结构
class Group a where
  multiply :: a -> a -> a
  identity :: a
  inverse :: a -> a

-- 群公理的验证
isGroup :: (Eq a, Group a) => [a] -> Bool
isGroup elements = 
  let e = identity
  in all (\a -> multiply e a == a && multiply a e == a) elements &&
     all (\a -> multiply a (inverse a) == e && multiply (inverse a) a == e) elements &&
     -- 结合律验证（简化版）
     True
```

### 2.2 群的例子

#### 2.2.1 数群

**整数加法群** $(\mathbb{Z}, +)$：

- 单位元：$0$
- 逆元：$-a$

**有理数乘法群** $(\mathbb{Q}^*, \cdot)$：

- 单位元：$1$
- 逆元：$\frac{1}{a}$

#### 2.2.2 对称群

**$n$次对称群** $S_n$：

- 元素：$\{1, 2, \ldots, n\}$ 的所有置换
- 运算：置换的复合
- 单位元：恒等置换
- 逆元：逆置换

```rust
// 对称群的实现
#[derive(Clone, Debug)]
struct Permutation {
    mapping: Vec<usize>,
}

impl Permutation {
    fn identity(n: usize) -> Self {
        Permutation {
            mapping: (0..n).collect(),
        }
    }
    
    fn compose(&self, other: &Permutation) -> Permutation {
        let mut result = vec![0; self.mapping.len()];
        for i in 0..self.mapping.len() {
            result[i] = other.mapping[self.mapping[i]];
        }
        Permutation { mapping: result }
    }
    
    fn inverse(&self) -> Permutation {
        let mut inv = vec![0; self.mapping.len()];
        for i in 0..self.mapping.len() {
            inv[self.mapping[i]] = i;
        }
        Permutation { mapping: inv }
    }
}
```

#### 2.2.3 矩阵群

**一般线性群** $\text{GL}_n(\mathbb{R})$：

- 元素：$n \times n$ 可逆实矩阵
- 运算：矩阵乘法
- 单位元：单位矩阵
- 逆元：逆矩阵

### 2.3 群的基本性质

#### 2.3.1 消去律

**左消去律**：如果 $a \cdot b = a \cdot c$，则 $b = c$
**右消去律**：如果 $b \cdot a = c \cdot a$，则 $b = c$

#### 2.3.2 单位元和逆元的唯一性

**定理**：群中的单位元和每个元素的逆元都是唯一的。

**证明**：

- 设 $e$ 和 $e'$ 都是单位元，则 $e = e \cdot e' = e'$
- 设 $a^{-1}$ 和 $a'$ 都是 $a$ 的逆元，则 $a^{-1} = a^{-1} \cdot e = a^{-1} \cdot (a \cdot a') = (a^{-1} \cdot a) \cdot a' = e \cdot a' = a'$

#### 2.3.3 幂运算

对任意 $a \in G$ 和正整数 $n$，定义：

- $a^n = a \cdot a \cdot \cdots \cdot a$（$n$ 个 $a$）
- $a^0 = e$
- $a^{-n} = (a^{-1})^n$

## 3. 子群与陪集

### 3.1 子群的定义

**定义**：群 $G$ 的子集 $H$ 是 $G$ 的子群，如果：

1. $H$ 非空
2. 对任意 $a, b \in H$，$a \cdot b \in H$
3. 对任意 $a \in H$，$a^{-1} \in H$

**等价条件**：$H$ 是子群当且仅当对任意 $a, b \in H$，$a \cdot b^{-1} \in H$。

#### 3.1.1 子群的例子

**平凡子群**：$\{e\}$ 和 $G$ 本身
**中心**：$Z(G) = \{a \in G \mid \forall g \in G, a \cdot g = g \cdot a\}$
**共轭类**：$C(a) = \{g \cdot a \cdot g^{-1} \mid g \in G\}$

### 3.2 陪集与拉格朗日定理

#### 3.2.1 陪集的定义

**左陪集**：$aH = \{a \cdot h \mid h \in H\}$
**右陪集**：$Ha = \{h \cdot a \mid h \in H\}$

#### 3.2.2 陪集的性质

1. **划分性质**：左陪集构成 $G$ 的一个划分
2. **等势性**：所有左陪集都与 $H$ 等势
3. **等价关系**：$a \sim b$ 当且仅当 $a^{-1} \cdot b \in H$

#### 3.2.3 拉格朗日定理

**定理**：如果 $H$ 是有限群 $G$ 的子群，则 $|H|$ 整除 $|G|$。

**证明**：$G$ 被 $H$ 的左陪集划分，每个陪集的大小等于 $|H|$，因此 $|G| = [G:H] \cdot |H|$。

**推论**：

- 有限群中元素的阶整除群的阶
- 素数阶群是循环群

### 3.3 正规子群

#### 3.3.1 正规子群的定义

**定义**：子群 $N$ 是群 $G$ 的正规子群，如果对任意 $g \in G$，$gN = Ng$。

**等价条件**：

- 对任意 $g \in G$，$g \cdot N \cdot g^{-1} = N$
- 对任意 $g \in G$ 和 $n \in N$，$g \cdot n \cdot g^{-1} \in N$

#### 3.3.2 正规子群的性质

1. **商群**：如果 $N$ 是正规子群，则 $G/N$ 构成群
2. **核**：同态的核是正规子群
3. **中心**：群的中心是正规子群

## 4. 群同态

### 4.1 同态的定义

**定义**：群 $G$ 到群 $H$ 的映射 $\phi: G \rightarrow H$ 是同态，如果对任意 $a, b \in G$，
$$\phi(a \cdot b) = \phi(a) \cdot \phi(b)$$

#### 4.1.1 同态的性质

1. **单位元保持**：$\phi(e_G) = e_H$
2. **逆元保持**：$\phi(a^{-1}) = \phi(a)^{-1}$
3. **子群保持**：如果 $K$ 是 $G$ 的子群，则 $\phi(K)$ 是 $H$ 的子群

#### 4.1.2 同态的实现

```haskell
-- 群同态
class GroupHomomorphism f where
  map :: (Group a, Group b) => f a b -> a -> b
  preservesOperation :: (Group a, Group b) => f a b -> a -> a -> Bool
  preservesOperation hom a b = 
    map hom (multiply a b) == multiply (map hom a) (map hom b)
```

### 4.2 同态基本定理

#### 4.2.1 核与像

**核**：$\ker \phi = \{a \in G \mid \phi(a) = e_H\}$
**像**：$\text{im} \phi = \{\phi(a) \mid a \in G\}$

#### 4.2.2 同态基本定理

**定理**：如果 $\phi: G \rightarrow H$ 是群同态，则：
$$G/\ker \phi \cong \text{im} \phi$$

**证明思路**：

1. $\ker \phi$ 是 $G$ 的正规子群
2. 定义映射 $\psi: G/\ker \phi \rightarrow \text{im} \phi$ 为 $\psi(a\ker \phi) = \phi(a)$
3. 证明 $\psi$ 是同构

### 4.3 同构

#### 4.3.1 同构的定义

**定义**：双射同态称为同构。

**性质**：

- 同构关系是等价关系
- 同构群具有相同的代数性质
- 同构保持群的阶

#### 4.3.2 自同构

**定义**：群到自身的同构称为自同构。

**自同构群**：$G$ 的所有自同构构成群 $\text{Aut}(G)$。

## 5. 群作用

### 5.1 群作用的定义

**定义**：群 $G$ 在集合 $X$ 上的作用是映射 $\cdot: G \times X \rightarrow X$，满足：

1. $e \cdot x = x$ 对所有 $x \in X$
2. $(g \cdot h) \cdot x = g \cdot (h \cdot x)$ 对所有 $g, h \in G$ 和 $x \in X$

#### 5.1.1 群作用的例子

**左平移**：$g \cdot x = g \cdot x$
**共轭作用**：$g \cdot x = g \cdot x \cdot g^{-1}$
**置换作用**：对称群在集合上的自然作用

### 5.2 轨道与稳定子

#### 5.2.1 轨道

**定义**：元素 $x \in X$ 的轨道是 $G \cdot x = \{g \cdot x \mid g \in G\}$

**性质**：

- 轨道构成 $X$ 的划分
- 轨道是传递的

#### 5.2.2 稳定子

**定义**：元素 $x \in X$ 的稳定子是 $G_x = \{g \in G \mid g \cdot x = x\}$

**性质**：

- $G_x$ 是 $G$ 的子群
- $|G \cdot x| = [G:G_x]$

### 5.3 轨道公式

**轨道公式**：如果 $G$ 是有限群，$X$ 是有限集合，则：
$$|X| = \sum_{x \in \text{轨道代表}} |G \cdot x| = \sum_{x \in \text{轨道代表}} [G:G_x]$$

**应用**：

- 计算群中元素的共轭类
- 证明西罗定理
- 计算对称群的结构

## 6. 有限群理论

### 6.1 西罗定理

#### 6.1.1 西罗子群

**定义**：如果 $p$ 是素数，$p$-西罗子群是阶为 $p^n$ 的子群，其中 $p^n$ 是 $|G|$ 中 $p$ 的最高幂。

#### 6.1.2 西罗定理

**第一西罗定理**：有限群 $G$ 的 $p$-西罗子群存在。

**第二西罗定理**：$G$ 的所有 $p$-西罗子群共轭。

**第三西罗定理**：$p$-西罗子群的个数 $n_p$ 满足 $n_p \equiv 1 \pmod{p}$ 且 $n_p$ 整除 $[G:P]$。

### 6.2 有限阿贝尔群

#### 6.2.1 基本定理

**定理**：有限阿贝尔群是循环群的直积。

**推论**：有限阿贝尔群的结构完全由其初等因子决定。

#### 6.2.2 分类

**循环群**：$\mathbb{Z}/n\mathbb{Z}$
**直积**：$\mathbb{Z}/m\mathbb{Z} \times \mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}/\text{lcm}(m,n)\mathbb{Z}$

### 6.3 单群

#### 6.3.1 单群的定义

**定义**：非平凡群 $G$ 是单群，如果它没有非平凡正规子群。

#### 6.3.2 有限单群分类

**定理**：有限单群分类定理（2004年完成）：

- 循环群 $\mathbb{Z}/p\mathbb{Z}$（$p$ 素数）
- 交错群 $A_n$（$n \geq 5$）
- 李型单群
- 26个散在单群

## 7. 范畴论视角

### 7.1 群范畴

#### 7.1.1 群范畴的定义

**群范畴** $\text{Grp}$：

- 对象：群
- 态射：群同态
- 复合：同态的复合
- 单位元：恒等同态

#### 7.1.2 范畴论性质

**积**：群的直积
**余积**：群的自由积
**极限**：群的逆极限
**余极限**：群的正向极限

### 7.2 自由群

#### 7.2.1 自由群的定义

**定义**：集合 $X$ 上的自由群 $F(X)$ 是包含 $X$ 的最一般的群。

**泛性质**：对任意群 $G$ 和映射 $f: X \rightarrow G$，存在唯一同态 $\phi: F(X) \rightarrow G$ 使得 $\phi|_X = f$。

#### 7.2.2 自由群的构造

**构造方法**：

1. 取字母表 $X \cup X^{-1}$
2. 考虑所有有限字
3. 模掉关系 $xx^{-1} = x^{-1}x = e$

### 7.3 泛性质

#### 7.3.1 积的泛性质

**定义**：群 $G_1 \times G_2$ 满足泛性质，如果存在投影同态 $\pi_i: G_1 \times G_2 \rightarrow G_i$，使得对任意群 $H$ 和同态 $f_i: H \rightarrow G_i$，存在唯一同态 $f: H \rightarrow G_1 \times G_2$ 使得 $\pi_i \circ f = f_i$。

#### 7.3.2 余积的泛性质

**定义**：群 $G_1 * G_2$ 满足余积的泛性质，如果存在包含同态 $\iota_i: G_i \rightarrow G_1 * G_2$，使得对任意群 $H$ 和同态 $f_i: G_i \rightarrow H$，存在唯一同态 $f: G_1 * G_2 \rightarrow H$ 使得 $f \circ \iota_i = f_i$。

## 8. 应用与扩展

### 8.1 物理学应用

#### 8.1.1 对称性

**空间对称性**：平移、旋转、反射
**时间对称性**：时间平移、时间反演
**规范对称性**：电磁学、弱相互作用、强相互作用

#### 8.1.2 表示论

**线性表示**：群到线性变换群的同态
**不可约表示**：群表示论的基础
**特征标**：表示的重要不变量

### 8.2 密码学应用

#### 8.2.1 离散对数问题

**问题**：给定 $g, h \in G$，求 $x$ 使得 $g^x = h$

**应用**：Diffie-Hellman密钥交换、ElGamal加密

struct GroupAction {
    group: Vec<Permutation>,
    set: Vec<usize>,
}

impl GroupAction {
    fn orbit(&self, element: usize) -> Vec<usize> {
        let mut orbit = vec![element];
        let mut visited = vec![false; self.set.len()];
        visited[element] = true;
        
        let mut queue = vec![element];
        while let Some(x) = queue.pop() {
            for g in &self.group {
                let y = g.apply(x);
                if !visited[y] {
                    visited[y] = true;
                    orbit.push(y);
                    queue.push(y);
                }
            }
        }
        orbit
    }
    
    fn stabilizer(&self, element: usize) -> Vec<Permutation> {
        self.group.iter()
            .filter(|g| g.apply(element) == element)
            .cloned()
            .collect()
    }
}
