## 1. 引言

群论是代数学的基础分支，研究具有一个二元运算的代数结构。
群的概念在数学的各个分支中都有重要应用，从基础的对称性研究到高级的代数几何和数论。

**核心思想**：

- **对称性**：群是描述对称性的自然语言
- **结构**：群的结构反映了数学对象的内部组织
- **变换**：群可以看作变换的集合

## 2. 群的基本概念

### 2.1 群的定义

**定义 2.1** (群)
群是一个集合 $G$ 配备一个二元运算 $\cdot: G \times G \rightarrow G$，满足以下公理：

1. **封闭性**：$\forall a, b \in G, a \cdot b \in G$
2. **结合律**：$\forall a, b, c \in G, (a \cdot b) \cdot c = a \cdot (b \cdot c)$
3. **单位元**：$\exists e \in G, \forall a \in G, e \cdot a = a \cdot e = a$
4. **逆元**：$\forall a \in G, \exists a^{-1} \in G, a \cdot a^{-1} = a^{-1} \cdot a = e$

**定义 2.2** (阿贝尔群)
如果群 $G$ 还满足交换律：$\forall a, b \in G, a \cdot b = b \cdot a$，则称 $G$ 为阿贝尔群。

### 2.2 基本性质

**定理 2.1** (单位元唯一性)
群的单位元是唯一的。

**证明**：
假设 $e$ 和 $e'$ 都是单位元，则：
$e = e \cdot e' = e'$

**定理 2.2** (逆元唯一性)
群中每个元素的逆元是唯一的。

**证明**：
假设 $a^{-1}$ 和 $a'$ 都是 $a$ 的逆元，则：
$a^{-1} = a^{-1} \cdot e = a^{-1} \cdot (a \cdot a') = (a^{-1} \cdot a) \cdot a' = e \cdot a' = a'$

**定理 2.3** (消去律)
在群中，如果 $a \cdot b = a \cdot c$，则 $b = c$；如果 $b \cdot a = c \cdot a$，则 $b = c$。

**证明**：
$a \cdot b = a \cdot c \Rightarrow a^{-1} \cdot (a \cdot b) = a^{-1} \cdot (a \cdot c) \Rightarrow (a^{-1} \cdot a) \cdot b = (a^{-1} \cdot a) \cdot c \Rightarrow e \cdot b = e \cdot c \Rightarrow b = c$

### 2.3 重要例子

**例 2.1** (整数加法群)
$(\mathbb{Z}, +)$ 是一个阿贝尔群：

- 单位元：$0$
- 逆元：$-n$

**例 2.2** (非零实数乘法群)
$(\mathbb{R}^*, \cdot)$ 是一个阿贝尔群：

- 单位元：$1$
- 逆元：$\frac{1}{x}$

**例 2.3** (对称群)
$S_n$ 是 $n$ 个元素的对称群：

- 元素：$n$ 个元素的排列
- 运算：排列的复合
- 单位元：恒等排列
- 逆元：逆排列

```haskell
-- Haskell中的群实现
class Group a where
    identity :: a
    inverse :: a -> a
    operation :: a -> a -> a

-- 整数加法群实例
instance Group Integer where
    identity = 0
    inverse x = -x
    operation = (+)

-- 验证群公理
isGroup :: (Eq a, Show a) => Group a => [a] -> Bool
isGroup elements = 
    all (\x -> operation identity x == x && operation x identity == x) elements &&
    all (\x -> operation x (inverse x) == identity && operation (inverse x) x == identity) elements &&
    all (\x y z -> operation (operation x y) z == operation x (operation y z)) (triples elements)
  where
    triples xs = [(x, y, z) | x <- xs, y <- xs, z <- xs]
```

## 3. 子群与正规子群

### 3.1 子群

**定义 3.1** (子群)
群 $G$ 的子集 $H$ 是 $G$ 的子群，如果：

1. $H$ 非空
2. $\forall a, b \in H, a \cdot b \in H$ (封闭性)
3. $\forall a \in H, a^{-1} \in H$ (逆元封闭性)

**定理 3.1** (子群判别法)
群 $G$ 的非空子集 $H$ 是子群当且仅当 $\forall a, b \in H, a \cdot b^{-1} \in H$。

**证明**：
必要性：如果 $H$ 是子群，则 $b^{-1} \in H$，所以 $a \cdot b^{-1} \in H$。

充分性：

1. 取 $a = b$，则 $e = a \cdot a^{-1} \in H$
2. 取 $a = e$，则 $b^{-1} = e \cdot b^{-1} \in H$
3. 取 $b^{-1}$ 代替 $b$，则 $a \cdot b = a \cdot (b^{-1})^{-1} \in H$

### 3.2 正规子群

**定义 3.2** (正规子群)
群 $G$ 的子群 $N$ 是正规子群，如果 $\forall g \in G, gNg^{-1} = N$。

**等价条件**：

- $\forall g \in G, \forall n \in N, gng^{-1} \in N$
- $\forall g \in G, gN = Ng$

**定理 3.2** (商群)
如果 $N$ 是群 $G$ 的正规子群，则 $G/N = \{gN : g \in G\}$ 在运算 $(gN)(hN) = (gh)N$ 下构成群。

**证明**：

1. **良定义性**：如果 $gN = g'N$ 和 $hN = h'N$，则 $(gh)N = (g'h')N$
2. **结合律**：$((gN)(hN))(kN) = (ghk)N = (gN)((hN)(kN))$
3. **单位元**：$eN$ 是单位元
4. **逆元**：$(gN)^{-1} = g^{-1}N$

```rust
// Rust中的子群和正规子群实现
#[derive(Debug, Clone)]
struct Group<T> {
    elements: Vec<T>,
    operation: fn(T, T) -> T,
    identity: T,
    inverse: fn(T) -> T,
}

impl<T: Clone + Eq> Group<T> {
    fn is_subgroup(&self, subset: &[T]) -> bool {
        // 检查非空
        if subset.is_empty() {
            return false;
        }
        
        // 检查封闭性
        for &a in subset {
            for &b in subset {
                let result = (self.operation)(a.clone(), b.clone());
                if !subset.contains(&result) {
                    return false;
                }
            }
        }
        
        // 检查逆元封闭性
        for &a in subset {
            let inv = (self.inverse)(a.clone());
            if !subset.contains(&inv) {
                return false;
            }
        }
        
        true
    }
    
    fn is_normal_subgroup(&self, subgroup: &[T]) -> bool {
        if !self.is_subgroup(subgroup) {
            return false;
        }
        
        // 检查正规性
        for &g in &self.elements {
            for &n in subgroup {
                let conjugate = (self.operation)(
                    (self.operation)(g.clone(), n.clone()),
                    (self.inverse)(g.clone())
                );
                if !subgroup.contains(&conjugate) {
                    return false;
                }
            }
        }
        
        true
    }
}
```

## 4. 群同态与同构

### 4.1 群同态

**定义 4.1** (群同态)
群 $G$ 到群 $H$ 的映射 $\phi: G \rightarrow H$ 是群同态，如果：
$\forall a, b \in G, \phi(a \cdot b) = \phi(a) \cdot \phi(b)$

**定义 4.2** (核与像)
群同态 $\phi: G \rightarrow H$ 的核和像定义为：

- $\ker \phi = \{g \in G : \phi(g) = e_H\}$
- $\text{im} \phi = \{\phi(g) : g \in G\}$

**定理 4.1** (同态基本性质)
群同态 $\phi: G \rightarrow H$ 满足：

1. $\phi(e_G) = e_H$
2. $\phi(g^{-1}) = \phi(g)^{-1}$
3. $\ker \phi$ 是 $G$ 的正规子群
4. $\text{im} \phi$ 是 $H$ 的子群

**证明**：

1. $\phi(e_G) = \phi(e_G \cdot e_G) = \phi(e_G) \cdot \phi(e_G)$，由消去律得 $\phi(e_G) = e_H$
2. $\phi(g^{-1}) \cdot \phi(g) = \phi(g^{-1} \cdot g) = \phi(e_G) = e_H$，所以 $\phi(g^{-1}) = \phi(g)^{-1}$
3. 如果 $a, b \in \ker \phi$，则 $\phi(a \cdot b^{-1}) = \phi(a) \cdot \phi(b)^{-1} = e_H \cdot e_H = e_H$，所以 $a \cdot b^{-1} \in \ker \phi$
4. 如果 $\phi(a), \phi(b) \in \text{im} \phi$，则 $\phi(a) \cdot \phi(b)^{-1} = \phi(a \cdot b^{-1}) \in \text{im} \phi$

### 4.2 群同构

**定义 4.3** (群同构)
群同态 $\phi: G \rightarrow H$ 是群同构，如果 $\phi$ 是双射。

**定理 4.2** (同构基本定理)
如果 $\phi: G \rightarrow H$ 是群同态，则：
$G/\ker \phi \cong \text{im} \phi$

**证明**：
定义映射 $\psi: G/\ker \phi \rightarrow \text{im} \phi$ 为 $\psi(g\ker \phi) = \phi(g)$。

1. **良定义性**：如果 $g\ker \phi = g'\ker \phi$，则 $g^{-1}g' \in \ker \phi$，所以 $\phi(g^{-1}g') = e_H$，即 $\phi(g) = \phi(g')$
2. **同态性**：$\psi((g\ker \phi)(h\ker \phi)) = \psi((gh)\ker \phi) = \phi(gh) = \phi(g)\phi(h) = \psi(g\ker \phi)\psi(h\ker \phi)$
3. **满射性**：显然
4. **单射性**：如果 $\psi(g\ker \phi) = e_H$，则 $\phi(g) = e_H$，所以 $g \in \ker \phi$，即 $g\ker \phi = \ker \phi$

## 5. 群作用

### 5.1 群作用的定义

**定义 5.1** (群作用)
群 $G$ 在集合 $X$ 上的作用是映射 $G \times X \rightarrow X$，记作 $(g, x) \mapsto g \cdot x$，满足：

1. $e \cdot x = x$
2. $(gh) \cdot x = g \cdot (h \cdot x)$

**定义 5.2** (轨道与稳定子)
群 $G$ 在集合 $X$ 上的作用：

- 轨道：$G \cdot x = \{g \cdot x : g \in G\}$
- 稳定子：$G_x = \{g \in G : g \cdot x = x\}$

**定理 5.1** (轨道-稳定子定理)
$|G \cdot x| = [G : G_x]$

**证明**：
定义映射 $\phi: G/G_x \rightarrow G \cdot x$ 为 $\phi(gG_x) = g \cdot x$。

1. **良定义性**：如果 $gG_x = hG_x$，则 $g^{-1}h \in G_x$，所以 $g^{-1}h \cdot x = x$，即 $g \cdot x = h \cdot x$
2. **满射性**：显然
3. **单射性**：如果 $g \cdot x = h \cdot x$，则 $g^{-1}h \cdot x = x$，所以 $g^{-1}h \in G_x$，即 $gG_x = hG_x$

### 5.2 重要例子

**例 5.1** (共轭作用)
群 $G$ 在自身上的共轭作用：$g \cdot h = ghg^{-1}$

**例 5.2** (左乘作用)
群 $G$ 在自身上的左乘作用：$g \cdot h = gh$

**例 5.3** (对称群作用)
对称群 $S_n$ 在 $\{1, 2, \ldots, n\}$ 上的作用：$\sigma \cdot i = \sigma(i)$

```haskell
-- Haskell中的群作用实现
class GroupAction g x where
    act :: g -> x -> x

-- 验证群作用公理
isGroupAction :: (Eq g, Eq x, Group g) => GroupAction g x => [g] -> [x] -> Bool
isGroupAction groupElements setElements = 
    all (\x -> act identity x == x) setElements &&
    all (\g h x -> act (operation g h) x == act g (act h x)) 
        (triples groupElements setElements)
  where
    triples gs xs = [(g, h, x) | g <- gs, h <- gs, x <- xs]

-- 轨道计算
orbit :: (Eq g, Eq x, GroupAction g x) => [g] -> x -> [x]
orbit groupElements x = nub [act g x | g <- groupElements]

-- 稳定子计算
stabilizer :: (Eq g, Eq x, GroupAction g x) => [g] -> x -> [g]
stabilizer groupElements x = [g | g <- groupElements, act g x == x]
```

## 6. 有限群论

### 6.1 拉格朗日定理

**定理 6.1** (拉格朗日定理)
如果 $H$ 是有限群 $G$ 的子群，则 $|H|$ 整除 $|G|$。

**证明**：
考虑 $G$ 在 $G/H$ 上的左乘作用。每个轨道的大小等于 $|H|$，所以 $|G| = |H| \cdot [G : H]$。

**推论 6.1**
有限群 $G$ 中每个元素的阶都整除 $|G|$。

**证明**：
元素 $g$ 的阶等于循环子群 $\langle g \rangle$ 的阶，由拉格朗日定理得证。

### 6.2 西罗定理

**定义 6.1** (西罗子群)
设 $p$ 是素数，$G$ 是有限群。如果 $p$ 整除 $|G|$，则 $G$ 的 $p$-西罗子群是阶为 $p$ 的最高幂的子群。

**定理 6.2** (西罗第一定理)
如果 $p$ 整除 $|G|$，则 $G$ 有 $p$-西罗子群。

**定理 6.3** (西罗第二定理)
$G$ 的所有 $p$-西罗子群都共轭。

**定理 6.4** (西罗第三定理)
$G$ 的 $p$-西罗子群的个数 $n_p$ 满足：

1. $n_p \equiv 1 \pmod{p}$
2. $n_p$ 整除 $[G : P]$，其中 $P$ 是任意 $p$-西罗子群

### 6.3 有限单群

**定义 6.2** (单群)
群 $G$ 是单群，如果 $G$ 没有非平凡的正规子群。

**定理 6.5** (有限单群分类)
有限单群分为以下几类：

1. 循环群 $\mathbb{Z}_p$，其中 $p$ 是素数
2. 交错群 $A_n$，其中 $n \geq 5$
3. 李型单群
4. 散在单群

## 7. 应用与示例

### 7.1 密码学应用

**RSA密码系统**：
基于群 $(\mathbb{Z}_n^*, \cdot)$ 的性质，其中 $n = pq$ 是两个大素数的乘积。

**椭圆曲线密码**：
基于椭圆曲线上的点群。

```rust
// Rust中的RSA实现示例
use num_bigint::{BigUint, RandBigInt};
use num_traits::One;

struct RSAKey {
    n: BigUint,
    e: BigUint,
    d: BigUint,
}

impl RSAKey {
    fn new(bit_length: usize) -> Self {
        let mut rng = rand::thread_rng();
        
        // 生成两个大素数
        let p = rng.gen_biguint(bit_length / 2);
        let q = rng.gen_biguint(bit_length / 2);
        
        let n = &p * &q;
        let phi = (&p - BigUint::one()) * (&q - BigUint::one());
        
        // 选择公钥指数
        let e = BigUint::from(65537u32);
        
        // 计算私钥指数
        let d = mod_inverse(&e, &phi).unwrap();
        
        RSAKey { n, e, d }
    }
    
    fn encrypt(&self, message: &BigUint) -> BigUint {
        message.modpow(&self.e, &self.n)
    }
    
    fn decrypt(&self, ciphertext: &BigUint) -> BigUint {
        ciphertext.modpow(&self.d, &self.n)
    }
}
```
