# MWE｜域论（最小例）

## 1. 目标

- 给出域论的最小示例，展示有限域构造、扩域、Galois理论的应用。

## 2. 有限域构造

### F4 的构造

- F4 = {0, 1, α, α+1}
- 其中 α² + α + 1 = 0
- 加法表：模2加法
- 乘法表：α³ = 1

### F8 的构造

- F8 = F2[x]/(x³ + x + 1)
- 元素：{0, 1, α, α², α+1, α²+1, α²+α, α²+α+1}
- 其中 α³ + α + 1 = 0

## 3. 扩域

### 二次扩域

- Q(√2) = {a + b√2 | a, b ∈ Q}
- 基：{1, √2}
- 维数：[Q(√2) : Q] = 2

### 三次扩域

- Q(∛2) = {a + b∛2 + c∛4 | a, b, c ∈ Q}
- 基：{1, ∛2, ∛4}
- 维数：[Q(∛2) : Q] = 3

## 4. Galois理论

### Q(√2) 的Galois群

- Gal(Q(√2)/Q) = {id, σ}
- σ(√2) = -√2
- |Gal(Q(√2)/Q)| = 2 = [Q(√2) : Q]

### 基本定理

- 中间域 ↔ Galois群的子群
- Q(√2) 的中间域：Q, Q(√2)
- 对应子群：Gal(Q(√2)/Q), {id}
