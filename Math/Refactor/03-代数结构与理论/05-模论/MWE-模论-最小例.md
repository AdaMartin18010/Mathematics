# MWE｜模论（最小例）

## 1. 目标

- 给出模论的最小示例，展示有限生成模、PID上的模结构定理的应用。

## 2. 有限生成模

### Z-模示例

- M = Z² = {(a, b) | a, b ∈ Z}
- 生成元：{(1, 0), (0, 1)}
- 子模：N = {(2a, 2b) | a, b ∈ Z}

### 商模

- M/N = {(a, b) + N | a, b ∈ Z}
- 等价类：[(a, b)] = [(a mod 2, b mod 2)]
- 结构：M/N ≅ Z₂ ⊕ Z₂

## 3. PID上的模结构定理

### 主理想整环

- Z 是 PID
- 每个理想都是主理想：I = ⟨n⟩

### 有限生成Z-模

- M ≅ Z^r ⊕ Z/(p₁^a₁) ⊕ ... ⊕ Z/(p_k^a_k)
- 其中 r 是自由秩，p_i 是素数

### 具体示例

- M = Z²/⟨(2, 0), (0, 3)⟩
- 结构：M ≅ Z/(2) ⊕ Z/(3) ≅ Z/(6)

## 4. 线性变换

### 模同态

- φ: Z² → Z², φ(a, b) = (2a, 3b)
- ker(φ) = {(0, 0)}
- im(φ) = {(2a, 3b) | a, b ∈ Z}

### 不变因子

- 矩阵表示：[2 0; 0 3]
- 不变因子：2, 3
- 初等因子：2, 3

## 5. 练习（3）

1) 构造 Z-模 Z³/⟨(2, 0, 0), (0, 3, 0), (0, 0, 5)⟩ 的结构。
2) 证明：有限生成Z-模的挠子模是有限群。
3) 计算模 Z² 上线性变换 φ(a, b) = (a+b, a-b) 的核与像。
