# 01-范畴的基本概念

## 1. 范畴的定义

范畴是范畴论中最基本的概念，它提供了一种统一的方式来描述具有类似结构的数学对象及其关系。

### 1.1 形式定义

一个**范畴** $\mathcal{C}$ 由以下三个组成部分构成：

1. **对象集合** $\text{Ob}(\mathcal{C})$：范畴中的对象集合。
2. **态射集合** $\text{Hom}(\mathcal{C})$：对于任意两个对象 $A, B \in \text{Ob}(\mathcal{C})$，存在一个态射集合 $\text{Hom}_{\mathcal{C}}(A, B)$，其中的元素 $f: A \to B$ 称为从 $A$ 到 $B$ 的态射（或箭头）。
3. **态射合成**：对于任意三个对象 $A, B, C \in \text{Ob}(\mathcal{C})$ 和态射 $f: A \to B$, $g: B \to C$，存在一个合成态射 $g \circ f: A \to C$。

这些组成部分需要满足以下公理：

1. **结合律**：对于任意态射 $f: A \to B$, $g: B \to C$, $h: C \to D$，有 $h \circ (g \circ f) = (h \circ g) \circ f$。
2. **单位律**：对于每个对象 $A \in \text{Ob}(\mathcal{C})$，存在一个恒等态射 $\text{id}_A: A \to A$，使得对于任意态射 $f: A \to B$，有 $f \circ \text{id}_A = f$ 和 $\text{id}_B \circ f = f$。

### 1.2 记号约定

范畴通常用黑板粗体或花体字母表示，如 $\mathcal{C}$, $\mathcal{D}$ 等。对象通常用大写字母 $A$, $B$, $C$ 表示，而态射则用小写字母 $f$, $g$, $h$ 表示。

## 2. 范畴的例子

### 2.1 集合范畴 Set

**集合范畴** Set 是最基本的范畴例子：

- **对象**：所有集合
- **态射**：集合间的函数
- **合成**：函数的通常合成
- **恒等态射**：恒等函数 $\text{id}_A: A \to A$，定义为 $\text{id}_A(a) = a$，对于所有 $a \in A$

### 2.2 群范畴 Grp

**群范畴** Grp 是另一个重要例子：

- **对象**：所有群
- **态射**：群同态
- **合成**：函数的通常合成
- **恒等态射**：恒等同态

### 2.3 拓扑空间范畴 Top

**拓扑空间范畴** Top 定义为：

- **对象**：所有拓扑空间
- **态射**：连续映射
- **合成**：函数的通常合成
- **恒等态射**：恒等映射

### 2.4 预序集范畴

任何预序集 $(P, \leq)$ 都可以视为一个范畴：

- **对象**：集合 $P$ 中的元素
- **态射**：对于 $a, b \in P$，如果 $a \leq b$，则存在唯一的态射 $a \to b$；否则不存在态射
- **合成**：由传递性保证
- **恒等态射**：由自反性保证

### 2.5 向量空间范畴 Vect

**向量空间范畴** Vect 定义为：

- **对象**：所有向量空间
- **态射**：线性映射
- **合成**：函数的通常合成
- **恒等态射**：恒等线性映射

## 3. 态射的性质

### 3.1 单态射

一个态射 $f: A \to B$ 称为**单态射**（或**单射**），如果对于任意对象 $Z$ 和任意态射 $g_1, g_2: Z \to A$，若 $f \circ g_1 = f \circ g_2$，则 $g_1 = g_2$。

在 Set 范畴中，单态射对应于通常意义上的单射（注入函数）。

### 3.2 满态射

一个态射 $f: A \to B$ 称为**满态射**（或**满射**），如果对于任意对象 $Z$ 和任意态射 $g_1, g_2: B \to Z$，若 $g_1 \circ f = g_2 \circ f$，则 $g_1 = g_2$。

在 Set 范畴中，满态射对应于通常意义上的满射（满函数）。

### 3.3 同构

一个态射 $f: A \to B$ 称为**同构**，如果存在一个态射 $g: B \to A$，使得 $g \circ f = \text{id}_A$ 且 $f \circ g = \text{id}_B$。此时，$g$ 称为 $f$ 的逆态射，记为 $f^{-1}$。

如果存在从对象 $A$ 到对象 $B$ 的同构，则称 $A$ 与 $B$ **同构**，记为 $A \cong B$。

在 Set 范畴中，同构对应于双射（既是单射又是满射的函数）。

## 4. 特殊对象

### 4.1 始对象

范畴 $\mathcal{C}$ 中的一个对象 $I$ 称为**始对象**（或**初始对象**），如果对于 $\mathcal{C}$ 中的任意对象 $A$，存在唯一的态射 $I \to A$。

在 Set 范畴中，空集 $\emptyset$ 是始对象，因为从空集到任何集合都存在唯一的函数（空函数）。

### 4.2 终对象

范畴 $\mathcal{C}$ 中的一个对象 $T$ 称为**终对象**（或**末端对象**），如果对于 $\mathcal{C}$ 中的任意对象 $A$，存在唯一的态射 $A \to T$。

在 Set 范畴中，任何单点集 $\{*\}$ 都是终对象，因为从任何集合到单点集都存在唯一的函数（常函数）。

### 4.3 零对象

范畴 $\mathcal{C}$ 中的一个对象 $Z$ 称为**零对象**，如果它既是始对象又是终对象。

在 Ab（阿贝尔群范畴）中，平凡群 $\{0\}$ 是零对象。

## 5. 子对象与商对象

### 5.1 子对象

在许多具体范畴中，可以定义**子对象**的概念，它通常通过单态射来表示。例如：

- 在 Set 中，子对象对应于子集
- 在 Grp 中，子对象对应于子群
- 在 Vect 中，子对象对应于子空间

### 5.2 商对象

类似地，**商对象**通常通过满态射来表示：

- 在 Set 中，商对象对应于商集（等价类）
- 在 Grp 中，商对象对应于商群
- 在 Vect 中，商对象对应于商空间

## 6. 范畴的类型

### 6.1 小范畴与大范畴

- **小范畴**：对象集合和所有态射集合都是集合（而非真类）的范畴
- **局部小范畴**：对于任意两个对象 $A$ 和 $B$，$\text{Hom}_{\mathcal{C}}(A, B)$ 是一个集合的范畴
- **大范畴**：对象集合或某些态射集合构成真类的范畴

### 6.2 满范畴与薄范畴

- **满范畴**：任意两个对象之间至多有一个态射的范畴
- **薄范畴**：任意两个对象之间至多有一个同构的范畴

### 6.3 具体范畴

**具体范畴**是一个配备了"忘却函子"到 Set 范畴的范畴，它将对象映射到其底层集合，将态射映射到其底层函数。大多数常见的数学结构范畴都是具体范畴。

## 7. 范畴论的思维方式

范畴论提供了一种独特的数学思维方式：

1. **关注结构而非元素**：范畴论关注对象之间的关系，而非对象的内部结构
2. **通过态射定义对象**：对象的性质通常通过其与其他对象的态射关系来定义
3. **抽象化与统一性**：提供了一种统一的语言来描述不同数学领域的结构
4. **图解思维**：使用图表来可视化和推理复杂的数学关系

## 8. 练习与应用

### 8.1 验证范畴公理

**练习**：验证 Set、Grp 和 Top 满足范畴的公理。

### 8.2 寻找同构

**练习**：在 Vect 范畴中，证明所有维数相同的向量空间都是同构的。

### 8.3 识别特殊对象

**练习**：找出以下范畴中的始对象和终对象：

- 有限集合范畴 FinSet
- 有限维向量空间范畴 FinVect
- 群范畴 Grp
