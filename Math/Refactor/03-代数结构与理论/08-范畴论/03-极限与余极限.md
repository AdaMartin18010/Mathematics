# 03-极限与余极限

## 1. 极限与余极限概述

极限和余极限是范畴论中描述通用构造的重要概念，它们统一了许多数学结构中常见的构造，如积、余积、等化子、余等化子、拉回、推出等。

### 1.1 基本思想

极限和余极限的核心思想是通过"通用性质"来定义数学对象，而不是通过具体的构造方式。这种方法有几个优点：

1. 提供了统一的语言来描述不同数学领域中的类似构造
2. 强调了构造的本质特性，而不是具体实现细节
3. 允许在不同范畴之间传递结构和性质

### 1.2 图与图范畴

在定义极限和余极限之前，我们需要引入图和图范畴的概念：

**图** $J$ 是一个由顶点和有向边组成的结构。形式上，图由两个集合组成：顶点集 $V$ 和边集 $E$，以及两个映射 $s, t: E \to V$，分别指定每条边的源点和目标点。

**图范畴**（或**小范畴**）$\mathcal{J}$ 是一个小范畴，可以看作是带有合成结构的图。

## 2. 锥与余锥

### 2.1 锥的定义

设 $F: \mathcal{J} \to \mathcal{C}$ 是一个函子，其中 $\mathcal{J}$ 是一个小范畴，$\mathcal{C}$ 是目标范畴。

$F$ 的一个**锥**是一个对 $(N, \psi)$，其中 $N$ 是 $\mathcal{C}$ 中的一个对象，$\psi$ 是一族态射 $\{\psi_j: N \to F(j) \mid j \in \mathcal{J}\}$，满足对于 $\mathcal{J}$ 中的任意态射 $f: j \to j'$，下图可交换：

```text
     N
    / \
   /   \
  /     \
ψ_j     ψ_{j'}
  \     /
   \   /
    v v
   F(j) --F(f)--> F(j')
```

即 $F(f) \circ \psi_j = \psi_{j'}$。

### 2.2 余锥的定义

类似地，$F$ 的一个**余锥**是一个对 $(N, \phi)$，其中 $N$ 是 $\mathcal{C}$ 中的一个对象，$\phi$ 是一族态射 $\{\phi_j: F(j) \to N \mid j \in \mathcal{J}\}$，满足对于 $\mathcal{J}$ 中的任意态射 $f: j \to j'$，下图可交换：

```text
   F(j) --F(f)--> F(j')
    |              |
    |              |
  φ_j            φ_{j'}
    |              |
    v              v
     N
```

即 $\phi_{j'} \circ F(f) = \phi_j$。

## 3. 极限与余极限的定义

### 3.1 极限的定义

函子 $F: \mathcal{J} \to \mathcal{C}$ 的**极限**是 $F$ 的一个锥 $(L, \psi)$，满足对于 $F$ 的任意其他锥 $(N, \gamma)$，存在唯一的态射 $u: N \to L$，使得对于所有 $j \in \mathcal{J}$，$\psi_j \circ u = \gamma_j$：

```text
     N
    / \
   /   \
  /     \
γ_j     γ_{j'}
  \     /
   \   /
    v v
   F(j) --F(f)--> F(j')
```

我们记 $L = \lim F$ 或 $L = \lim_{j \in \mathcal{J}} F(j)$。

### 3.2 余极限的定义

函子 $F: \mathcal{J} \to \mathcal{C}$ 的**余极限**是 $F$ 的一个余锥 $(L, \phi)$，满足对于 $F$ 的任意其他余锥 $(N, \gamma)$，存在唯一的态射 $u: L \to N$，使得对于所有 $j \in \mathcal{J}$，$u \circ \phi_j = \gamma_j$：

```text
   F(j) --F(f)--> F(j')
    |              |
    |              |
  φ_j            φ_{j'}
    |              |
    v              v
     L
     |
     |
     u
     |
     v
     N
```

我们记 $L = \text{colim} F$ 或 $L = \text{colim}_{j \in \mathcal{J}} F(j)$。

### 3.3 极限和余极限的唯一性

如果极限（或余极限）存在，那么它在同构意义下是唯一的。这意味着如果 $(L, \psi)$ 和 $(L', \psi')$ 都是 $F$ 的极限，则存在唯一的同构 $i: L \to L'$，使得对于所有 $j \in \mathcal{J}$，$\psi'_j \circ i = \psi_j$。

## 4. 特殊的极限和余极限

### 4.1 终对象和始对象

- **终对象**是空图（没有顶点和边的图）上的常函子的极限。
- **始对象**是空图上的常函子的余极限。

### 4.2 积和余积

考虑离散范畴 $\mathcal{J}$（只有对象和恒等态射的范畴）：

- $F: \mathcal{J} \to \mathcal{C}$ 的**积**是 $F$ 的极限，通常记为 $\prod_{j \in \mathcal{J}} F(j)$。
- $F: \mathcal{J} \to \mathcal{C}$ 的**余积**（或**直和**）是 $F$ 的余极限，通常记为 $\coprod_{j \in \mathcal{J}} F(j)$。

对于有限集合 $\{1, 2, \ldots, n\}$ 上的函子，积和余积分别记为 $A_1 \times A_2 \times \cdots \times A_n$ 和 $A_1 \sqcup A_2 \sqcup \cdots \sqcup A_n$（或 $A_1 + A_2 + \cdots + A_n$）。

### 4.3 等化子和余等化子

考虑由两个对象和它们之间的两个平行态射组成的范畴：

- **等化子**是两个平行态射 $f, g: A \to B$ 的极限，它是一个态射 $e: E \to A$，满足 $f \circ e = g \circ e$，且对于任何满足 $f \circ h = g \circ h$ 的态射 $h: X \to A$，存在唯一的态射 $u: X \to E$，使得 $e \circ u = h$。

- **余等化子**是两个平行态射 $f, g: A \to B$ 的余极限，它是一个态射 $c: B \to C$，满足 $c \circ f = c \circ g$，且对于任何满足 $h \circ f = h \circ g$ 的态射 $h: B \to X$，存在唯一的态射 $u: C \to X$，使得 $u \circ c = h$。

### 4.4 拉回和推出

考虑由三个对象和两个态射（形成一个角）组成的范畴：

- **拉回**（或**纤维积**）是两个态射 $f: A \to C$ 和 $g: B \to C$ 的极限，通常记为 $A \times_C B$。

- **推出**（或**余纤维积**）是两个态射 $f: C \to A$ 和 $g: C \to B$ 的余极限，通常记为 $A \sqcup_C B$。

## 5. 在具体范畴中的实现

### 5.1 集合范畴中的极限和余极限

在 Set 范畴中：

- **积**是笛卡尔积，即 $A \times B = \{(a, b) \mid a \in A, b \in B\}$。
- **余积**是不交并，即 $A \sqcup B = (A \times \{0\}) \cup (B \times \{1\})$。
- **等化子**是集合 $E = \{a \in A \mid f(a) = g(a)\}$，带有包含映射 $e: E \to A$。
- **余等化子**是商集 $B/{\sim}$，其中 $b_1 \sim b_2$ 当且仅当存在 $a_1, a_2, \ldots, a_n \in A$ 使得 $b_1 = f(a_1)$, $g(a_1) = f(a_2)$, $\ldots$, $g(a_{n-1}) = f(a_n)$, $g(a_n) = b_2$。
- **拉回**是集合 $\{(a, b) \in A \times B \mid f(a) = g(b)\}$。
- **推出**是商集 $(A \sqcup B)/{\sim}$，其中 $\sim$ 是由 $f(c) \sim g(c)$ 对于所有 $c \in C$ 生成的等价关系。

### 5.2 群范畴中的极限和余极限

在 Grp 范畴中：

- **积**是直积群，即 $G \times H = \{(g, h) \mid g \in G, h \in H\}$，带有分量式的群运算。
- **余积**是自由积（对于非交换群）或直和（对于交换群）。
- **等化子**是子群 $E = \{g \in G \mid f(g) = g(g)\}$。
- **余等化子**是商群 $H/N$，其中 $N$ 是由 $\{f(g)g(g)^{-1} \mid g \in G\}$ 生成的正规子群。
- **拉回**类似于集合的情况，但带有群结构。
- **推出**是商群 $(G * H)/N$，其中 $G * H$ 是自由积，$N$ 是由 $\{f(c)g(c)^{-1} \mid c \in C\}$ 生成的正规子群。

### 5.3 拓扑空间范畴中的极限和余极限

在 Top 范畴中：

- **积**是笛卡尔积，带有积拓扑。
- **余积**是不交并，带有和拓扑。
- **等化子**是子空间 $E = \{x \in X \mid f(x) = g(x)\}$，带有子空间拓扑。
- **余等化子**是商空间 $Y/{\sim}$，其中 $\sim$ 是由 $f(x) \sim g(x)$ 对于所有 $x \in X$ 生成的等价关系。
- **拉回**是子空间 $\{(a, b) \in A \times B \mid f(a) = g(b)\}$，带有子空间拓扑。
- **推出**是商空间，类似于集合的情况。

## 6. 极限和余极限的性质

### 6.1 函子与极限的交互

函子可以保持、反射或创造极限：

1. **保持极限**：如果 $F: \mathcal{C} \to \mathcal{D}$ 是一个函子，且对于任何函子 $G: \mathcal{J} \to \mathcal{C}$，如果 $\lim G$ 存在，则 $F(\lim G) \cong \lim (F \circ G)$，则称 $F$ 保持 $\mathcal{J}$-极限。

2. **反射极限**：如果对于任何函子 $G: \mathcal{J} \to \mathcal{C}$，如果 $\lim (F \circ G)$ 存在且 $F(\lim G) \cong \lim (F \circ G)$ 蕴含 $\lim G$ 存在，则称 $F$ 反射 $\mathcal{J}$-极限。

3. **创造极限**：如果对于任何函子 $G: \mathcal{J} \to \mathcal{C}$，如果 $\lim (F \circ G)$ 存在，则 $\lim G$ 存在且 $F(\lim G) \cong \lim (F \circ G)$，则称 $F$ 创造 $\mathcal{J}$-极限。

类似地，可以定义函子保持、反射或创造余极限。

### 6.2 完备范畴和余完备范畴

- **完备范畴**：如果范畴 $\mathcal{C}$ 有所有小极限（即，对于任何小范畴 $\mathcal{J}$ 和任何函子 $F: \mathcal{J} \to \mathcal{C}$，极限 $\lim F$ 存在），则称 $\mathcal{C}$ 是完备的。

- **余完备范畴**：如果范畴 $\mathcal{C}$ 有所有小余极限，则称 $\mathcal{C}$ 是余完备的。

- **双完备范畴**：如果范畴 $\mathcal{C}$ 既是完备的又是余完备的，则称 $\mathcal{C}$ 是双完备的。

大多数常见的范畴，如 Set, Grp, Top, Vect 等，都是双完备的。

### 6.3 有限完备范畴和有限余完备范畴

- **有限完备范畴**：如果范畴 $\mathcal{C}$ 有所有有限极限（即，对于任何有限范畴 $\mathcal{J}$ 和任何函子 $F: \mathcal{J} \to \mathcal{C}$，极限 $\lim F$ 存在），则称 $\mathcal{C}$ 是有限完备的。

- **有限余完备范畴**：如果范畴 $\mathcal{C}$ 有所有有限余极限，则称 $\mathcal{C}$ 是有限余完备的。

有一个重要的结果：一个范畴是有限完备的当且仅当它有所有有限积和所有等化子。类似地，一个范畴是有限余完备的当且仅当它有所有有限余积和所有余等化子。

## 7. 极限和余极限的计算

### 7.1 通过已知极限构造新极限

可以通过已知的极限来构造更复杂的极限：

1. **极限的极限**：如果 $F_i: \mathcal{J}_i \to \mathcal{C}$ 是一族函子，其中 $i$ 取自某个索引集 $I$，且所有极限 $\lim F_i$ 都存在，则 $\lim_{i \in I} (\lim_{j \in \mathcal{J}_i} F_i(j)) \cong \lim_{(i,j) \in \prod_{i \in I} \mathcal{J}_i} F_i(j)$。

2. **等化子的积**：等化子的积是积的等化子。

类似地，可以通过已知的余极限来构造更复杂的余极限。

### 7.2 通过伴随函子计算极限

如果 $F: \mathcal{C} \to \mathcal{D}$ 是左伴随（即存在右伴随 $G: \mathcal{D} \to \mathcal{C}$），则 $F$ 保持所有余极限。类似地，如果 $G: \mathcal{D} \to \mathcal{C}$ 是右伴随，则 $G$ 保持所有极限。

这一结果在计算极限和余极限时非常有用，因为它允许我们将计算转移到可能更简单的范畴中。

## 8. 极限和余极限的应用

### 8.1 代数拓扑中的应用

在代数拓扑中，极限和余极限用于定义和计算同调群和上同调群：

- **同调群**：可以通过链复形的余极限（具体来说，是商群）来定义。
- **上同调群**：可以通过上链复形的极限（具体来说，是子群）来定义。
- **Mayer-Vietoris 序列**：利用余极限（具体来说，是推出）来研究空间的同调。

### 8.2 代数几何中的应用

在代数几何中，极限和余极限用于构造和研究概形：

- **仿射概形的粘合**：通过余极限（具体来说，是推出）来构造一般概形。
- **层的限制和延拓**：通过极限和余极限来定义层的操作。

### 8.3 计算机科学中的应用

在计算机科学中，极限和余极限有多种应用：

- **数据类型**：积类型和和类型分别对应于积和余积。
- **程序语义**：通过极限和余极限来定义递归类型和程序的固定点语义。
- **并发理论**：使用极限和余极限来模型化并发系统的行为。

## 9. 练习与应用

### 9.1 计算极限

**练习**：在 Set 范畴中，计算以下极限：

1. 两个集合 $A$ 和 $B$ 的积。
2. 两个函数 $f, g: A \to B$ 的等化子。
3. 两个函数 $f: A \to C$ 和 $g: B \to C$ 的拉回。

### 9.2 计算余极限

**练习**：在 Grp 范畴中，计算以下余极限：

1. 两个群 $G$ 和 $H$ 的余积。
2. 两个群同态 $f, g: G \to H$ 的余等化子。
3. 两个群同态 $f: G \to H$ 和 $g: G \to K$ 的推出。

### 9.3 验证性质

**练习**：验证忘却函子 $U: \text{Grp} \to \text{Set}$ 保持极限但不保持余极限。
