# MWE｜李代数（最小例）

## 1. 目标

- 给出李代数的最小示例，展示李代数定义、李括号的计算。

## 2. 李代数定义

### 向量空间

- 载体：向量空间 g
- 运算：李括号 [·, ·]: g × g → g

### 公理

- 双线性：[ax + by, z] = a[x, z] + b[y, z]
- 反对称：[x, y] = -[y, x]
- 雅可比恒等式：[x, [y, z]] + [y, [z, x]] + [z, [x, y]] = 0

## 3. 经典李代数

### sl(2, R)

- 载体：2×2 实矩阵，迹为0
- 基：{e, f, h}
- 李括号：[h, e] = 2e, [h, f] = -2f, [e, f] = h

### so(3)

- 载体：3×3 反对称实矩阵
- 基：{X₁, X₂, X₃}
- 李括号：[X₁, X₂] = X₃, [X₂, X₃] = X₁, [X₃, X₁] = X₂

## 4. 李代数同态

### 同态定义

- φ: g → h 是线性映射
- 满足：φ([x, y]) = [φ(x), φ(y)]

### 例子

- ad: g → gl(g), ad(x)(y) = [x, y]
- 伴随表示

## 5. 练习（3）

1) 验证 sl(2, R) 的李括号满足雅可比恒等式。
2) 构造李代数 so(2) 的基和李括号。
3) 证明：李代数的伴随表示是李代数同态。
