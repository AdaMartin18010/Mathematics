## 1. 公理化体系

### 1.1 欧几里得公理

**公理 1.1.1 (点、线、面的存在性)**

1. 两点确定一条直线
2. 直线可以无限延长
3. 以任意点为圆心，任意距离为半径可以画圆
4. 所有直角都相等

**公理 1.1.2 (平行公理)**
过直线外一点，有且仅有一条直线与已知直线平行。

**公理 1.1.3 (连续性公理)**

1. 如果一条直线与两条直线相交，且在同一侧的内角之和小于两直角，则这两条直线在该侧延长后必相交
2. 直线是连续的，没有空隙

### 1.2 基本关系

**定义 1.2.1 (平行)**
两条直线在同一平面内，如果它们不相交，则称这两条直线平行。

**定义 1.2.2 (垂直)**
两条直线相交成直角，则称这两条直线垂直。

**定理 1.2.1 (平行线的性质)**
如果两条直线都与第三条直线平行，则这两条直线互相平行。

**证明**: 设 $l_1 \parallel l_3$，$l_2 \parallel l_3$。如果 $l_1$ 与 $l_2$ 相交于点 $P$，则过 $P$ 有两条直线都与 $l_3$ 平行，这与平行公理矛盾。

## 2. 基本概念与定义

### 2.1 基本几何图形

**定义 2.1.1 (线段)**
直线上两点之间的部分称为线段。

**定义 2.1.2 (射线)**
直线上一点向一个方向无限延伸的部分称为射线。

**定义 2.1.3 (角)**
两条射线从同一点出发所形成的图形称为角。

**定义 2.1.4 (三角形)**
由三条线段围成的平面图形称为三角形。

**定义 2.1.5 (四边形)**
由四条线段围成的平面图形称为四边形。

### 2.2 度量概念

**定义 2.2.1 (距离)**
两点之间的距离是连接这两点的线段长度。

**定义 2.2.2 (角度)**
角的大小用度数或弧度度量。

**定义 2.2.3 (面积)**
平面图形所占区域的大小称为面积。

## 3. 平面几何基础

### 3.1 直线与角

**定理 3.1.1 (对顶角相等)**
两条直线相交，对顶角相等。

**证明**: 设直线 $AB$ 与 $CD$ 相交于点 $O$。角 $AOC$ 和角 $BOD$ 是对顶角。由于角 $AOC + \angle AOD = 180°$，角 $BOD + \angle AOD = 180°$，因此 $\angle AOC = \angle BOD$。

**定理 3.1.2 (平行线的性质)**
两条平行直线被第三条直线所截，则：

1. 同位角相等
2. 内错角相等
3. 同旁内角互补

**证明**: 设 $l_1 \parallel l_2$，$l_3$ 与 $l_1, l_2$ 相交。过 $l_3$ 上一点作 $l_1$ 的平行线，由平行公理，这条线也与 $l_2$ 平行。因此同位角相等，其他性质由此可得。

### 3.2 三角形的基本性质

**定理 3.2.1 (三角形内角和)**
三角形的三个内角之和等于 $180°$。

**证明**: 过三角形的一个顶点作对边的平行线，利用平行线的性质可得。

**定理 3.2.2 (三角形外角定理)**
三角形的外角等于与它不相邻的两个内角之和。

**证明**: 设 $\triangle ABC$ 中，$\angle ACD$ 是外角。则 $\angle ACD + \angle ACB = 180°$，而 $\angle A + \angle B + \angle ACB = 180°$，因此 $\angle ACD = \angle A + \angle B$。

## 4. 三角形理论

### 4.1 全等三角形

**定义 4.1.1 (全等)**
两个三角形如果能够完全重合，则称这两个三角形全等。

**定理 4.1.1 (边角边全等)**
如果两个三角形的两边及其夹角分别相等，则这两个三角形全等。

**定理 4.1.2 (角边角全等)**
如果两个三角形的两角及其夹边分别相等，则这两个三角形全等。

**定理 4.1.3 (边边边全等)**
如果两个三角形的三边分别相等，则这两个三角形全等。

### 4.2 相似三角形

**定义 4.2.1 (相似)**
两个三角形如果对应角相等，对应边成比例，则称这两个三角形相似。

**定理 4.2.1 (角角相似)**
如果两个三角形的两个角分别相等，则这两个三角形相似。

**证明**: 设 $\triangle ABC$ 和 $\triangle DEF$ 中，$\angle A = \angle D$，$\angle B = \angle E$。则 $\angle C = \angle F$。由正弦定理，$\frac{AB}{DE} = \frac{BC}{EF} = \frac{AC}{DF}$。

**定理 4.2.2 (边角边相似)**
如果两个三角形的一个角相等，且夹这个角的两边成比例，则这两个三角形相似。

### 4.3 特殊三角形

**定义 4.3.1 (等腰三角形)**
有两条边相等的三角形称为等腰三角形。

**定理 4.3.1 (等腰三角形性质)**
等腰三角形的底角相等。

**证明**: 设 $\triangle ABC$ 中，$AB = AC$。作角平分线 $AD$，则 $\triangle ABD \cong \triangle ACD$（边角边），因此 $\angle B = \angle C$。

**定义 4.3.2 (等边三角形)**
三条边都相等的三角形称为等边三角形。

**定理 4.3.2 (等边三角形性质)**
等边三角形的三个角都等于 $60°$。

**定义 4.3.3 (直角三角形)**
有一个角是直角的三角形称为直角三角形。

**定理 4.3.3 (勾股定理)**
在直角三角形中，斜边的平方等于两直角边的平方和。

**证明**: 设 $\triangle ABC$ 中，$\angle C = 90°$。作高 $CD$，则 $\triangle ABC \sim \triangle ACD \sim \triangle CBD$。因此 $\frac{AC}{AB} = \frac{AD}{AC}$，$\frac{BC}{AB} = \frac{BD}{BC}$。两式相加得 $AC^2 + BC^2 = AB^2$。

## 5. 圆的理论

### 5.1 圆的基本性质

**定义 5.1.1 (圆)**
平面上到定点（圆心）距离相等的点的集合称为圆。

**定理 5.1.1 (圆的对称性)**
圆关于圆心对称，关于任意直径对称。

**定理 5.1.2 (圆周角定理)**
圆周角等于圆心角的一半。

**证明**: 设圆周角 $\angle ACB$ 对应圆心角 $\angle AOB$。连接 $OC$，则 $\angle AOC = 2\angle ABC$，$\angle BOC = 2\angle BAC$。因此 $\angle AOB = 2\angle ACB$。

### 5.2 圆与直线的关系

**定义 5.2.1 (切线)**
与圆只有一个公共点的直线称为圆的切线。

**定理 5.2.1 (切线性质)**
圆的切线与过切点的半径垂直。

**证明**: 设直线 $l$ 与圆 $O$ 相切于点 $P$。如果 $OP$ 不垂直于 $l$，则过 $O$ 作 $l$ 的垂线，垂足为 $Q$。则 $OQ < OP$，因此 $Q$ 在圆内，$l$ 与圆有两个交点，矛盾。

**定理 5.2.2 (切线长定理)**
从圆外一点到圆的两条切线长度相等。

**证明**: 设 $P$ 是圆外一点，$PA, PB$ 是切线。则 $\triangle OAP \cong \triangle OBP$（边角边），因此 $PA = PB$。

### 5.3 圆与圆的关系

**定义 5.3.1 (两圆的位置关系)**
两圆的位置关系有：相离、外切、相交、内切、内含。

**定理 5.3.1 (两圆相交)**
两圆相交当且仅当圆心距离小于两半径之和且大于两半径之差的绝对值。

## 6. 立体几何基础

### 6.1 基本立体图形

**定义 6.1.1 (多面体)**
由平面多边形围成的立体图形称为多面体。

**定义 6.1.2 (棱柱)**
两个全等的多边形平行放置，对应顶点用平行线段连接所形成的多面体称为棱柱。

**定义 6.1.3 (棱锥)**
一个多边形（底面）和一个点（顶点），顶点到底面各顶点的连线所形成的多面体称为棱锥。

**定义 6.1.4 (圆柱)**
一个圆沿垂直于圆面的方向平移所形成的立体图形称为圆柱。

**定义 6.1.5 (圆锥)**
一个直角三角形绕其一条直角边旋转所形成的立体图形称为圆锥。

### 6.2 立体几何的基本定理

**定理 6.2.1 (平行平面性质)**
两个平行平面被第三个平面所截，交线平行。

**定理 6.2.2 (三垂线定理)**
如果一条直线垂直于一个平面，则过这条直线上任意一点作平面的垂线，垂足在第一条直线在该平面上的射影上。

### 6.3 体积计算

**定理 6.3.1 (棱柱体积)**
棱柱的体积等于底面积乘以高。

**定理 6.3.2 (棱锥体积)**
棱锥的体积等于底面积乘以高的三分之一。

**定理 6.3.3 (圆柱体积)**
圆柱的体积等于底面积乘以高。

**定理 6.3.4 (圆锥体积)**
圆锥的体积等于底面积乘以高的三分之一。

**定理 6.3.5 (球体积)**
球的体积等于 $\frac{4}{3}\pi r^3$，其中 $r$ 是半径。

## 8. 应用与联系

### 8.1 在物理学中的应用

1. **力学**: 力的分解与合成
2. **光学**: 反射定律、折射定律
3. **天文学**: 行星轨道、天体运动

### 8.2 在工程学中的应用

1. **建筑学**: 结构设计、空间布局
2. **机械工程**: 齿轮设计、机构分析
3. **土木工程**: 桥梁设计、道路规划

### 8.3 在计算机科学中的应用

1. **计算机图形学**: 几何建模、渲染
2. **机器人学**: 路径规划、运动控制
3. **地理信息系统**: 空间分析、地图投影
