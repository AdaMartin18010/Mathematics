3. **多表征体系**：建立符号、图形、语言、程序等多种表征方式
4. **本地跳转链接**：建立完整的文件间和章节间跳转链接

## 3. 核心主题

### 3.1 数系演化理论
- **自然数系统**：皮亚诺公理和归纳原理
- **整数系统**：整数的构造和性质
- **有理数系统**：有理数的构造和稠密性
- **实数系统**：实数的构造和完备性
- **复数系统**：复数的构造和代数闭性
- **超实数系统**：非标准分析中的超实数

### 3.2 极限理论
- **数列极限**：数列收敛的定义和性质
- **函数极限**：函数极限的定义和性质
- **单侧极限**：左极限和右极限
- **无穷极限**：无穷大和无穷小的极限
- **极限运算**：极限的四则运算和复合运算

### 3.3 连续性理论
- **函数连续性**：连续函数的定义和性质
- **一致连续性**：一致连续的定义和性质
- **间断点分类**：第一类、第二类间断点
- **连续函数性质**：介值定理、最值定理等

### 3.4 微分学
- **导数定义**：导数的几何和物理意义
- **导数计算**：基本求导法则和技巧
- **高阶导数**：二阶导数和高阶导数
- **微分应用**：极值、单调性、凹凸性
- **泰勒展开**：泰勒级数和麦克劳林级数

### 3.5 积分学
- **定积分**：黎曼积分的定义和性质
- **不定积分**：原函数和不定积分
- **积分技巧**：换元积分、分部积分等
- **反常积分**：无穷积分和瑕积分
- **积分应用**：面积、体积、弧长等

### 3.6 非标准微积分
- **超实数系统**：非标准实数域
- **无穷小分析**：无穷小和无穷大的性质
- **非标准导数**：非标准导数定义
- **非标准积分**：非标准积分定义
- **转换原理**：标准与非标准之间的转换

### 3.7 多变量微积分
- **偏导数**：多变量函数的偏导数
- **方向导数**：方向导数和梯度
- **多重积分**：二重积分和三重积分
- **曲线积分**：第一类和第二类曲线积分
- **曲面积分**：第一类和第二类曲面积分

### 3.8 微分方程
- **常微分方程**：一阶和二阶常微分方程
- **线性微分方程**：齐次和非齐次线性方程
- **微分方程解法**：分离变量、积分因子等
- **初值问题**：初值问题的存在唯一性
- **边值问题**：边值问题的解法和性质

## 4. 学习路径

### 4.1 基础阶段
1. **数系基础**：自然数、整数、有理数、实数
2. **极限概念**：数列极限、函数极限
3. **连续性**：函数连续性、一致连续性
4. **导数基础**：导数定义、基本求导法则

### 4.2 进阶阶段
1. **积分基础**：定积分、不定积分
2. **微分应用**：极值、单调性、凹凸性
3. **积分技巧**：换元积分、分部积分
4. **泰勒展开**：泰勒级数和应用

### 4.3 高级阶段
1. **多变量微积分**：偏导数、多重积分
2. **微分方程**：常微分方程和解法
3. **非标准分析**：超实数和无穷小分析
4. **实分析**：测度论和勒贝格积分

### 4.4 应用阶段
1. **物理应用**：力学、电磁学中的应用
2. **工程应用**：控制理论、信号处理
3. **经济应用**：边际分析、优化理论
4. **生物应用**：种群动力学、流行病学
