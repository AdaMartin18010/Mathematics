# MWE｜代数几何深化（最小例）

## 1. 目标

- 给出代数几何深化的最小示例，展示概形、上同调的应用。

## 2. 概形

### 仿射概形

- Spec A = {素理想 p ⊂ A}
- 拓扑：闭集 V(I) = {p | I ⊂ p}
- 结构层：O_{Spec A}(D(f)) = A_f
- 示例：Spec k[x] 是仿射直线

### 概形

- 局部仿射的环层空间
- 概形间的态射：环层空间的态射
- 示例：P¹_k 是射影直线概形

## 3. 上同调

### 切赫上同调

- H^i(X,F) = lim H^i(U,F)
- 其中 U 是开覆盖
- 示例：H¹(P¹,O) = 0
- H¹(P¹,O(1)) = k

### 塞尔对偶

- H^i(X,F) ≅ H^{n-i}(X,F^∨ ⊗ ω_X)
- 其中 ω_X 是典范层
- 示例：在 P^n 上，ω_{P^n} = O(-n-1)

## 4. 相交理论

### 相交数

- 两个子概形的相交数
- 贝祖定理：两条平面曲线的相交数
- 示例：直线与圆锥曲线的相交数

### 陈类

- 向量丛的陈类
- c₁(E) ∈ H²(X,Z)
- 示例：切丛的陈类

## 5. 练习（3）

1) 计算仿射概形 Spec k[x,y]/(xy) 的素理想。
2) 证明：P¹ 上的线丛 O(n) 的切赫上同调。
3) 计算两条直线在 P² 中的相交数。
