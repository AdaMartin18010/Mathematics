## 1. 概述

### 1.1 数论与离散数学的地位

数论与离散数学是数学的重要分支，研究离散结构和整数性质。它们在计算机科学、密码学、算法设计等领域有广泛应用。

**核心特征**：
- **离散性**：研究对象是离散的、可数的
- **精确性**：每个结论都有严格的证明
- **应用性**：与计算机科学紧密相关
- **抽象性**：从具体问题抽象出一般规律

### 1.2 主要分支

1. **初等数论**：研究整数的基本性质
2. **代数数论**：将代数方法应用于数论问题
3. **解析数论**：使用分析方法研究数论
4. **组合数学**：研究离散结构的计数和排列
5. **图论**：研究图的结构和性质
6. **离散优化**：研究离散优化问题

## 5. 重要定理

### 5.1 数论核心定理

**算术基本定理**：
每个大于1的整数都可以唯一地分解为素数的乘积。

**费马小定理**：
如果 $p$ 是素数，$a$ 是不能被 $p$ 整除的整数，则 $a^{p-1} \equiv 1 \pmod{p}$。

**中国剩余定理**：
设 $m_1, m_2, \ldots, m_k$ 是两两互素的正整数，对于任意整数 $a_1, a_2, \ldots, a_k$，同余方程组有唯一解模 $M = m_1m_2\cdots m_k$。

**二次互反律**：
对于不同的奇素数 $p, q$，有：
$$\left(\frac{p}{q}\right)\left(\frac{q}{p}\right) = (-1)^{\frac{p-1}{2}\frac{q-1}{2}}$$

### 5.2 组合数学核心定理

**二项式定理**：
$$(a + b)^n = \sum_{r=0}^n \binom{n}{r} a^{n-r} b^r$$

**容斥原理**：
$$\left|\bigcup_{i=1}^n A_i\right| = \sum_{i=1}^n |A_i| - \sum_{1 \leq i < j \leq n} |A_i \cap A_j| + \cdots$$

**鸽巢原理**：
如果将 $n$ 个物体放入 $m$ 个盒子中，且 $n > m$，则至少有一个盒子包含至少 $\lceil\frac{n}{m}\rceil$ 个物体。

### 5.3 图论核心定理

**握手定理**：
对于任意图 $G = (V, E)$，有 $\sum_{v \in V} \deg(v) = 2|E|$。

**树的性质**：
$n$ 个顶点的树有 $n-1$ 条边，且任意两个顶点之间有唯一路径。

## 6. 算法与计算

### 6.1 数论算法

**欧几里得算法**：
```rust
fn gcd(mut a: i64, mut b: i64) -> i64 {
    while b != 0 {
        let temp = b;
        b = a % b;
        a = temp;
    }
    a
}
```

**扩展欧几里得算法**：
```rust
fn extended_gcd(a: i64, b: i64) -> (i64, i64, i64) {
    if b == 0 {
        (a, 1, 0)
    } else {
        let (gcd, x, y) = extended_gcd(b, a % b);
        (gcd, y, x - (a / b) * y)
    }
}
```

**快速幂算法**：
```rust
fn fast_pow(mut base: i64, mut exp: i64, modulus: i64) -> i64 {
    let mut result = 1;
    base %= modulus;
    while exp > 0 {
        if exp % 2 == 1 {
            result = (result * base) % modulus;
        }
        base = (base * base) % modulus;
        exp /= 2;
    }
    result
}
```

### 6.2 组合算法

**组合数计算**：
```rust
fn combination(n: i64, r: i64, modulus: i64) -> i64 {
    if r > n { return 0; }
    if r > n - r { r = n - r; }
    
    let mut result = 1;
    for i in 0..r {
        result = (result * (n - i)) % modulus;
        result = (result * mod_inverse(i + 1, modulus)) % modulus;
    }
    result
}
```

**卡特兰数计算**：
```rust
fn catalan(n: i64, modulus: i64) -> i64 {
    let c = combination(2 * n, n, modulus);
    (c * mod_inverse(n + 1, modulus)) % modulus
}
```

### 6.3 图论算法

**深度优先搜索**：
```rust
fn dfs(graph: &Vec<Vec<usize>>, start: usize, visited: &mut Vec<bool>) {
    visited[start] = true;
    for &neighbor in &graph[start] {
        if !visited[neighbor] {
            dfs(graph, neighbor, visited);
        }
    }
}
```

**广度优先搜索**：
```rust
fn bfs(graph: &Vec<Vec<usize>>, start: usize) -> Vec<usize> {
    let mut visited = vec![false; graph.len()];
    let mut queue = std::collections::VecDeque::new();
    let mut result = Vec::new();
    
    queue.push_back(start);
    visited[start] = true;
    
    while let Some(node) = queue.pop_front() {
        result.push(node);
        for &neighbor in &graph[node] {
            if !visited[neighbor] {
                visited[neighbor] = true;
                queue.push_back(neighbor);
            }
        }
    }
    result
}
```

## 7. 历史发展

### 7.1 古代数论

**古希腊时期**：
- 欧几里得：最大公约数算法
- 毕达哥拉斯：完全数、亲和数
- 丢番图：不定方程

**中国古代**：
- 《九章算术》：同余理论
- 孙子定理：中国剩余定理
- 秦九韶：大衍求一术

### 7.2 近代发展

**17-18世纪**：
- 费马：费马小定理、费马大定理
- 欧拉：欧拉函数、欧拉定理
- 拉格朗日：二次剩余理论

**19世纪**：
- 高斯：二次互反律、高斯整数
- 黎曼：黎曼猜想、解析数论
- 戴德金：理想理论

**20世纪**：
- 哈代-李特尔伍德：解析数论方法
- 塞尔伯格：素数定理初等证明
