## 1. 引言

概率论是研究随机现象数量规律的数学分支，为统计学、机器学习、金融学等领域提供理论基础。

### 1.1 概率论的发展历程

```text
古典概率 → 公理化概率 → 现代概率论 → 随机过程
    ↓         ↓           ↓         ↓
  组合计数   测度论基础   随机变量   时间序列
```

## 2. 概率空间

### 2.1 样本空间

**定义**：随机试验所有可能结果的集合，记作 Ω。

**示例**：

- 掷骰子：Ω = {1, 2, 3, 4, 5, 6}
- 掷硬币：Ω = {H, T}
- 测量温度：Ω = ℝ

### 2.2 σ-代数

**定义**：Ω 的子集族 ℱ 是 σ-代数，如果：

1. Ω ∈ ℱ
2. A ∈ ℱ ⇒ Aᶜ ∈ ℱ
3. A₁, A₂, ... ∈ ℱ ⇒ ∪ᵢAᵢ ∈ ℱ

**示例**：

- 离散情况：ℱ = 2^Ω
- 连续情况：Borel σ-代数

### 2.3 概率测度

**定义**：函数 P: ℱ → [0,1] 是概率测度，如果：

1. P(Ω) = 1
2. 可列可加性：A₁, A₂, ... 互不相交 ⇒ P(∪ᵢAᵢ) = ΣᵢP(Aᵢ)

**性质**：

- P(∅) = 0
- P(Aᶜ) = 1 - P(A)
- P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

## 3. 随机变量

### 3.1 随机变量的定义

**定义**：随机变量 X 是从 (Ω, ℱ) 到 (ℝ, ℬ) 的可测函数。

**分类**：

- **离散随机变量**：取值可数
- **连续随机变量**：取值连续
- **混合随机变量**：既有离散又有连续部分

### 3.2 分布函数

**定义**：F(x) = P(X ≤ x)

**性质**：

1. 单调不减
2. 右连续
3. lim_{x→-∞} F(x) = 0
4. lim_{x→+∞} F(x) = 1

### 3.3 概率质量函数

**定义**：离散随机变量 X 的 PMF 为 p(x) = P(X = x)

**性质**：

- p(x) ≥ 0
- Σᵢ p(xᵢ) = 1

### 3.4 概率密度函数

**定义**：连续随机变量 X 的 PDF 为 f(x)，满足：
$$F(x) = \int_{-\infty}^x f(t) dt$$

**性质**：

- f(x) ≥ 0
- ∫_{-∞}^{+∞} f(x) dx = 1

## 4. 分布理论

### 4.1 离散分布

#### 4.1.1 伯努利分布

**PMF**：P(X = k) = pᵏ(1-p)¹⁻ᵏ, k = 0,1

**参数**：p ∈ [0,1]

**应用**：二分类问题

#### 4.1.2 二项分布

**PMF**：P(X = k) = C(n,k) pᵏ(1-p)ⁿ⁻ᵏ

**参数**：n ∈ ℕ, p ∈ [0,1]

**应用**：n 次独立伯努利试验的成功次数

#### 4.1.3 泊松分布

**PMF**：P(X = k) = λᵏe⁻ᵏ/k!

**参数**：λ > 0

**应用**：稀有事件的发生次数

### 4.2 连续分布

#### 4.2.1 均匀分布

**PDF**：f(x) = 1/(b-a), x ∈ [a,b]

**参数**：a < b

**应用**：随机选择

#### 4.2.2 正态分布

**PDF**：f(x) = (1/√(2πσ²)) exp(-(x-μ)²/(2σ²))

**参数**：μ ∈ ℝ, σ > 0

**应用**：误差分布，自然现象

#### 4.2.3 指数分布

**PDF**：f(x) = λe⁻ᵏˣ, x ≥ 0

**参数**：λ > 0

**应用**：等待时间，寿命分布

## 5. 期望与方差

### 5.1 数学期望

**定义**：

- 离散：E[X] = Σᵢ xᵢ p(xᵢ)
- 连续：E[X] = ∫_{-∞}^{+∞} x f(x) dx

**性质**：

- 线性性：E[aX + bY] = aE[X] + bE[Y]
- 单调性：X ≤ Y ⇒ E[X] ≤ E[Y]

### 5.2 方差

**定义**：Var(X) = E[(X - E[X])²]

**计算**：Var(X) = E[X²] - (E[X])²

**性质**：

- Var(aX + b) = a²Var(X)
- 非负性：Var(X) ≥ 0

### 5.3 协方差

**定义**：Cov(X,Y) = E[(X - E[X])(Y - E[Y])]

**性质**：

- 对称性：Cov(X,Y) = Cov(Y,X)
- 双线性：Cov(aX + bY, Z) = aCov(X,Z) + bCov(Y,Z)

## 6. 大数定律与中心极限定理

### 6.1 大数定律

**弱大数定律**：设 X₁, X₂, ... 独立同分布，E[X₁] = μ，则：
$$\frac{X_1 + X_2 + \cdots + X_n}{n} \xrightarrow{P} \mu$$

**强大数定律**：在相同条件下：
$$\frac{X_1 + X_2 + \cdots + X_n}{n} \xrightarrow{a.s.} \mu$$

### 6.2 中心极限定理

**定理**：设 X₁, X₂, ... 独立同分布，E[X₁] = μ，Var(X₁) = σ²，则：
$$\frac{X_1 + X_2 + \cdots + X_n - n\mu}{\sqrt{n}\sigma} \xrightarrow{d} N(0,1)$$

**应用**：

- 样本均值的分布
- 误差分析
- 统计推断
