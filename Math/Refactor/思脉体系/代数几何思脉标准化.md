# 代数几何思脉标准化

## 国际标准定义补全

### 严格数学定义

#### 核心概念体系

**代数簇 (Algebraic Variety)**:

- **形式定义**: 仿射空间𝔸ⁿ中多项式方程组的零点集V(f₁,...,fₘ) = {x ∈ 𝔸ⁿ | fᵢ(x) = 0, ∀i}
- **射影定义**: 射影空间ℙⁿ中齐次多项式方程组的零点集
- **几何意义**: 代数几何研究的基本对象，描述几何与代数的统一

**概形 (Scheme)**:

- **形式定义**: 局部环化空间(X,𝒪_X)，其中𝒪_X是结构层
- **仿射概形**: Spec A，其中A是交换环
- **几何意义**: 代数几何的现代语言，统一处理代数与几何

**层论 (Sheaf Theory)**:

- **形式定义**: 拓扑空间X上的层F是满足局部化公理的预层
- **结构层**: 𝒪_X，定义概形的代数结构
- **几何意义**: 提供局部到整体的桥梁

**上同调理论 (Cohomology Theory)**:

- **形式定义**: Hⁱ(X,ℱ) = RⁱΓ(X,ℱ)，其中Γ是整体截面函子
- **Čech上同调**: 基于开覆盖的上同调计算
- **几何意义**: 研究几何对象的拓扑不变量

**相交理论 (Intersection Theory)**:

- **形式定义**: 在代数簇上定义有理等价类和相交积
- **贝祖定理**: 两个平面曲线的相交点数等于次数乘积
- **几何意义**: 研究代数簇的相交性质

#### 核心定理体系

**希尔伯特零点定理 (Hilbert's Nullstellensatz)**:

- **形式表述**: I(V(I)) = √I，其中I是理想，V(I)是零点集
- **几何意义**: 代数与几何的对应关系

**黎曼-罗赫定理 (Riemann-Roch Theorem)**:

- **形式表述**: χ(ℒ) = deg(ℒ) + 1 - g，其中ℒ是线丛，g是亏格
- **几何意义**: 线丛的拓扑性质与几何性质的联系

**格罗滕迪克对偶性 (Grothendieck Duality)**:

- **形式表述**: Extⁱ(ℱ,ω_X) ≅ H^{n-i}(X,ℱ)*，其中ω_X是典范层
- **几何意义**: 上同调的对偶性质

**韦尔猜想 (Weil Conjectures)**:

- **形式表述**: Z(X,t) = ∏ᵢ Pᵢ(t)^{(-1)^{i+1}}，其中Pᵢ(t)是特征多项式
- **几何意义**: 有限域上代数簇的ζ函数性质
