# 数值分析思脉标准化

## 国际标准定义补全

### 严格数学定义

#### 核心概念体系

**数值方法 (Numerical Methods)**:

- **形式定义**: 用有限精度算术求解数学问题的算法
- **误差分析**: 研究数值误差的来源和传播
- **稳定性**: 算法对输入扰动的敏感程度
- **几何意义**: 用离散方法逼近连续问题

**插值理论 (Interpolation Theory)**:

- **形式定义**: 构造通过给定数据点的函数
- **拉格朗日插值**: P(x) = Σᵢ yᵢ Lᵢ(x)，其中Lᵢ(x)是拉格朗日基函数
- **样条插值**: 分段多项式插值，保证光滑性
- **几何意义**: 在数据点间构造光滑曲线

**数值积分 (Numerical Integration)**:

- **形式定义**: 用数值方法近似计算定积分
- **梯形法则**: ∫ᵇₐ f(x)dx ≈ (b-a)/2[f(a) + f(b)]
- **辛普森法则**: ∫ᵇₐ f(x)dx ≈ (b-a)/6[f(a) + 4f((a+b)/2) + f(b)]
- **几何意义**: 用简单几何形状近似曲线下面积

**数值微分 (Numerical Differentiation)**:

- **形式定义**: 用数值方法近似计算导数
- **前向差分**: f'(x) ≈ [f(x+h) - f(x)]/h
- **中心差分**: f'(x) ≈ [f(x+h) - f(x-h)]/(2h)
- **几何意义**: 用割线斜率近似切线斜率

**线性方程组求解 (Linear System Solving)**:

- **形式定义**: 求解Ax = b的数值方法
- **高斯消元**: 通过行变换将矩阵化为上三角形式
- **LU分解**: A = LU，其中L是下三角，U是上三角
- **迭代方法**: 雅可比、高斯-赛德尔、共轭梯度法

#### 核心定理体系

**误差分析定理 (Error Analysis Theorems)**:

- **形式表述**: 数值误差的上界估计
- **舍入误差**: 浮点运算引起的误差
- **截断误差**: 近似方法引起的误差
- **几何意义**: 数值方法的精度保证

**收敛性定理 (Convergence Theorems)**:

- **形式表述**: 数值方法收敛的条件和速率
- **线性收敛**: |e_{n+1}| ≤ c|e_n|，其中c < 1
- **二次收敛**: |e_{n+1}| ≤ c|e_n|²
- **几何意义**: 迭代方法的收敛性质

**稳定性定理 (Stability Theorems)**:

- **形式表述**: 算法对扰动的稳定性条件
- **条件数**: κ(A) = ‖A‖‖A⁻¹‖
- **向后误差分析**: 实际误差的向后估计
- **几何意义**: 算法的数值稳定性

**插值误差定理 (Interpolation Error Theorems)**:

- **形式表述**: 插值误差的上界估计
- **拉格朗日插值误差**: |f(x) - P(x)| ≤ M_{n+1}/(n+1)!|ω(x)|
- **几何意义**: 插值精度的理论保证
