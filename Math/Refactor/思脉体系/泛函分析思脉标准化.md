# 泛函分析思脉标准化

## 国际标准定义补全

### 严格数学定义

#### 核心概念体系

**巴拿赫空间 (Banach Space)**:

- **形式定义**: 完备的赋范线性空间(X,‖·‖)，满足柯西序列收敛
- **公理化表述**:
  - X是向量空间
  - ‖·‖是范数，满足三角不等式
  - 每个柯西序列都收敛到X中的元素
- **几何意义**: 无限维向量空间的几何结构

**希尔伯特空间 (Hilbert Space)**:

- **形式定义**: 完备的内积空间(H,⟨·,·⟩)，诱导的范数‖x‖ = √⟨x,x⟩
- **正交性**: 向量x,y正交若⟨x,y⟩ = 0
- **几何意义**: 具有角度和距离概念的无限维空间

**线性算子 (Linear Operator)**:

- **形式定义**: 线性映射T: X → Y，满足T(αx + βy) = αT(x) + βT(y)
- **有界性**: ‖T‖ = sup{‖Tx‖ : ‖x‖ ≤ 1} < ∞
- **紧性**: 将有界集映射为相对紧集

**对偶空间 (Dual Space)**:

- **形式定义**: X* = {f: X → ℝ | f是连续线性泛函}
- **弱拓扑**: 由对偶空间定义的拓扑
- **几何意义**: 描述原空间的线性函数结构

**谱理论 (Spectral Theory)**:

- **形式定义**: 算子T的谱σ(T) = {λ ∈ ℂ | T - λI不可逆}
- **特征值**: 点谱σ_p(T) = {λ | ker(T - λI) ≠ {0}}
- **几何意义**: 描述算子的代数性质

#### 核心定理体系

**哈恩-巴拿赫定理 (Hahn-Banach Theorem)**:

- **形式表述**: 在子空间上定义的线性泛函可延拓到整个空间
- **几何意义**: 保证线性泛函的存在性和延拓性

**开映射定理 (Open Mapping Theorem)**:

- **形式表述**: 有界满射线性算子T是开映射
- **推论**: 有界双射线性算子的逆算子有界

**闭图像定理 (Closed Graph Theorem)**:

- **形式表述**: 闭算子在巴拿赫空间间有界
- **几何意义**: 算子的连续性与其图像的闭性等价

**里斯表示定理 (Riesz Representation Theorem)**:

- **形式表述**: 希尔伯特空间H上的连续线性泛函f可表示为f(x) = ⟨x,y⟩
- **几何意义**: 希尔伯特空间与其对偶空间等距同构

**谱定理 (Spectral Theorem)**:

- **形式表述**: 自伴紧算子有正交特征向量基
- **几何意义**: 自伴算子的对角化理论
