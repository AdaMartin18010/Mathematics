## 🌐 国际标准定义

### 测度论 (Measure Theory)

**严格数学定义**: 测度论是研究集合上测度函数的数学分支。测度μ是定义在σ-代数F上的函数μ: F → [0,∞]，满足：μ(∅) = 0；对互斥集合A_i有μ(∪A_i) = Σμ(A_i)。重要的测度包括：勒贝格测度，R^n上的标准测度；计数测度，μ(A) = |A|；狄拉克测度，μ(A) = 1若0 ∈ A，否则为0。测度论的核心定理包括：单调收敛定理、法图引理、控制收敛定理等。

**认知科学视角**: 测度论反映了人类认知的一个重要特征：我们能够将"大小"、"长度"、"面积"、"体积"等直观概念推广到抽象集合。这种推广能力反映了人类认知系统对抽象化的偏好。测度论的概念（如可测集、测度）虽然抽象，但都有直观的几何意义，这反映了人类认知系统对几何直觉的依赖。

**哲学反思**: 测度论的发展反映了数学对"大小"概念的深刻哲学思考。什么是"大小"？测度是否反映了客观的几何性质，还是仅仅是数学构造？这些问题触及了数学哲学的核心。

### 测度 (Measure)

**严格数学定义**: 测度μ是定义在σ-代数F上的函数μ: F → [0,∞]，满足：μ(∅) = 0；对互斥集合A_i有μ(∪A_i) = Σμ(A_i)。测度空间是三元组(X,F,μ)，其中X是集合，F是σ-代数，μ是测度。测度的性质包括：单调性，A ⊆ B ⇒ μ(A) ≤ μ(B)；可数可加性，μ(∪A_i) = Σμ(A_i)对互斥集合；连续性，A_n ↑ A ⇒ μ(A_n) → μ(A)。

**认知科学视角**: 测度反映了人类认知的一个重要特征：我们能够将"大小"的概念从具体的几何对象推广到抽象的集合。这种推广能力反映了人类认知系统对模式识别和抽象化的偏好。测度的概念也展示了人类如何通过数学工具来处理复杂的几何问题。

**哲学反思**: 测度引发了关于"大小"本质的深刻哲学问题。抽象的测度是否反映了客观的几何性质？这种抽象化是否失去了直观性？这些问题触及了数学抽象化的哲学意义。

### 勒贝格积分 (Lebesgue Integral)

**严格数学定义**: 勒贝格积分是相对于测度μ的积分。对简单函数f = Σa_iχ_A_i，勒贝格积分定义为∫fdμ = Σa_iμ(A_i)。对非负可测函数f，勒贝格积分定义为∫fdμ = sup{∫φdμ : φ ≤ f, φ简单函数}。对一般可测函数f = f^+ - f^-，勒贝格积分定义为∫fdμ = ∫f^+dμ - ∫f^-dμ。勒贝格积分的性质包括：线性性、单调性、绝对连续性等。

**认知科学视角**: 勒贝格积分反映了人类认知的一个重要特征：我们能够将"面积"的概念推广到复杂的函数。这种推广能力反映了人类认知系统对几何直觉的依赖。勒贝格积分的概念也展示了人类如何通过数学工具来处理复杂的分析问题。

**哲学反思**: 勒贝格积分引发了关于积分本质的深刻哲学问题。什么是"积分"？勒贝格积分是否比黎曼积分更"自然"？这些问题触及了分析学哲学的核心。

### 可测函数 (Measurable Function)

**严格数学定义**: 函数f: X → R是可测的若对任意开集U ⊆ R，f^(-1)(U) ∈ F。可测函数的性质包括：可测函数的和、积、商（分母不为零）都是可测的；可测函数的极限是可测的；连续函数是可测的。可测函数空间L^p(X,μ)是所有满足∫|f|^pdμ < ∞的可测函数f的集合。

**认知科学视角**: 可测函数反映了人类认知的一个重要特征：我们能够将"函数"的概念推广到抽象的空间。这种推广能力反映了人类认知系统对抽象化的偏好。可测函数的概念也展示了人类如何通过数学工具来处理复杂的函数问题。

**哲学反思**: 可测函数引发了关于函数本质的深刻哲学问题。什么是"函数"？可测性是否反映了函数的某种"自然"性质？这些问题触及了函数论哲学的核心。

### 收敛定理 (Convergence Theorems)

**严格数学定义**: 测度论中的收敛定理包括：单调收敛定理，若0 ≤ f_n ↑ f，则∫f_ndμ → ∫fdμ；法图引理，若f_n ≥ 0，则∫lim inf f_ndμ ≤ lim inf ∫f_ndμ；控制收敛定理，若|f_n| ≤ g且∫gdμ < ∞，f_n → f a.e.，则∫f_ndμ → ∫fdμ；有界收敛定理，若|f_n| ≤ M且f_n → f a.e.，则∫f_ndμ → ∫fdμ。

**认知科学视角**: 收敛定理反映了人类认知的一个重要特征：我们能够将极限的概念推广到积分运算。这种推广能力反映了人类认知系统对推理和逻辑的偏好。收敛定理也展示了人类如何通过数学工具来处理复杂的极限问题。

**哲学反思**: 收敛定理引发了关于极限本质的深刻哲学问题。什么是"收敛"？积分与极限的交换是否总是合理的？这些问题触及了分析学哲学的核心。

## 📖 目录结构

### 第一部分：测度空间基础

#### 1.1 σ-代数

**定义 1.1.1** (σ-代数). 集合X的子集族F是σ-代数若：

- X ∈ F
- A ∈ F ⇒ A^c ∈ F  
- A_i ∈ F ⇒ ∪A_i ∈ F

**定义 1.1.2** (生成σ-代数). 由集合族A生成的σ-代数σ(A)是包含A的最小σ-代数。

**定义 1.1.3** (博雷尔σ-代数). 博雷尔σ-代数B(R)是由开集生成的σ-代数。

#### 1.2 测度的定义

**定义 1.2.1** (测度). 测度μ: F → [0,∞]满足：

- μ(∅) = 0
- μ(∪A_i) = Σμ(A_i)对互斥集合

**定义 1.2.2** (测度空间). 测度空间是三元组(X,F,μ)。

**定义 1.2.3** (完全测度). 测度完全若零集的子集都可测。

### 第二部分：勒贝格测度

#### 2.1 外测度

**定义 2.1.1** (外测度). 外测度μ*: P(R^n) → [0,∞]定义为：
μ*(A) = inf{Σ|I_i| : A ⊆ ∪I_i, I_i为开区间}

**定义 2.1.2** (勒贝格可测集). 集合A勒贝格可测若对任意E ⊆ R^n，
μ*(E) = μ*(E∩A) + μ*(E∩A^c)

**定理 2.1.3** (卡拉西奥多里定理). 勒贝格可测集构成σ-代数。

#### 2.2 勒贝格测度的性质

**定理 2.2.1** (平移不变性). 勒贝格测度在平移下不变。

**定理 2.2.2** (正则性). 对勒贝格可测集A，存在开集G和闭集F使得F ⊆ A ⊆ G且μ(G\F) < ε。

### 第三部分：可测函数

#### 3.1 可测函数的定义

**定义 3.1.1** (可测函数). 函数f: X → R可测若对任意开集U，f^(-1)(U) ∈ F。

**定义 3.1.2** (简单函数). 简单函数是有限个特征函数的线性组合。

**定理 3.1.3** (可测函数的构造). 任意非负可测函数都是简单函数的递增极限。

#### 3.2 可测函数的性质

**定理 3.2.1** (可测函数的运算). 可测函数的和、积、商（分母不为零）都是可测的。

**定理 3.2.2** (可测函数的极限). 可测函数序列的极限是可测的。

### 第四部分：勒贝格积分

#### 4.1 积分的定义

**定义 4.1.1** (简单函数积分). 对简单函数φ = Σa_iχ_A_i，
∫φdμ = Σa_iμ(A_i)

**定义 4.1.2** (非负函数积分). 对非负可测函数f，
∫fdμ = sup{∫φdμ : φ ≤ f, φ简单函数}

**定义 4.1.3** (一般函数积分). 对可测函数f = f^+ - f^-，
∫fdμ = ∫f^+dμ - ∫f^-dμ

#### 4.2 积分的性质

**定理 4.2.1** (积分的线性性). ∫(af + bg)dμ = a∫fdμ + b∫gdμ

**定理 4.2.2** (积分的单调性). f ≤ g ⇒ ∫fdμ ≤ ∫gdμ

**定理 4.2.3** (积分的绝对连续性). |∫fdμ| ≤ ∫|f|dμ

### 第五部分：收敛定理

#### 5.1 单调收敛定理

**定理 5.1.1** (单调收敛定理). 若0 ≤ f_n ↑ f，则∫f_ndμ → ∫fdμ。

**定理 5.1.2** (法图引理). 若f_n ≥ 0，则∫lim inf f_ndμ ≤ lim inf ∫f_ndμ。

**定理 5.1.3** (控制收敛定理). 若|f_n| ≤ g且∫gdμ < ∞，f_n → f a.e.，则∫f_ndμ → ∫fdμ。

#### 5.2 有界收敛定理

**定理 5.2.1** (有界收敛定理). 若|f_n| ≤ M且f_n → f a.e.，则∫f_ndμ → ∫fdμ。

**定理 5.2.2** (依测度收敛). 若f_n → f依测度，则存在子序列f_{n_k} → f a.e.

### 第六部分：L^p空间

#### 6.1 L^p空间的定义

**定义 6.1.1** (L^p空间). L^p(X,μ)是所有满足∫|f|^pdμ < ∞的可测函数f的集合。

**定义 6.1.2** (L^p范数). ‖f‖_p = (∫|f|^pdμ)^(1/p)

**定义 6.1.3** (L^∞空间). L^∞(X,μ)是所有本质有界可测函数的集合。

#### 6.2 L^p空间的性质

**定理 6.2.1** (赫尔德不等式). 若1/p + 1/q = 1，则|∫fgdμ| ≤ ‖f‖_p‖g‖_q

**定理 6.2.2** (闵可夫斯基不等式). ‖f + g‖_p ≤ ‖f‖_p + ‖g‖_p

**定理 6.2.3** (里斯-费舍尔定理). L^p空间是完备的。

### 第七部分：乘积测度

#### 7.1 乘积测度的定义

**定义 7.1.1** (乘积σ-代数). F⊗G是由{A×B : A ∈ F, B ∈ G}生成的σ-代数。

**定义 7.1.2** (乘积测度). 乘积测度μ⊗ν定义为(μ⊗ν)(A×B) = μ(A)ν(B)。

**定理 7.1.3** (富比尼定理). 若f ∈ L^1(μ⊗ν)，则∫fd(μ⊗ν) = ∫∫f(x,y)dμ(x)dν(y)

#### 7.2 富比尼定理

**定理 7.2.1** (富比尼定理). 对非负可测函数f，∫fd(μ⊗ν) = ∫∫f(x,y)dμ(x)dν(y)

**定理 7.2.2** (富比尼定理的推广). 对f ∈ L^1(μ⊗ν)，∫fd(μ⊗ν) = ∫∫f(x,y)dμ(x)dν(y)
