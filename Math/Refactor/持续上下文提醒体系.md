- 群运算：⋅, *, ∘
- 单位元：e, 1, 0
- 逆元：a⁻¹, -a
- 同态：φ, ψ, f, g

**分析符号**：
- 极限：lim
- 导数：f', df/dx, Df
- 积分：∫
- 级数：∑

#### 4.1.2 定义标准

**数学定义格式**：
```markdown
**定义 X.Y.Z (概念名称)**
设...，如果...，则称...为...，记作...

**性质**：
1. 性质1
2. 性质2

**定理 X.Y.Z (定理名称)**
设...，则...

**证明**：
1. 步骤1
2. 步骤2
...
```

### 4.2 关联性维护

#### 4.2.1 跨分支联系

**集合论→代数**：
- 群、环、域都是集合上的代数结构
- 同态映射是保持结构的函数
- 商结构通过等价关系构造

**代数→分析**：
- 向量空间是线性代数的研究对象
- 线性变换是分析中的重要工具
- 群论在微分方程中有应用

**分析→几何**：
- 微分几何在流形上做分析
- 拓扑学提供连续性基础
- 几何分析结合几何与分析

#### 4.2.2 概念层次

**基础概念**：
