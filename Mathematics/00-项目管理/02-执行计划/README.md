## 📝 文档管理规则

### 新增计划文档

1. **当前计划**: 正在执行的计划文档应放在 `当前计划/`
2. **模块计划**: 模块特定计划应放在 `模块计划/[模块名]/`
3. **历史计划**: 已完成的计划应归档到 `历史计划/[YYYY-MM]/`

### 文档命名规范

- 使用清晰的描述性名称
- 包含日期标记（如适用）
- 使用中文名称（与项目一致）

### 文档更新

- 当前计划文档应定期更新
- 完成的计划应及时归档
- 保持文档索引的准确性
