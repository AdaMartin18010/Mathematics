# 自动化检查工具使用说明

**创建日期**: 2025-12-20
**版本**: v1.0
**状态**: 可用

---

## 📋 工具概述

自动化检查工具用于检查项目文档的格式、链接有效性和数学公式语法，帮助提高文档质量。

---

## 🚀 使用方法

### 基本使用

```powershell
# 检查所有项目
.\自动化检查工具脚本.ps1

# 检查指定目录
.\自动化检查工具脚本.ps1 -TargetPath ".\Math\Analysis"

# 只检查格式
.\自动化检查工具脚本.ps1 -CheckType format

# 只检查链接
.\自动化检查工具脚本.ps1 -CheckType links

# 只检查数学公式
.\自动化检查工具脚本.ps1 -CheckType math
```

---

## 🔍 检查内容

### 1. 文档格式检查

- **Markdown表格格式**: 检查表格是否有正确的分隔符
- **标题层级**: 检查标题层级是否连续（不能跳跃）

**示例输出**:
```
1. 检查文档格式...
   检查了 250 个Markdown文件
```

---

### 2. 链接有效性检查

- **相对路径链接**: 检查相对路径链接是否有效
- **文件存在性**: 验证链接目标文件是否存在
- **锚点链接**: 支持锚点链接检查

**示例输出**:
```
2. 检查链接有效性...
   检查了 1500 个链接，发现 5 个断链
```

---

### 3. 数学公式语法检查

- **括号匹配**: 检查LaTeX公式中的括号是否匹配
- **基础语法**: 检查常见语法错误

**示例输出**:
```
3. 检查数学公式语法...
   检查了 800 个数学公式，发现 2 个潜在问题
```

---

---

## ⚙️ 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `-CheckType` | string | "all" | 检查类型：all, format, links, math |
| `-TargetPath` | string | "." | 目标路径（相对或绝对路径） |

---

## 🔧 高级用法

### 集成到CI/CD

```powershell
# 在CI/CD管道中使用
$result = .\自动化检查工具脚本.ps1 -CheckType all
if ($LASTEXITCODE -ne 0) {
    exit 1
}
```

### 批量检查

```powershell
# 检查所有模块
$modules = @("Analysis", "Refactor", "Lean", "Matter", "AI-Mathematics")
foreach ($module in $modules) {
    Write-Host "检查 $module 模块..." -ForegroundColor Cyan
    .\自动化检查工具脚本.ps1 -TargetPath ".\Math\$module"
}
```

---

## 📝 注意事项

1. **执行策略**: 可能需要设置PowerShell执行策略
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
   ```

2. **性能**: 大型项目检查可能需要几分钟时间

3. **误报**: 某些检查可能产生误报，需要人工验证

4. **外部链接**: 工具不检查外部HTTP/HTTPS链接的有效性
