**时间**: 30-60分钟

**步骤**:

1. 创建备份目录

   ```powershell
   # Windows PowerShell
   New-Item -ItemType Directory -Path "..\Mathematics_Backup_2025-12-20"
   ```

2. 复制整个Math目录

   ```powershell
   Copy-Item -Path "..\Math" -Destination "..\Mathematics_Backup_2025-12-20\Math" -Recurse
   ```

## ⚠️ 注意事项

### 风险控制

1. **备份优先**: 迁移前必须创建完整备份
2. **分阶段执行**: 不要一次性迁移所有内容
3. **验证完整性**: 每个阶段迁移后验证
4. **记录过程**: 详细记录每个步骤

### 常见问题

1. **路径过长**: Windows系统可能有路径长度限制
2. **权限问题**: 确保有足够的文件系统权限
3. **链接失效**: 迁移后需要更新所有相对路径
