## 📋 格式检查概述

本清单提供了完整的格式检查项目，用于确保所有文档符合项目格式标准。格式标准化工作已完成（51个文档），本清单用于后续文档的格式验证和维护。

---

## ✅ 文档头部格式检查

### 标准头部格式

所有文档应包含以下标准头部：

```markdown
# 文档标题

**创建日期**: YYYY-MM-DD
**版本**: v1.0
**状态**: 草案/进行中/完成
**适用范围**: [说明]

---
```

### 检查项目

- [ ] 文档标题格式正确（一级标题 `#`）
- [ ] 创建日期格式正确（YYYY-MM-DD）
- [ ] 版本号格式正确（v1.0）
- [ ] 状态值正确（草案/进行中/完成）
- [ ] 适用范围说明清晰
- [ ] 分隔线 `---` 存在且正确

---

## 📐 标题层级检查

### 标题层级标准

- **一级标题**: `# 标题`
- **二级标题**: `## 标题`
- **三级标题**: `### 标题`
- **四级标题**: `#### 标题`
- **五级标题**: `##### 标题`

### 检查项目

- [ ] 标题层级使用正确（不跳级）
- [ ] 标题格式统一（无多余空格）
- [ ] 标题内容清晰简洁
- [ ] 标题层级符合文档结构

---

## 📝 列表格式检查

### 列表格式标准

- **无序列表**: `- 项目`
- **有序列表**: `1. 项目`
- **任务列表**: `- [ ] 任务` 或 `- [x] 已完成任务`

### 检查项目

- [ ] 列表格式统一（使用 `-` 或数字）
- [ ] 列表缩进正确（2空格）
- [ ] 任务列表格式正确（`- [ ]` 或 `- [x]`）
- [ ] 列表项内容完整

---

## 🔗 链接格式检查

### 链接格式标准

- **内部链接**: `[文本](./路径.md)` 或 `[文本](../路径.md)`
- **外部链接**: `[文本](URL)`
- **锚点链接**: `[文本](#锚点)`

### 检查项目

- [ ] 链接格式正确（Markdown格式）
- [ ] 链接路径正确（相对路径）
- [ ] 链接文本清晰
- [ ] 链接有效性（目标文件存在）
- [ ] 锚点链接正确（标题ID匹配）

---

- [ ] 表格内容完整
- [ ] 表格标题清晰

### 数学公式格式标准

- **行内公式**: `$公式$` 或 `` `公式` ``

### 检查项目

## 📋 分隔线格式检查

### 分隔线格式标准

- **水平分隔线**: `---`（三个连字符）

### 检查项目

- [ ] 分隔线格式正确（三个连字符）
- [ ] 分隔线位置恰当（章节之间）
- [ ] 分隔线前后空行正确

---

## 📝 段落格式检查

### 段落格式标准

- **段落间距**: 段落间空一行
- **段落长度**: 建议每段不超过200字
- **段落结构**: 主题句 + 支持句 + 总结句

### 检查项目

- [ ] 段落间距正确（空一行）
- [ ] 段落长度合理（不超过200字）
- [ ] 段落结构清晰（主题句明确）

### 检查项目

- [ ] 强调格式正确（使用 `**` 或 `*`）
- [ ] 强调使用恰当（不过度使用）
- [ ] 强调内容重要

## 📋 格式检查流程

### 1. 创建新文档时

1. 使用标准头部模板
2. 按照格式规范编写
3. 使用本清单检查
4. 确认格式正确后提交

### 2. 更新现有文档时

1. 检查头部格式
2. 检查更新内容格式
3. 使用本清单验证
4. 确认格式一致后提交

### 3. 定期检查时

1. 选择检查范围
2. 使用本清单逐项检查
3. 记录发现的问题
4. 修复格式问题
5. 更新检查记录
