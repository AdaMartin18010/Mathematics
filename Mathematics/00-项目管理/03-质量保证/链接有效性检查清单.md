## 📋 链接检查概述

本清单提供了完整的链接有效性检查项目，用于确保所有文档中的链接都有效且导航系统运行顺畅。

---

## ✅ 内部链接检查

### 文件链接检查

#### 相对路径链接

- [ ] 相对路径格式正确（`./文件.md` 或 `../目录/文件.md`）
- [ ] 目标文件存在
- [ ] 路径大小写正确（Windows/Linux兼容性）
- [ ] 路径中无多余空格或特殊字符

#### 目录链接

- [ ] 目录路径正确（`./目录/` 或 `../目录/`）
- [ ] 目标目录存在
- [ ] 目录路径格式正确

### 锚点链接检查

#### 同文件锚点

- [ ] 锚点格式正确（`#锚点名称`）
- [ ] 锚点名称与标题匹配
- [ ] 锚点名称格式正确（空格替换为连字符，小写）

#### 跨文件锚点

- [ ] 文件路径正确（`./文件.md#锚点`）
- [ ] 目标文件存在
- [ ] 锚点名称与目标文件中的标题匹配

### 导航链接检查

#### 返回导航链接

- [ ] 返回导航链接指向正确的上级目录
- [ ] 返回导航链接格式统一
- [ ] 返回导航链接有效

#### 规范入口链接

- [ ] 规范入口链接指向正确的规范文档
- [ ] 规范入口链接格式统一
- [ ] 规范入口链接有效

#### 交叉引用链接

- [ ] 交叉引用链接形成完整的引用网络
- [ ] 交叉引用链接双向性（如需要）
- [ ] 交叉引用链接有效

### 检查范围

- **内部链接**: 所有相对路径链接、锚点链接、导航链接

### 检查频率

- **新文档**: 创建时立即检查
- **更新文档**: 更新时检查相关链接
- **定期检查**:
  - 内部链接：每周检查一次
  - 外部链接：每月检查一次
  - 全面检查：每季度检查一次

## 📋 链接检查流程

### 1. 创建新文档时

1. 检查所有内部链接
2. 检查所有外部链接
3. 验证锚点链接有效性
4. 确认链接格式正确后提交

### 2. 更新现有文档时

1. 检查新增链接
2. 检查修改的链接
3. 验证相关锚点链接
4. 确认链接有效性后提交

### 3. 定期检查时

1. 选择检查范围（模块或目录）
2. 使用本清单逐项检查
3. 记录发现的问题
4. 修复无效链接
5. 更新检查记录

## 🚨 常见问题及解决方案

### 内部链接问题

#### 问题1: 文件重命名导致链接失效

**解决方案**:
1. 使用全局搜索替换更新所有引用
2. 更新目录和索引文件
3. 验证所有相关链接

#### 问题2: 锚点名称变更

**解决方案**:
1. 更新所有指向该锚点的链接
2. 检查页面内目录结构
3. 确保锚点名称一致性

#### 问题3: 路径错误

**解决方案**:
1. 检查相对路径计算
2. 验证目录结构
3. 更新路径引用

### 外部链接问题

#### 问题1: 大学课程链接失效

**解决方案**:
1. 查找课程的新链接
2. 更新访问日期
3. 如果无法找到替代，标记为历史链接

#### 问题2: Wikipedia链接不稳定

**解决方案**:
1. 使用主词条入口链接
2. 避免使用特定语言版本链接
3. 定期检查链接有效性

#### 问题3: 学术资源链接失效

**解决方案**:
1. 查找替代资源
2. 使用DOI或永久链接
3. 标记为历史链接（如需要）
