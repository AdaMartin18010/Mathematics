## 📚 学习路径

### 按学科学习

1. **分析学路径**：
   - 从实数完备性开始，建立实数系理论基础
   - 学习极限与连续性，掌握ε-δ语言
   - 学习微分学基础，掌握中值定理
   - 学习Riemann积分，掌握微积分基本定理
   - 学习级数理论，掌握收敛判别法

2. **代数学路径**：
   - 从群论基础开始，掌握群的基本概念
   - 学习环论基础，掌握理想与商环
   - 学习向量空间，掌握基与维数
   - 学习线性变换，掌握特征值与对角化
   - 学习多项式环，掌握唯一分解

3. **拓扑与几何路径**：
   - 从度量空间开始，掌握距离与收敛
   - 学习拓扑空间，掌握开集与连续性
   - 学习赋范空间，掌握Banach空间理论
   - 学习内积空间，掌握Hilbert空间理论
   - 学习微分流形，掌握现代几何基础

### 按难度学习

1. **中等难度**（⭐⭐⭐）：
   - 实数完备性、极限与连续性
   - 群论基础、环论基础
   - 度量空间、向量空间

2. **中等偏难**（⭐⭐⭐⭐）：
   - 微分学基础、Riemann积分、级数理论
   - 线性变换、多项式环
   - 拓扑空间、赋范空间、内积空间

3. **困难**（⭐⭐⭐⭐⭐）：
   - 微分流形

### 综合学习路径

**基础阶段**：
1. 分析学基础（实数完备性 → 极限与连续性 → 微分学基础 → Riemann积分 → 级数理论）
2. 代数基础（群论基础 → 环论基础 → 向量空间 → 线性变换 → 多项式环）

**进阶阶段**：
3. 拓扑与几何（度量空间 → 拓扑空间 → 赋范空间 → 内积空间 → 微分流形）
