## 📋 项目概述

本项目是一个**全面重构的数学知识体系项目**，旨在建立符合2025年国际标准的数学知识库，整合形式化验证、认知科学、哲学批判等多个维度，为全球数学学习者、研究者和教育者提供高质量、系统化、易用的数学知识资源。
